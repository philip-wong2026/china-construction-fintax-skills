# 脱敏 Demo

本目录提供三个可复现的脱敏示例。它们不是企业真实报表，也不代表正式财税意见，只用于让新用户理解 CCFTS 的使用方式和输出结构。

## Demo 1：项目部 / 总包部快报检查

目录：`demo-project-unit-flash-report/`

适合：

- 非法人项目部
- 总包部
- 施工总承包类主体
- 项目财务或报表人员

主要演示：

- 如何根据科目余额表判断 Type B 施工总承包类主体
- 如何检查利润表和资产负债表映射
- 如何输出人工复核清单

## Demo 2：项目公司 SPV 快报检查

目录：`demo-spv-flash-report/`

适合：

- 独立法人项目公司
- 投资管理类 SPV
- 子公司或项目公司财务

主要演示：

- 如何判断 Type A 投资管理类主体
- 如何处理资产、负债、权益平衡关系
- 如何识别项目公司和项目部口径差异

## Demo 3：增值税跨区域预缴

目录：`demo-vat-prepayment/`

适合：

- 税务岗
- 项目财务
- 涉及跨区域施工项目的共享中心人员

主要演示：

- 如何区分一般计税和简易计税
- 如何计算跨区域预缴税额
- 如何提示分包扣除凭证、项目所在地附加税费和地方口径复核

## 推荐使用方式

1. 先读 [docs/quick-start-10min.md](../docs/quick-start-10min.md)
2. 填一份 [项目背景包](../docs/context-pack-template.md)
3. 选择一个 demo 目录
4. 把 demo 的 `README.md`、`input/` 和相关 skill 交给 AI
5. 对照 `expected/` 检查 AI 输出是否接近

## 注意

Demo 的价值不是证明 AI 一定正确，而是提供一个可练习、可复核、可讨论的最小样本。正式业务使用前，必须换成你的脱敏资料，并由具备专业能力的人员复核。
