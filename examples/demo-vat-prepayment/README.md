# Demo: VAT 跨区域预缴计算

## 目的

验证 CCFTS 技能对施工企业跨区域 VAT 预缴的计算流程——包括一般计税 2% 预征率、简易计税 3% 预征率、分包款扣除、附加税费预缴。

## 场景

某施工企业的一项公路工程项目位于外省，需要计算跨区域 VAT 预缴税额。项目含一般计税部分（9%）和清包工简易计税部分（3%），且有分包。

## 输入文件

| 文件 | 路径 | 说明 |
|------|------|------|
| 项目合同信息 | `input/project-info.yaml` | 项目地址/合同金额/计税方式/分包信息 |
| 收款记录 | `input/payment-receipts.csv` | 本期收款金额/日期/对应合同段 |

## 应加载的技能（按顺序）

1. `ccfts-intel-sta-vat-law-2026`
2. `ccfts-tax-all-vat-cross-region-prepayment`
3. `ccfts-tax-all-vat-general-filing`
4. `ccfts-tax-all-vat-simplified-filing`（如含简易计税部分）

## 期望输出

| 输出项 | 内容 |
|--------|------|
| 计税方式判定 | 一般计税部分：预征率 2%；简易计税部分：预征率 3% |
| 预缴税额计算 | 一般：(价款−分包款)/(1+9%)×2%；简易：(价款−分包款)/(1+3%)×3% |
| 附加税费 | 城建税 + 教育费附加 + 地方教育附加（按项目所在地税率） |
| 预缴台账 | 按项目核算的完整预缴记录 |

## 如何判断结果合理

- 一般计税预征率 2%，简易计税 3%——不可互换
- 分包款扣除凭证必须是增值税发票（非白条/收据）
- 附加税费按项目所在地城建税税率（7%/5%/1%），非机构所在地
- 预缴税额 ≤ 机构所在地申报的应纳税额（否则有退税 or 留抵）

## 当前限制

- **All data in this demo is synthetic, generated for pipeline validation only.**
- 本 demo 覆盖预缴税额计算和附加税费跟随计算，不覆盖金税四期三方比对和电子税务局实际操作。
- 各省税务机关对跨区域预缴的具体操作流程可能有差异，本 demo 基于通用规则。
