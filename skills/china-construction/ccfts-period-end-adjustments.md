# CCFTS — 期末调整规则

## 快速参考

| 项目 | 默认 | 调整模式 |
|---|---|---|
| 季度末（3/6/9/12月） | 正常模式 | 不应用调整 |
| 非季度末（1/2/4/5/7/8/10/11月） | 需显式激活 | 应用调整 |
| 激活方式 | 用户明确要求或代码参数 --apply-open-period-adjustments | |

## 一、为什么非季度末需要调整

非季度末月，企业通常不进行完整的月末结账。以下科目可能存在未结转余额：
- 福利费用（2211-02）— 当期发生但未分配至损益
- 管理费用（6602）— 当期发生但未结转至本年利润

这些未结转余额若不加处理，会导致：
- 利润表中的管理费用和福利费用不完整
- 资产负债表中的合同资产（其他非流动资产）少计

## 二、标准调整规则

### 调整 1：未结转福利费用

```
条件：2211-02 福利费用有借方余额
动作：视同结转至管理费用
  借：管理费用（6602）  ← 增加管理费用
  贷：应付职工薪酬—福利费用（2211-02） ← 清零借方余额
```

**取数口径**：2211-02 期末借方余额（元）。

### 调整 2：未结转管理费用

```
条件：6602 管理费用有借方余额（含调整 1 后的金额）
动作：视同结转至合同资产
  借：合同资产（1483） ← 增加合同资产
  贷：管理费用（6602） ← 清零
```

**取数口径**：6602 期末借方余额（含上述 2211-02 转入部分）。

**资产负债表影响**：其他非流动资产（1483-1484 净额）等额增加。

### 调整 3：银行存款重分类（一次性特例）

```
条件：用户提供具体金额和理由
动作：银行存款（1002）增加，其他应收款—资金集中管理款减少
```

**这不是通用规则**，仅适用于特定月份的资金归集调整。每次使用必须由用户显式提供金额和原因。

## 三、调整对快报的影响

| 调整 | 影响 Sheet | 影响行 |
|---|---|---|
| 2211-02 结转 | 利润指标表 | 管理费用（6602）增加 |
| 6602 结转 | 资产负债指标表 | 其他非流动资产（1483-1484）增加 |
| 6602 结转 | 利润指标表 | 管理费用恢复为 0（已全部结转） |
| 银行重分类 | 资产负债指标表 | 货币资金增加，其他应收款减少 |

## 四、不应用调整的默认情况

| 期间类型 | 处理 | 原因 |
|---|---|---|
| 季度末（3/6/9月） | 不调整 | 企业应已完成结账 |
| 年末（12月） | 不调整 | 必须完成全年结账和决算 |
| 用户未显式要求 | 不调整 | 调整属于判断性处理，不能自动应用 |

## 五、决策流程

```
读取科目余额表
    │
    ├── 季度末或年末？ ──YES──→ 不应用调整
    │
    ├── 用户要求应用调整？ ──NO──→ 不应用调整
    │
    └── YES
        │
        ├── 2211-02 有借方余额？ ──YES──→ 结转至 6602
        │
        ├── 6602 有借方余额（含上述转入）？ ──YES──→ 结转至 1483
        │
        └── 用户指定了银行重分类金额？ ──YES──→ 调整 1002 和 1221
```

## 六、自检清单

1. [ ] 是否为非季度末？若为 3/6/9/12 月，不应应用调整
2. [ ] 用户是否显式要求了调整？
3. [ ] 2211-02 结转金额是否正确（期末借方余额）？
4. [ ] 6602 结转金额是否正确（含 2211-02 转入）？
5. [ ] 资产负债表其他非流动资产是否正确反映了调整后的 1483/1484 净额？
6. [ ] 利润表管理费用是否正确反映了调整后的金额？
7. [ ] 银行重分类是否为一次性特例（非通用默认）？

## 七、已知局限

- 只覆盖已观测到的未结转情况（2211-02、6602、银行重分类）。其他损益类科目（如 6403 税金及附加、6601 销售费用等）的未结转调整尚未验证
- 调整逻辑假设福利费用全部计入管理费用，未考虑计入合同履约成本（5601）或在建工程的情况
- 银行重分类仅覆盖已验证的一次性场景（660000 元）

## 八、参考材料

- 基于 2026M05 实际快报编制验证
- 调整前：净利润显示为 86 万元（取 4105 余额），调整后恢复为 31 万元（取 4105-14 净发生额 + 调整后的正确结果）

## 免责声明

本调整规则基于有限期间的实际操作经验，可能不覆盖所有企业的期末调整需求。调整属于会计判断，应经审核后使用。
