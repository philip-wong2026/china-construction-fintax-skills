## 五、特殊科目处理说明

### 1483/1484 合同资产（最重要）

**错误做法**：ROUND(1483各子科目合计 - 1484各子科目合计)
**正确做法**：ROUND(1483-01) + ROUND(1483-02) + ... — ROUND(1484-01) — ROUND(1484-02) — ...

Type A：各子科目分别 ROUND_HALF_UP 后相减
Type B：各子科目分别 ROUND_DOWN 后相减

### 2241 其他应付款（Type A）

不能用直接取数！必须倒推：
```
2241 = 负债合计 - 2221(应交税费) - 2211(应付职工薪酬)
```
原因：直接取 2241 会产生 ±1 万元差异（Q4 验证发现 265000 元恰在 0.5 万元临界点）。

### 4105-14 净利润（Type A）

必须取 `4105-14` 本期贷方净发生额（credit - debit），不能取 `4105` 期末余额。
原因：期末余额包含以前年度未分配利润，跨年度时会严重错误。

### 6603 财务费用

当利息收入（6603-02）大于利息支出（6603-01）时，净额为负数。快报中财务费用以负数填列。
