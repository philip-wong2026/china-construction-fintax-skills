# CCFTS — 利润表编制规则

## 快速参考

| 项目 | Type A（投资管理类） | Type B（施工总承包类） |
|---|---|---|
| 营业收入科目 | 6051 其他业务收入 | 6001 主营业务收入 |
| 营业成本科目 | 6402 其他业务成本 | 6401 主营业务成本 |
| 投资收益 | 6111（有） | 通常无 |
| 其他收益 | 6605（有） | 通常无 |
| 所得税费用 | 6801（有） | 通常无 |
| 净利润公式 | 利润总额 - 所得税费用 | 利润总额（无所得税） |
| 净利润验证 | 4105-14 net credit-debit | 利润总额一致 |
| 取整模式 | ROUND_HALF_UP | ROUND_HALF_UP |

## 一、利润计算链

```
                    6051/6001 贷方发生额
  营业总收入      = Type A: 6051, Type B: 6001
—
                    6402/6401 借方发生额
  营业成本        = Type A: 6402, Type B: 6401
—
  税金及附加      = 6403 借方发生额
—
  财务费用        = 6603 借方净发生额（借—贷）
                    → 可为负数（利息收入 > 利息支出时）
+
  其他收益        = 6605 贷方发生额（仅 Type A）
+
  投资收益        = 6111 贷方发生额（仅 Type A）
—
  资产减值损失    = 6701 借方净发生额
—
  信用减值损失    = 6702 借方净发生额
                    → Type B: 可为负（贷方发生额 > 借方时）
=
  营业利润
=
  利润总额        （无营业外收支时）
—
  所得税费用      = 6801 借方发生额（仅 Type A）
=
  净利润
```

## 二、科目映射详表

| 快报项目 | 科目 | 取数口径 | 符号处理 | 适用主体 |
|---|---|---|---|---|
| 营业总收入 | 6001 / 6051 | 贷方发生额 | 正数 | A + B |
| 其中：主营业务收入 | 6001 | 贷方发生额 | 正数 | B |
| 营业成本 | 6401 / 6402 | 借方发生额 | 正数 | A + B |
| 税金及附加 | 6403 | 借方发生额 | 正数 | A + B |
| 财务费用 | 6603 | 借方净发生额（借-贷） | 可为负 | A + B |
| 利息支出 | 6603-01 | 借方发生额 | 正数 | A + B |
| 利息收入 | 6603-02 | 贷方发生额 | **取反**（正数填列） | A + B |
| 其他收益 | 6605 | 贷方发生额 | 正数 | A |
| 投资收益 | 6111 | 贷方发生额 | 正数 | A |
| 资产减值损失 | 6701 | 借方净发生额 | 利润指标表：正数 | A + B |
| 信用减值损失 | 6702 | 借方净发生额 | Type B 可为负 | A + B |
| 所得税费用 | 6801 | 借方发生额 | 正数 | A |

| 净利润 | 计算（A）/ 利润总额（B） | — | 正数 | A + B |

## 三、关键注意事项

### 1. 净利润取数口径（Type A）

**正确**：取 4105-14 本期贷方净发生额（credit — debit）
**错误**：取 4105 期末余额

原因：4105 期末余额包含以前年度未分配利润和盈余公积转入，非本期经营成果。

验证方法：净利润 = 利润总额 — 所得税费用，应与 4105-14 净发生额一致。

### 2. 利息收入符号

6603-02 利息收入在科目余额表中通常为贷方发生额。填快报时须**取反为正数**（快报中以正数列示利息收入，财务费用净额 = 利息支出 — 利息收入）。

### 3. 资产减值损失双重列示

| Sheet | 列示方式 | 符号 |
|---|---|---|
| 利润指标表 | 以正数填列 | + |
| 资产负债指标表 / 主表 | 以损失填列 | — |

### 4. Type B 信用减值损失可为负

Type B 总包部的 6702 信用减值损失可能为负（当年冲回大于计提），直接以净额填列。

### 5. Type B 无所得税费用

总包部作为内部管理单位，通常不单独核算所得税。净利润 = 利润总额。

## 四、业务板块分配

利润指标表中的"业务板块"sheet 的利润分配：

| 行 | 内容 | 取值 |
|---|---|---|
| 基建建设行 | 利润总额 | = 利润总额 |
| 市政行 | 利润总额 | = 利润总额 |
| 营业收入/营业成本明细行 | — | 不填（保持空白） |

**不按标签模糊匹配，按固定行号写入**。

## 五、非季度/非年末月处理

非季度末月（1/2/4/5/7/8/10/11 月）的损益类科目可能未完全结转。需结合 `ccfts-period-end-adjustments` 判断是否需要补结转调整。

## 六、自检清单

1. [ ] 营业收入是否使用了正确的科目（6001 vs 6051）？
2. [ ] 营业成本是否使用了正确的科目（6401 vs 6402）？
3. [ ] 净利润 Type A 是否取自 4105-14 净发生额而非 4105 余额？
4. [ ] 净利润 Type B 是否等于利润总额？
5. [ ] 利息收入是否已取反为正数？
6. [ ] 资产减值损失在利润指标表是否为正数？
7. [ ] 业务板块分配是否正确？（基建建设 + 市政）
8. [ ] D 列与 G 列逻辑是否正确？（非年末月 vs 年末月）
