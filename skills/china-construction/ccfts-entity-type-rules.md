# CCFTS — 施工企业主体类型判定规则

## 快速参考

| 项目 | Type A（投资管理类） | Type B（施工总承包类） |
|---|---|---|
| 主营业务 | 投资管理、资产管理 | 建设施工总承包 |
| 主营收入科目 | 6051 其他业务收入 | 6001 主营业务收入 |
| 主营成本科目 | 6402 其他业务成本 | 6401 主营业务成本 |
| 合同结算科目(5801) | 无 | 有 |
| 合同履约成本(5601) | 无 | 有 |
| 应付账款(2202) | 无（填 0） | 有 |
| 长期应付款(2701) | 无 | 有 |
| 内部往来(3001) | 无 | 有 |
| 待结算进项税额(1126) | 无 | 有 |
| 权益结构 | 正常（实收资本+资本公积+盈余公积+未分配利润） | 内部总包部结构（资产≈负债，权益≈0） |
| 默认取整规则 | 全局 ROUND_HALF_UP | 资产负债表 ROUND_DOWN，利润表 ROUND_HALF_UP |
| 合同资产净额计算 | ROUND(1483) - ROUND(1484) | ROUND_DOWN(1483) - ROUND_DOWN(1484) |
| 其他应付款计算 | 倒推：负债合计 - 应交税费 - 应付职工薪酬 | ROUND_DOWN(2241-01) + ROUND_DOWN(3001) |
| 利润总额 = 净利润？ | 否（需减所得税费用 6801） | 是（通常无所得税费用科目） |

## 一、判定决策树

```
读取科目余额表科目列表
    │
    ├── 有 5801（合同结算）？ ──YES──→ Type B（施工总承包类）
    │
    ├── 有 6001（主营业务收入）？ ──YES──→ Type B
    │
    ├── 有 6401（主营业务成本）？ ──YES──→ Type B
    │
    ├── 有 3001（内部往来）？ ──YES──→ Type B
    │
    ├── 有 6051（其他业务收入）且无 6001？ ──YES──→ Type A（投资管理类）
    │
    └── 仍不确定？ → 列出可能的科目特征，请求用户确认
```

**关键判定信号（按优先级）**：
1. **5801 合同结算** — 最可靠的 Type B 信号。投资管理类绝对没有此科目。
2. **6001 vs 6051** — 主营收入科目直接区分业务性质。
3. **权益结构** — 权益≈零（资产 = 负债）→ Type B。正常权益 → Type A。
4. **2202 应付账款** — Type A 无（填 0）；Type B 有，需减去 1123 预付账款。

## 二、Type A — 投资管理类详细规则

### 科目特征
- 收入：6051 其他业务收入（贷方发生额）
- 成本：6402 其他业务成本（借方发生额）
- 无：6001、6401、5801、5601、2202、2701、3001、1126
- 有正常权益结构：4001（实收资本）、4002（资本公积）、4101（盈余公积）、4105-14（未分配利润）

### 资产负债表关键计算
| 快报项目 | 计算方式 | 注意 |
|---|---|---|
| 货币资金 | 1002 各子行 ROUND_HALF_UP 后汇总 | 区分内部财务公司 vs 外部金融机构 |
| 预付账款 | 1123 期末借方余额 | 直接取 |
| 其他应收款 | ROUND(1221) - ROUND(1231-02) | 净额四舍五入 |
| 固定资产净额 | ROUND(1601) - ROUND(1602) | 分项取整后相减 |
| 其他非流动资产 | ROUND(1483各子行) - ROUND(1484各子行) | 分别取整后相减 |
| 其他应付款 | 负债合计 - 应交税费 - 应付职工薪酬 | 倒推！不直接取 2241 |
| 递延所得税资产 | 1811 期末借方余额 | |

### 利润表关键计算
- 利润总额 = 营业收入 - 营业成本 - 税金及附加 - 财务费用 + 其他收益 + 投资收益 - 资产减值损失 - 信用减值损失
- 净利润 = 利润总额 - 所得税费用
- 净利润取 4105-14 本期贷方净发生额（credit - debit），**不能取 4105 期末余额**（含以前年度留存收益）

### 业务板块分配
- 利润指标表"基建建设"行 = 利润总额
- 利润指标表"市政"行 = 利润总额
- 营业收入/营业成本的明细行不填

## 三、Type B — 施工总承包类详细规则

### 科目特征
- 收入：6001 主营业务收入（贷方发生额）
- 成本：6401 主营业务成本（借方发生额）
- 有：5801（合同结算）、5601（合同履约成本）、2202（应付账款）、2701（长期应付款）、3001（内部往来）、1126（待结算进项税额）
- 权益≈零（内部总包部结构，资产 = 负债）

### 取整规则（与 Type A 完全不同）
- 资产负债表所有行：**ROUND_DOWN**（截尾取整到万位）
- 利润表所有行：**ROUND_HALF_UP**（四舍五入）
- 1483/1484：分别 ROUND_DOWN 后相减
- 货币资金各子行：分别 ROUND_DOWN 后汇总

### 资产负债表关键计算
| 快报项目 | 计算方式 |
|---|---|
| 货币资金 | 各银行账户 ROUND_DOWN 后相加 |
| 其他应收款 | ROUND_DOWN(1221) - ROUND_DOWN(2241-29) |
| 合同资产 | ROUND_DOWN(1483) - ROUND_DOWN(1484) |
| 其他非流动资产合计 | ROUND_DOWN(1531净) + 合同资产 |
| 应付账款 | ROUND_DOWN(2202) - ROUND_DOWN(1123) |
| 合同负债 | ROUND_DOWN(5801 贷方余额) |
| 其他应付款 | ROUND_DOWN(2241-01) + ROUND_DOWN(3001) |
| 其他流动负债 | ROUND_DOWN(1126) |
| 长期应付款 | ROUND_DOWN(2701) |
| 权益合计 | 约 0 |

### 利润表关键计算
- 营业总收入 = 6001 本年累计贷方 → 其中基建建设（城轨等）同值
- 信用减值损失(6702)可为负（冲回大于计提时）
- **净利润 = 利润总额**（总包部通常无所得税费用科目 6801）
- 利息收入 = 6603-02 贷方发生额取反（以正数填列）

## 四、分类置信度

| 科目信号数 | 置信度 | 处理方式 |
|---|---|---|
| ≥3 个信号一致 | Classified | 直接确定类型 |
| 1-2 个信号 | Assumed | 标记假设，列出推断依据 |
| 0 个信号或信号矛盾 | Needs Input | 列出科目特征，请求用户确认 |

## 五、混合场景处理

如果同一套系统下存在不同主体类型（如一家项目公司和一家总包部），**必须分别处理、分别编制快报**，不能交叉引用对方的科目映射规则。
