# 国家审计与巡视整改财务配合

## 快速参考

| 审计类型 | 审计主体 | 频率 | 重点 |
|---------|---------|------|------|
| 国家审计署审计 | 审计署 | 不定期（通常 2-3 年一次） | 财务收支真实性、国有资产保值增值 |
| 巡视 | 上级党委巡视组 | 不定期 | 党建+廉洁+选人用人+整改落实 |
| 专项审计 | 审计署/国资委 | 按需 | 特定领域（PPP/境外/债务风险） |
| 经济责任审计 | 审计署/国资委 | 离任时 | 领导人员任期经济责任 |

## 一、国家审计署审计——施工企业重点

### 审计重点领域

| 领域 | 审计内容 | 施工企业常见问题 |
|------|---------|----------------|
| 收入真实性 | 有无虚增收入（提前确认/虚构合同） | 已完工未结算规模是否合理 |
| 成本真实性 | 有无虚列成本（虚假发票/虚构分包） | 分包结算依据是否充分 |
| 资产质量 | 应收款项/合同资产可回收性 | 长期挂账应收款是否足额计提 |
| PPP 项目 | 合规性/资产负债率影响 | 是否违规出表、隐性债务 |
| 境外投资 | 投资决策/风险管控 | 境外项目亏损是否及时披露 |
| 关联交易 | 定价公允性 | 内部总分包是否转移利润 |
| 工资福利 | 薪酬合规性 | 有无违规发放津贴补贴 |

### 审计配合要点

| 阶段 | 财务部门任务 |
|------|------------|
| 审前（接到通知后 1-2 周） | 整理资料清单、自查摸底、准备解释口径 |
| 审中（1-3 个月） | 及时提供资料、书面回复询问、沟通解释 |
| 审后（审计报告征求意见） | 核对事实、反馈意见（10-15 日）、整改方案 |

### 资料准备清单

1. 会计凭证/账簿/报表（审计期间的完整档案）
2. 银行对账单及余额调节表
3. 合同台账（施工/分包/采购/PPP）
4. 关联交易汇总表
5. 重大事项说明（会计政策变更、合并范围变更、资产处置）
6. 前次审计/巡视发现问题及整改情况

## 二、巡视整改——财务角度

### 巡视关注的财务问题

| 类别 | 典型问题 |
|------|---------|
| 违反中央八项规定 | 违规公款吃喝/旅游、超标配车/办公用房 |
| "小金库" | 账外资金、个人卡收付工程款 |
| 违规职务消费 | 高档烟酒、高尔夫、会所消费 |
| 违规发放津补贴 | 超标准/超范围发放奖金福利 |
| 选人用人 | 领导人员亲属经商办企业 |
| 招投标 | 围标串标、利益输送、应招标未招标 |

### 整改回复模板

```
【整改事项】[问题描述]
【问题性质】[违规类型]
【涉及金额】[金额]（如适用）
【整改措施】
  1. 立即整改：[具体措施]
  2. 追责问责：[责任人及处理结果]
  3. 制度建设：[修订制度/流程]
  4. 长效机制：[防止再犯]
【完成时限】[已完成 / 预计 XX 日前完成]
【责任人】[姓名+职务]
```

## 三、常见审计发现——施工企业版

| 发现 | 审计定性 | 整改要求 |
|------|---------|---------|
| 未完施工提前确认收入 | 收入不实 | 冲回虚增收入、追责 |
| 合同资产减值不充分 | 资产不实 | 补提减值、完善政策 |
| PPP 项目应合未合 | 合并范围错误 | 追溯合并、重述报表 |
| 内部拆借未计息 | 关联交易不公允 | 补提利息、完善定价 |
| 安全资金被挪用 | 违规 | 追回资金、追责 |
| 工程款转入个人卡 | "小金库" | 取缔、追回、追责 |

## 四、自检清单

1. [ ] 上次审计/巡视发现的问题是否都已整改完毕？
2. [ ] 有无"小金库"（个人卡/账外资金）？
3. [ ] PPP 项目合并范围是否正确？
4. [ ] 关联交易定价是否有依据？
5. [ ] 薪酬福利是否符合规定？
6. [ ] 审计资料是否已在日常归档（而非临时拼凑）？

## 免责声明

国家审计和巡视的具体要求以审计署和国资委当年通知为准。
