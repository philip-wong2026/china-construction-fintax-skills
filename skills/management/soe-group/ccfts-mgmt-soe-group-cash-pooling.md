# 资金集中管理（资金池）

## 快速参考

| 项目 | 规则 |
|---|---|
| 归集模式 | 集团财务公司 or 结算中心 → 统一归集各子企业资金 |
| 归集比例 | 通常 **80-100%**（各央企要求不同） |
| 内部计息 | 归集资金按内部利率计息（通常 ≥ 同期存款利率） |
| 下拨 | 子企业申请 → 集团审批 → 下拨（通常 T+1） |
| 银行账户 | 子企业银行账户数量/类型受集团严格管控 |

## 一、资金池运作模式

### 标准归集流程

```
各子企业银行账户（每日终了）
  ↓ 自动归集（银企直连）
集团财务公司 / 结算中心（主账户）
  ↓ 统一调度
  ├── 偿还集团外部贷款
  ├── 投资理财（合规范围内）
  └── 下拨子公司（按需申请）
```

### 归集的两个层次

| 层次 | 归集范围 | 管理主体 |
|------|---------|---------|
| 第一层：工程局→集团 | 工程局层面资金归集至集团财务公司 | 集团 |
| 第二层：项目部→工程局 | 项目部资金归集至工程局 | 工程局 |

**项目部层面**：项目部银行账户通常设置为"收入账户"（只收不付，自动上划）和"支出账户"（限额管理，按预算下拨）。

## 二、内部计息

### 计息规则

| 方向 | 利率基础 | 说明 |
|------|---------|------|
| 子公司存入（上划） | ≥ 同期银行存款利率 | 保证子公司资金收益不低于外部存款 |
| 子公司借用（下拨） | ≥ 存入利率 + 利差 | 利差覆盖集团管理成本 |
| 内部拆借 | 参照同期银行贷款利率 | 见关联交易文件 |

### 会计处理

```
子公司（上划资金）：
  借：其他应收款—资金集中管理—集团财务公司  XXX
    贷：银行存款                              XXX

  收到利息：
    借：银行存款                              XXX
      贷：财务费用—利息收入（6603-02）        XXX

集团财务公司：
  借：银行存款                                XXX
    贷：其他应付款—资金集中管理—xx子公司      XXX
```

## 三、银行账户管控

### 账户开立管控

| 管控要求 | 说明 |
|---------|------|
| 账户开立审批 | 子企业开立银行账户须经集团审批 |
| 账户数量限制 | 每个子企业通常限 2-5 个账户 |
| 账户类型限制 | 基本户 + 一般户 + 专户（农民工工资/安全资金） |
| 收入账户 | 只收不付，自动上划 |
| 支出账户 | 限额管理，按周/月预算下拨 |

### 禁止行为

- ❌ 子企业未经审批私自开立银行账户（"小金库"）
- ❌ 项目部将工程款存入非归集账户
- ❌ 通过个人卡收付工程款项
- ❌ 利用资金池变相抽逃子企业注册资本

## 四、SASAC 合规要求

### 资金集中度考核

SASAC 对央企资金集中度有考核要求：

```
资金集中度 = 归集至集团财务公司的资金 / 全部可归集资金

目标：通常要求 ≥ 80%
```

### 大额资金监控

| 监控内容 | 阀值 |
|---------|------|
| 大额资金支付预警 | 单笔 ≥ 50 万元（各企业设定不同） |
| 大额资金调拨审批 | 超出子公司日预算的部分 |
| 异常交易监控 | 频繁/大额/非关联方/非交易时间 |

## 五、自检清单

1. [ ] 资金归集比例是否达到集团要求？
2. [ ] 有无未经审批的银行账户？
3. [ ] 内部计息利率是否合理（不低于银行存款利率）？
4. [ ] 资金下拨是否及时（不影响子公司正常经营）？
5. [ ] 有无"小金库"（个人卡/未归集账户）？
6. [ ] 大额资金支付是否经过审批？

## 免责声明

资金集中管理的具体规则以各央企集团的财务管理制度为准。
