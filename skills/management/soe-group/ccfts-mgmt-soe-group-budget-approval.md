# 集团全面预算管理

## 快速参考

| 阶段 | 时间 | 负责 |
|------|------|------|
| 预算编制 | 9-10月（下年度） | 各子企业→集团汇总 |
| 预算审批 | 11-12月 | 集团董事会/SASAC |
| 预算下达 | 次年1月 | 集团→各子企业 |
| 执行监控 | 月度/季度 | 各级财务部门 |
| 预算调整 | 年中（7-8月） | 集团董事会审批 |
| 决算与考核 | 次年4-5月 | 集团+审计 |

## 一、预算编制——自上而下 + 自下而上

### 编制流程

```
集团总部下达预算编制指引（9月）
  ↓
各子企业编制本单位预算（10月）
  ↓
集团汇总审核 → 平衡调整（11月）
  ↓
提交董事会审批（12月）
  ↓
报 SASAC 备案（央企）
```

### 预算编制内容

| 预算类型 | 内容 |
|---------|------|
| 业务预算 | 新签合同额、施工产值、营业收入（分板块） |
| 成本预算 | 人工/材料/机械/分包/间接费（分项目） |
| 费用预算 | 管理费用、销售费用、研发费用 |
| 投资预算 | 固定资产投资、股权投资、PPP 项目投资 |
| 资金预算 | 经营/投资/筹资现金流 |
| 财务预算 | 预计资产负债表、预计利润表、预计现金流量表 |

### 关键编制假设

| 假设项 | 依据 |
|--------|------|
| 新签合同额增长率 | 市场形势 + 在手合同 |
| 毛利率 | 历史水平 + 在手项目预计 |
| 人工成本增长率 | 工资增长政策 |
| 材料价格走势 | 市场价格预测 |
| 汇率（境外项目） | 金融机构预测 |
| 税率政策 | 最新税法 |

## 二、预算审批

### 集团内审批

| 层级 | 审批内容 |
|------|---------|
| 子企业董事会 | 本单位预算 |
| 集团预算管理委员会 | 汇总预算草案 |
| 集团董事会 | 集团合并预算 |
| SASAC | 央企年度预算（备案） |

### 审批关注要点

- "一利五率"指标是否满足 SASAC 考核要求？
- 资产负债率是否在红线（75%）以内？
- "两金"是否有压降计划？
- 利润增长是否与收入增长匹配？
- 是否有重大亏损项目拖累？

## 三、预算下达与分解

### 下达方式

```
集团合并预算（董事会批准后）
  ↓ 分解下达
各子企业年度预算目标（"责任书"）
  ↓ 继续分解
各项目部责任成本预算
```

### 下达核心指标

| 指标 | 说明 |
|------|------|
| 新签合同额 | 最低目标 + 挑战目标 |
| 营业收入 | |
| 利润总额 | |
| 资产负债率 | 不高于红线 |
| "两金"压降额 | |
| 经营现金流 | |

## 四、执行监控

### 月度监控

| 监控内容 | 频率 | 工具 |
|---------|------|------|
| 收入/成本/利润 vs 预算 | 月度 | 经营快报 |
| 现金流 vs 预算 | 月度 | 资金月报 |
| 新签合同 vs 预算 | 月度 | 市场月报 |

### 季度分析

| 分析内容 | 说明 |
|---------|------|
| 预算偏差率排名 | 各子企业排名、红黄绿灯 |
| 重大偏差原因 | 逐项说明 |
| 趋势预测 | 全年预测 vs 年度预算目标 |

### 预警机制

| 预警 | 条件 | 行动 |
|------|------|------|
| 黄色 | 累计偏差 5-10% | 子企业书面说明 |
| 橙色 | 累计偏差 10-20% | 集团专项督导 |
| 红色 | 累计偏差 > 20% | 集团约谈+调整预算 |

## 五、预算调整

### 调整条件

| 条件 | 说明 |
|------|------|
| 市场环境重大变化 | 材料价格暴涨/暴跌 |
| 国家政策重大调整 | 税率变动/会计准则变更 |
| 企业重大重组 | 合并/分立 |
| 不可抗力 | 自然灾害/疫情 |

### 调整流程

```
子企业提出调整申请 → 集团预算办审核 → 董事会审批 → SASAC 备案
```

**注意**：预算调整不能成为"完不成任务的借口"——SASAC 对央企预算调整有严格限制。

## 六、预算考核

### 考核指标体系

| 指标类别 | 指标 | 权重 |
|---------|------|------|
| 规模 | 营业收入、新签合同额 | 20% |
| 效益 | 利润总额、ROE | 35% |
| 风险 | 资产负债率、"两金" | 25% |
| 发展 | 研发强度、营业收现率 | 20% |

### 考核结果应用

| 等级 | 结果 |
|------|------|
| A | 通报表扬、负责人薪酬上浮 |
| B | 正常 |
| C | 关注、负责人薪酬下浮 |
| D | 通报批评、或予调整 |

## 七、自检清单

1. [ ] 预算编制指引是否在 9 月前下发？
2. [ ] "一利五率"预算目标是否满足 SASAC 要求？
3. [ ] 各子企业的预算是否经董事会审批？
4. [ ] 月度/季度预算执行分析是否按时完成？
5. [ ] 预算偏差是否触发红黄绿灯预警并采取了行动？
6. [ ] 是否需要在年中调整预算？

## 免责声明

预算管理的具体流程和权限以各央企集团的预算管理制度为准。
