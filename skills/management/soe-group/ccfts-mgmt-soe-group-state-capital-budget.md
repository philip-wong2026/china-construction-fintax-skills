# 国有资本经营预算与利润上缴

## 快速参考

| 项目 | 规则 |
|---|---|
| 上缴比例（一般竞争类央企） | **30%**（净利润） |
| 上缴比例（资源类央企） | 35% |
| 上缴比例（公益类央企） | 15% |
| 建筑央企适用 | **一般竞争类 → 30%** |
| 上缴时间 | 次年（按上年度净利润计算） |
| 编报时间 | 每年 5-6 月 |

## 一、国有资本经营预算的基本概念

国有资本经营预算是国家以所有者身份从央企取得国有资本收益的预算制度。

```
央企净利润 → 提取10%法定盈余公积 → 可供分配利润
  → 按30%上缴国有资本收益（进入国有资本经营预算）
  → 剩余70%留存企业（补充资本金/再投资）
```

## 二、建筑央企的上缴计算

### 基础计算

```
应缴国有资本收益 = 归属母公司股东的净利润 × 30%

注意：
- 归属母公司股东的净利润 ≠ 合并净利润
- 合并净利润包含了少数股东损益——少数股东的利润不上缴国家
- 计算基础 = 合并净利润 − 少数股东损益 = 归母净利润
```

### 计算示例

```
某建筑央企集团：
  合并净利润：300 亿元
  少数股东损益：50 亿元
  归母净利润：250 亿元

  应缴国有资本收益：250 × 30% = 75 亿元
  留存集团：250 × 70% = 175 亿元
```

## 三、利润上缴的资金安排

### 资金筹备

财务部门需在上缴前筹措足额资金：

| 资金来源 | 说明 |
|---------|------|
| 经营积累 | 各子企业留存收益中的可分配部分 |
| 资金集中调度 | 通过资金池归集各子企业的可分配资金 |
| 外部融资（应急） | 如资金不足可短期借款周转（不推荐） |

### 资金流路径

```
各子企业 → 上划利润至集团
  ↓
集团归集 → 向财政上缴
  ↓
财政部 → 纳入国有资本经营预算收入
  ↓（部分返还）
国有资本经营预算支出 → 资本金注入/政策性补贴（特定行业）
```

## 四、编报流程

| 时间 | 步骤 |
|------|------|
| 4月30日前 | 上市公司年报披露（归母净利润确定） |
| 5月 | 集团编制国有资本收益申报表 |
| 5-6月 | 报 SASAC 审核 |
| 6-7月 | SASAC 下达收益收缴通知 |
| 7-8月 | 集团向财政部缴款 |

## 五、与年度决算的关系

国有资本经营预算的数据基础是**经审计的年度财务决算**：

| 数据来源 | 说明 |
|---------|------|
| 归母净利润 | 审计后的合并利润表 |
| 法定盈余公积 | 审计后按 10% 计提 |
| 可供分配利润 | 审计后数据 |

决算报表中有专门的附表用于填报国有资本经营预算相关数据。

## 六、建筑央企的利润分配策略

### 上缴 vs 留存

| 考量 | 倾向 |
|------|------|
| 集团有重大投资需求（PPP/海外并购） | 留存更多（但不能低于 30%） |
| 集团现金流充裕 | 正常上缴 30% |
| SASAC 有明确的"利润上缴"考核 | 确保足额上缴 |

### 对子企业的影响

集团上缴 30% 归母净利润的压力，会传递到各子企业：
- 各子企业需按比例上划利润给集团
- 子企业自身的新项目投资需求 vs 上划利润可能存在矛盾
- 需要在"子企业发展"和"集团利润上缴义务"之间平衡

## 七、自检清单

1. [ ] 归母净利润是否基于审计后的数据？
2. [ ] 上缴比例是否正确（建筑央企 30%）？
3. [ ] 上缴金额 = 归母净利润 × 30%？
4. [ ] 资金是否在上缴截止日前安排到位？
5. [ ] 各子企业的利润上划是否与集团上缴匹配？

## 免责声明

国有资本经营预算的具体比例以 SASAC 每年通知为准。不同行业和战略定位的央企上缴比例可能不同。
