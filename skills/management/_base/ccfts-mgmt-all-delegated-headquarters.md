# 代局指管理模式

> **SLOT 说明**：代局指是建筑央企的常见管理模式。适用于 {ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 角色 | 职责 |
|------|------|
| 工程局 | 合同签约主体、对外负责、收取管理费 |
| 代局指（代局指挥部） | 局授权管理项目、协调各工程公司、不承担经济盈亏 |
| 工程公司（第一责任人） | 实际施工主体、承担经济盈亏、缴纳管理费 |

## 一、代局指模式的本质

### 为什么需要代局指

大型项目（尤其铁路/城轨/水利等线性工程）通常由工程局中标，但工程局自身不直接施工，而是将工程分配给下属各工程公司。代局指就是工程局派驻项目的"管理代理"。

```
业 主
  │
  ├── 工程局（合同签约方、对业主负总责）
  │     │
  │     └── 代局指（局授权管理项目部）
  │           │
  │           ├── 工程公司 A（区段1、经济责任人）
  │           ├── 工程公司 B（区段2、经济责任人）
  │           └── 工程公司 C（区段3、经济责任人）
```

### 职权划分

| 事项 | 工程局 | 代局指 | 工程公司 |
|------|--------|--------|---------|
| 合同签约 | ✅ | ❌ | ❌ |
| 对业主结算 | ❌ | ✅（代局签认） | ❌ |
| 内部责任划分 | ❌ | ✅（协调/分配） | ❌ |
| 施工组织 | ❌ | ❌ | ✅ |
| 成本控制 | ❌ | ❌（监督） | ✅（主责） |
| 盈亏承担 | ❌ | ❌ | ✅ |

## 二、管理费机制

### 管理费收取

工程局向各工程公司（区段）收取管理费：

```
工程公司管理费 = 该区段合同金额 × 管理费率

管理费率：通常 1-3%（按项目规模和复杂程度约定）
```

### 管理费的使用

| 用途 | 比例 |
|------|------|
| 代局指日常运转（人员薪酬、差旅、办公） | 30-50% |
| 局总部管理费用分摊 | 30-40% |
| 项目协调/不可预见费 | 10-20% |

### 管理费的会计处理

```
工程局收到管理费：
  借：银行存款 / 内部往来              XXX
    贷：其他业务收入—管理费收入（6051） XXX

工程公司缴纳管理费：
  借：合同履约成本—间接费—管理费（5601） XXX
    贷：银行存款 / 内部往来              XXX
```

## 三、经济责任划分

### 盈亏归工程公司

代局指模式的核心原则：**经济盈亏全部归工程公司，代局指和工程局不承担经济责任**。

| 责任 | 承担方 |
|------|--------|
| 成本超支 | 工程公司 |
| 材料价格上涨 | 工程公司（除非合同约定调差由业主承担） |
| 工期延误罚款（工程公司原因） | 工程公司 |
| 安全质量事故 | 工程公司 |
| 对业主的索赔收益 | 工程公司（管理费另计） |

### 代局指的考核

代局指不承担经济盈亏，但其考核指标包括：
- 项目整体进度达标率
- 对业主计量结算的及时性
- 各工程公司之间协调的有效性
- 管理费使用的合规性

## 四、总分包结算——代局指模式下的特殊性

### 对业主结算（代局指代局签认）

```
业主→代局指：按月/按进度确认工程计量
  借：合同结算—价款结算（5801）        XXX
    贷：合同结算—收入结转（5801）       XXX
```

### 内部分劈（代局指→各工程公司）

代局指将业主确认的计量金额按合同比例**分劈**给各工程公司：

```
区段 A（工程公司 A）：40%
区段 B（工程公司 B）：35%
区段 C（工程公司 C）：25%
```

## 五、常见管理问题

| 问题 | 表现 | 改进 |
|------|------|------|
| 代局指"权大责小" | 协调不力但工程公司承担后果 | 明确代局指考核指标+问责 |
| 管理费挪用 | 管理费不用于项目运转 | 管理费专户管理 |
| 分劈不均 | 区段比例分配争议 | 合同明确约定分劈比例 |
| 经济责任不清 | 局和工程公司对亏损互相推诿 | 经济责任书明确 |

## 六、自检清单

1. [ ] 管理费率是否在合同中明确约定？
2. [ ] 代局指与各工程公司的经济责任是否清晰划分？
3. [ ] 管理费是否专款专用？
4. [ ] 对业主结算→内部分劈的流程是否顺畅？
5. [ ] 代局指的考核指标是否覆盖协调和管理两个维度？
