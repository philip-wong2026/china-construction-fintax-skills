# 施工企业内部控制框架

> **SLOT 说明**：本文件为施工企业内控通用框架。适用于 {ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 内控要素 | 施工企业核心应用 |
|---------|---------------|
| 不相容职务分离 | 投标报价 vs 合同签订 / 采购申请 vs 采购审批 / 工程计量 vs 结算审批 |
| 授权审批 | 资金支付分级授权 / 合同签署限额 / 变更签证审批 |
| 财产保护 | 材料验收/领用/盘点 / 设备台账 / 银行账户管控 |
| 预算控制 | 责任成本预算→实际对比→偏差分析 |
| 绩效考评 | 项目利润考核 / 回款率 / "两金"压降 |

## 一、不相容职务分离——施工企业关键分离

| 业务环节 | 必须分离的职务 | 违反后果 |
|---------|--------------|---------|
| 材料采购 | 申请 vs 审批 vs 验收 vs 付款 | 虚假采购、回扣 |
| 分包管理 | 队伍选择 vs 合同签订 vs 结算审批 | 利益输送 |
| 工程计量 | 现场计量 vs 结算审核 vs 付款 | 虚报工程量 |
| 资金管理 | 出纳 vs 记账 vs 对账 | 挪用资金 |
| 合同管理 | 投标报价 vs 合同签订 vs 履约管理 | 围标/合同陷阱 |

## 二、授权审批——施工企业分级授权体系

### 资金支付审批

| 金额 | 审批层级 |
|------|---------|
| ≤ 5 万元 | 项目经理 |
| 5-50 万元 | 工程公司总经理 |
| 50-500 万元 | 工程局分管领导 |
| > 500 万元 | 工程局总经理/董事长 |

### 合同签署限额

| 合同类型 | 项目部 | 工程公司 | 工程局 |
|---------|--------|---------|--------|
| 分包合同 | ≤ 500 万 | 500-5,000 万 | > 5,000 万 |
| 材料采购 | ≤ 100 万 | 100-1,000 万 | > 1,000 万 |
| 设备租赁 | ≤ 50 万 | 50-500 万 | > 500 万 |

## 三、财产保护——施工企业资产管理

### 材料管理

| 环节 | 控制措施 |
|------|---------|
| 进场验收 | 数量 + 质量 + 规格 三方验收（施工/物资/监理） |
| 领用出库 | 限额领料单 → 按责任成本预算控制 |
| 月末盘点 | 实地盘点 → 与账面对比 → 差异分析 |
| 废料处理 | 招标 or 询价 → 收入入账 |
| 甲供材 | 单独建账 → 与业主定期对账 |

### 机械设备管理

| 环节 | 控制措施 |
|------|---------|
| 购置/租赁决策 | 经济比选（购置 vs 租赁的净现值比较） |
| 台班记录 | 操作人员填写 → 现场管理人员审核 |
| 维修保养 | 计划性保养 + 故障维修记录 |
| 闲置设备 | 上报调拨 or 对外出租 or 处置 |

## 四、预算控制——责任成本管理

施工企业以"责任成本预算"为核心进行成本控制（详见 `ccfts-mgmt-project-unit-cost-control`）。

| 预算类型 | 编制频率 | 控制对象 |
|---------|---------|---------|
| 标前预算 | 投标前 | 投标报价决策 |
| 责任成本预算 | 中标后/开工前 | 项目部考核基准 |
| 月度/季度预算 | 每月/季 | 短期控制 |
| 竣工决算 | 竣工后 | 最终考核 |

## 五、内控评价与报告

### 内控缺陷等级

| 等级 | 定义 | 施工企业示例 |
|------|------|------------|
| 重大缺陷 | 可能导致财务报表重大错报 | 虚构工程量/收入 |
| 重要缺陷 | 可能导致较大错报但不到"重大" | 材料采购审批缺失 |
| 一般缺陷 | 其他 | 台账记录不完整 |

### 年度内控自评报告

央企须每年编制内控自评报告，随年度决算一并上报 SASAC。

## 六、自检清单
