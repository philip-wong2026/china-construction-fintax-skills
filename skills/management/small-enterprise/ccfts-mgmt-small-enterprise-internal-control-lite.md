# 小型施工企业最小合规内控

## 快速参考

| 内控要素 | 大型央企 | 小型企业（精简版） |
|---------|---------|-----------------|
| 不相容职务分离 | 多人多岗 | 最少 2 人（钱账分离） |
| 授权审批 | 多级审批 | 老板审批 + 财务复核 |
| 资金管控 | 资金池归集 | 个人卡 vs 对公账户分离 |
| 材料管理 | 限额领料+盘点 | 采购审批+月末盘点 |
| 合同管理 | 法务审核+多级审批 | 老板审核+关键条款检查 |

## 一、最小人员配置

小型施工企业通常人员有限（10-50 人），无法做到完全的不相容职务分离。以下是**最低要求**：

| 必须分离 | 说明 |
|---------|------|
| 钱账分离 | 出纳（管钱）≠ 记账（管账）——**绝对不能一人兼任** |
| 采购与验收 | 采购员 ≠ 验收员（至少不同人签字） |
| 投标与合同 | 投标报价人 ≠ 合同签订人（最低要求：老板审批合同） |

如果只有 2-3 个管理人员：
- 老板：审批所有支出 + 签订合同
- 财务（1人）：记账 + 报税 + 保管印章（兼任出纳时需老板定期检查银行流水）
- 现场管理员（1人）：材料验收 + 进度记录

## 二、资金管控——小型企业最容易出问题的地方

### 铁律

| 铁律 | 原因 |
|------|------|
| **对公账户收付工程款** | 个人卡收工程款 = "小金库" = 严重违规 |
| **所有支出有发票** | 无票支出 = 不能税前列支 + 25% CIT 损失 + 税务风险 |
| **老板个人消费不进公司账** | 公私不分 → 税务稽查重点 |
| **每月核对银行流水** | 防止挪用资金 |

### 简易资金日报

```
昨日余额 + 本日收入 − 本日支出 = 本日余额

收入：工程款到账、借款到账
支出：材料款、分包款、工资、税费
```

## 三、材料管理——小型企业简化版

| 环节 | 最小措施 |
|------|---------|
| 采购申请 | 施工员提需求 → 老板审批 |
| 进场验收 | 施工员 + 材料员（or 老板） 签字验收 |
| 领用 | 谁用谁签字（领料单） |
| 盘点 | 月末一次（老板 + 施工员） |

不需要复杂的限额领料系统，但必须有**签字记录**。

## 四、合同管理——小型企业的关键条款

小型企业可能无法负担法务审核，但老板应亲自检查：

| 必须检查的条款 | 注意 |
|--------------|------|
| 工程款支付节点 | 是否明确？是否有预付款？ |
| 变更/签证程序 | 是否必须先批准后施工？ |
| 质保金比例 | 是否 ≤ 3%？ |
| 违约责任 | 工期罚款是否合理？ |
| 争议解决 | 仲裁 or 诉讼？管辖地？ |
| VAT 税率 | 发票类型和税率是否明确？ |

## 五、税务最小合规

| 事项 | 频率 | 外包 or 自己做 |
|------|------|-------------|
| VAT 申报 | 月/季 | 可外包给代账公司 |
| CIT 预缴 | 季 | 同上 |
| CIT 汇算清缴 | 年 | 建议找税务师 |
| 个税（工资） | 月 | 同上 |
| 发票管理 | 日常 | 自己（老板 or 财务） |

## 六、自检清单

1. [ ] 出纳和记账是否由不同人负责？
2. [ ] 工程款是否全部通过对公账户收付？
3. [ ] 所有支出是否都有发票？
4. [ ] 老板个人消费是否与公司账严格分离？
5. [ ] 是否每月核对银行流水？
6. [ ] 合同关键条款是否经老板审批？

## 免责声明

小型企业的内控设计需在"合规"和"成本"之间取得平衡。本文件为最低合规要求，具体实施因企业实际情况而异。
