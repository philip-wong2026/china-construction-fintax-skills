# 项目部工程分包管理

## 快速参考

| 分包类型 | 典型内容 | 资质要求 |
|---------|---------|---------|
| 专业分包 | 地基/钢结构/机电/装修 | 相应专业承包资质 |
| 劳务分包 | 提供劳务作业人员 | 施工劳务资质 |
| 设备租赁+操作 | 带操作人员的设备租赁 | 按建筑服务（视为分包） |

## 一、分包商准入——资质审查

### 必须核查的资质

| 核查项 | 依据 |
|--------|------|
| 营业执照 | 经营范围含所分包工程 |
| 建筑业企业资质证书 | 资质等级与工程规模匹配 |
| 安全生产许可证 | 在有效期内 |
| 项目负责人资格 | 建造师/安全B证 |
| 近三年业绩 | 同类工程经验 |
| 信用状况 | 失信被执行人/行政处罚 |

### 禁止作为分包商的情形

- 个人（无资质自然人）→ 严禁
- 资质等级不够 → 严禁
- 已被列入"黑名单" → 严禁
- "转包"（将全部工程分包） → **法律严禁**

## 二、分包合同关键条款

| 条款 | 要点 |
|------|------|
| 工程范围 | 明确分包的具体工程内容（不得模糊） |
| 合同价款 | 固定总价 or 单价 + 调价机制 |
| 结算方式 | 按月进度/按节点/竣工一次 |
| 质保金 | 比例（通常 3-5%）、缺陷责任期 |
| 安全生产 | 安全责任划分、安全资金拨付 |
| 农民工工资 | 专户管理、总包代发 or 分包自付 |
| 违约责任 | 工期延误/质量不合格的罚则 |
| 甲供材 | 甲供材清单、超耗扣款 |

## 三、分包进度计量

### 月度计量流程

```
分包商提交月度工程量确认单
  ↓
现场技术员核实（含监理确认）
  ↓
计划合同部核算金额
  ↓
物资部核扣甲供材
  ↓
生成分包结算单 → 双方签字确认
```

### 不得结算的情形

- 未经监理/业主验收合格的工程量
- 超出分包合同范围的工程量（无补充协议）
- 质量不合格且未返修的部分
- 甲供材超耗未扣回

## 四、农民工工资专户管理

### 法规要求

《保障农民工工资支付条例》（2020年5月1日施行）的核心要求：

| 要求 | 说明 |
|------|------|
| 工资专户 | 施工总承包单位须开设农民工工资专用账户 |
| 总包代发 | 鼓励总承包单位通过专户代发 |
| 实名制管理 | 农民工进场须实名登记+签订劳动合同 |
| 工资保证金 | 总承包单位须缴存工资保证金 |
| 按月足额 | 工资须按月足额发放，不得拖欠 |

### 总包代发流程

```
分包商提交月度工资清单（农民工签字确认）
  ↓
总包项目部审核（与实名制台账核对）
  ↓
总包通过工资专户直接发放至农民工个人银行卡
  ↓
发放记录留存（银行回单+农民工签到表）
```

**关键**：总包代发≠总包垫付——总包从应付分包款中扣除代发工资。

## 五、分包结算的税务管理

| 事项 | 要求 |
|------|------|
| 增值税发票 | 分包商必须提供增值税发票（专票或普票）方可付款 |
| 跨区域预缴 | 分包商也需在项目所在地预缴 VAT |
| 印花税 | 分包合同万分之三（与总包各自缴纳） |
| 预提税 | 不适用（国内分包） |

## 六、常见分包风险

| 风险 | 后果 | 防范 |
|------|------|------|
| 分包商无资质 | 行政处罚+合同无效 | 严格资质审查 |
| 转包（分包再转包） | 连带责任 | 合同禁止转包 |
| 农民工工资拖欠 | 总包垫付+声誉损失 | 专户管理+总包代发 |
| 分包商虚报工程量 | 成本超支 | 现场核实+监理确认 |
| 分包商安全事故 | 总包连带责任 | 安全协议+安全培训 |

## 七、自检清单

1. [ ] 分包商资质是否齐全且有效？
2. [ ] 分包合同是否明确了工程范围和价款？
3. [ ] 月度结算是否经过完整审核流程？
4. [ ] 农民工工资专户是否开设并正常运作？
5. [ ] 分包商是否提供了增值税发票？
6. [ ] 有无转包行为？

## 免责声明

分包管理的具体流程以企业的工程管理制度为准。农民工工资管理以《保障农民工工资支付条例》为准。
