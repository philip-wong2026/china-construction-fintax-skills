# 项目部清收清欠管理

## 快速参考

| 阶段 | 工具 | 目标 |
|------|------|------|
| 台账管理 | 应收账款台账（按项目/业主/账龄） | 全面掌握债权底数 |
| 常规催收 | 催收函/电话/会议纪要 | 维持正常回款节奏 |
| 重点清收 | 现场催收+高层对接 | 攻克难点项目 |
| 法律手段 | 律师函→诉讼→财产保全 | 强制回款 |
| 终极手段 | 以资抵债→债务重组→核销 | 止损出表 |

## 一、应收账款台账

### 台账字段

| 字段 | 用途 |
|------|------|
| 项目名称 | |
| 业主名称 | |
| 合同金额 | |
| 累计结算金额 | |
| 累计收款金额 | |
| **应收账款余额** | = 累计结算 − 累计收款 |
| 其中：未到期 | 按合同付款节点尚未到期的 |
| 其中：已到期未付 | **这是催收重点** |
| 其中：质保金 | 见质保金文件 |
| 账龄 | 1年以内/1-2年/2-3年/3年以上 |
| 业主类型 | 政府/城投/国企/民企 |
| 催收责任人 | |
| 最近催收日期 | |
| 预计回收时间 | |

### 预警指标

| 指标 | 预警阀值 | 行动 |
|------|---------|------|
| 账龄 > 6 个月（已到期未付） | 黄色预警 | 发送催收函 |
| 账龄 > 1 年 | 橙色预警 | 现场催收 + 高层对接 |
| 账龄 > 2 年 | 红色预警 | 法律评估 + 考虑诉讼 |
| 账龄 > 3 年 | 黑色 | 考虑核销 or 以资抵债 |
| 业主出现重大负面（破产/失信） | 任何时点 | 立即启动法律保全 |

## 二、分级催收策略

### 常规催收（账龄 < 6 个月）

| 手段 | 频率 |
|------|------|
| 电话/微信沟通 | 每周 |
| 发送催收函（盖章） | 每月/每季 |
| 结算/验收时当面沟通 | 每次节点 |

### 重点清收（账龄 6 个月 - 2 年）

| 升级手段 | 说明 |
|---------|------|
| 现场催收 | 财务+商务人员到业主单位登门催收 |
| 高层对接 | 工程公司/局领导出面与业主高层沟通 |
| 暂停后续项目投标 | 以不投后续项目作为谈判筹码（需审慎） |
| 发律师函 | 正式法律文书，施压效果 |

### 法律手段（账龄 > 2 年 + 催收无效）

| 阶段 | 行动 |
|------|------|
| 诉前财产保全 | 冻结业主银行账户/查封资产（需提供担保） |
| 提起诉讼/仲裁 | 按合同约定管辖 |
| 强制执行 | 判决后申请法院执行 |
| 列入失信被执行人 | 对业主施压（但对政府/城投效果有限） |

**政府项目的特殊性**：
政府/城投客户不能轻易起诉（影响后续业务关系），但可通过以下方式施压：
- 向上级主管部门/财政部门反映
- 联合其他施工企业（该业主的多个债权人）共同施压
- 通过人大/政协提案途径

## 三、以资抵债

当业主确实无力支付工程款时，可能接受以资抵债：

### 常见抵债资产

| 资产类型 | 风险 |
|---------|------|
| 房产（住宅/商业） | 估值虚高、变现困难 |
| 土地 | 闲置风险、税费成本 |
| 股权 | 估值不确定、退出困难 |
| 经营权（收费公路/停车场） | 未来收益不确定 |

### 以资抵债决策流程

```
业主提议以资抵债
  ↓
评估资产评估（第三方评估机构）
  ↓
与应收账款金额比较 → 是否足额覆盖？
  ├── 是 → 考虑接受
  └── 否 → 差额部分继续追偿 or 计提减值
  ↓
内部审批（按资产金额分级审批）
  ↓
签订以资抵债协议 + 资产过户
  ↓
后续管理（出租/出售/持有）
```

## 四、坏账核销

当穷尽所有催收手段后仍无法收回，可申请坏账核销。

### 核销条件

| 条件 | 说明 |
|------|------|
| 已穷尽所有催收手段 | 催收函+律师函+诉讼（or 明确不可诉） |
| 有证据证明债务人无力偿还 | 破产/注销/失信/法院终结执行裁定 |
| 内部审批 | 按金额分级审批 |
| 已足额计提坏账准备 | 核销金额 ≤ 已计提的坏账准备 |
| SASAC 备案（央企） | 大额核销须报 SASAC 备案 |

### 核销后的管理

"账销案存"——核销后不放弃追偿权：
- 继续保留债权资料
- 债务人有偿还能力时继续追偿
- 追偿收回的款项计入营业外收入

## 五、自检清单

1. [ ] 应收账款台账是否完整（按项目/业主/账龄）？
2. [ ] 账龄 > 1 年的应收款是否有明确的催收计划？
3. [ ] 账龄 > 3 年的应收款是否已评估坏账计提和核销？
4. [ ] 有无业主已破产/失信但未做任何处理的应收款？
5. [ ] 以资抵债的资产评估是否由第三方机构完成？
6. [ ] 核销后的债权是否"账销案存"继续追偿？

## 免责声明

清收清欠的具体措施因企业管理制度和债务人的实际情况而异。法律手段建议咨询专业律师。
