# 项目部成本控制

## 快速参考

| 控制维度 | 方法 | 频率 |
|---------|------|------|
| 责任成本预算 | 中标后/开工前编制 | 一次性（+ 季度更新） |
| 材料费控制 | 限额领料 + 价差量差分析 | 月度 |
| 机械费控制 | 台班记录 + 利用率分析 | 月度 |
| 分包费控制 | 分包结算审核 | 按月进度 |
| 间接费控制 | 预算包干 | 月度 |
| 偏差分析 | 实际 vs 责任预算 → 原因 → 对策 | 月度 |

## 一、责任成本预算编制

### 编制依据

| 来源 | 内容 |
|------|------|
| 中标合同 | 合同清单、单价、工程量 |
| 企业内部定额 | 各工序的企业内部成本标准 |
| 施工组织设计 | 施工方案、资源配置 |
| 市场询价 | 材料/设备/分包的市场价格 |

### 预算分解

```
项目总收入（合同价 − 管理费 − 税金）
  │
  ├── 人工费预算
  ├── 材料费预算（分主材/辅材/周转材料）
  ├── 机械费预算（自有/租赁）
  ├── 分包费预算（专业分包/劳务分包）
  ├── 间接费预算（管理人员/临建/办公/差旅）
  └── 风险预留金（不可预见费）
```

## 二、材料费控制

### 量差控制

```
量差 = 实际用量 − 定额用量（按实际完成工程量 × 定额消耗）

量差 > 0 → 超耗 → 追查原因（浪费/偷盗/定额不合理）
量差 < 0 → 节约 → 总结经验（是否有偷工减料风险？）
```

### 价差控制

```
价差 = 实际采购价 − 预算价

价差 > 0 → 超预算 → 市场涨价/采购渠道有问题
价差 < 0 → 节约 → 确认质量是否合格
```

### 限额领料制度

```
每月根据施工计划计算各种材料的限额用量
  → 签发限额领料单
  → 仓库按限额发料
  → 超额领料须项目经理审批
  → 月末盘点 + 分析超耗原因
```

## 三、机械费控制

| 控制点 | 方法 |
|--------|------|
| 台班真实性 | 操作人员填写→现场管理人员审核→不得虚报台班 |
| 利用率 | 实际工作台班 / 在勤台班 ≥ 80% |
| 自有 vs 租赁 | 月度分析：若闲置率高→考虑退租 or 调拨 |
| 油料消耗 | 按台班定额核算→超耗追查 |

## 四、分包费控制

### 分包结算审核

```
分包商提交月度结算单
  ↓
现场工程师审核（工程量是否属实）
  ↓
计划合同部审核（单价是否按分包合同）
  ↓
物资部审核（甲供材扣回）
  ↓
财务部审核（预付款扣回、质保金扣留）
  ↓
项目经理审批
```

### 分包成本的红线

- 不得超分包合同金额结算（超额须签订补充协议）
- 不得对未验收的分包工程全额结算
- 不得对未提供发票的分包商付款（VAT 合规）

## 五、月度成本分析会

### 标准议程

1. **上月成本执行通报**：实际 vs 责任预算、偏差率
2. **偏差原因分析**：量差（材料/人工）、价差、进度偏差
3. **降本增效措施跟踪**：上月提出的措施执行情况
4. **本月控制重点**：预计超支项目、拟采取对策
5. **责任追究**：对明显超支且无合理解释的责任人问责

### 偏差分析模板

| 成本项目 | 责任预算 | 实际 | 偏差 | 偏差率 | 原因 | 对策 |
|---------|---------|------|------|--------|------|------|
| 钢材 | 500万 | 530万 | +30万 | +6% | 市场涨价 | 加快后续采购、锁价 |
| 商品混凝土 | 200万 | 215万 | +15万 | +7.5% | 超耗（浪费） | 加强现场管理 |
