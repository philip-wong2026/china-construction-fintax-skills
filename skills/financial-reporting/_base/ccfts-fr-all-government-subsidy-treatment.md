# 政府补贴/专项资金操作处理

> **SLOT 说明**：本文件定义收到政府补贴后的全流程操作。`{SLOT_XXX}` 由层级覆盖文件填充。

## 快速参考

| 项目 | 说明 |
|---|---|
| 适用主体 | {ENTITY_LABEL}（{ENTITY_ALIAS}） |
| 是否需 SASAC 考核口径调整 | {IS_SOE_GROUP} |
| 核心决策 | 与资产相关 vs 与收益相关 → 总额法 vs 净额法 → SASAC 考核剔除 |

## 一、补贴分类决策树

```
收到政府拨款/补贴
  │
  ├── 有明确指定用途（购设备/建工程/搞研发）？
  │   ├── YES → 专项资金 → 需专款专用 + 单独核算
  │   └── NO → 一般性补贴 → 继续判断
  │
  ├── 用于购建长期资产？
  │   ├── YES → 与资产相关 → 递延收益 → 分期转入
  │   └── NO → 继续判断
  │
  ├── 补偿已发生的费用/损失？
  │   ├── YES → 与收益相关（已发生） → 直接计入其他收益
  │   └── NO → 继续判断
  │
  └── 补偿未来期间的费用？
      └── YES → 与收益相关（未来） → 递延收益 → 分期转入
```

## 二、分类操作流程

### Step 1：收到款项——确认资金来源

| 来源 | 典型文件 | 操作 |
|------|---------|------|
| 发改委 | 投资补助批复文件 | 核对批复金额、用途、期限 |
| 科技厅/局 | 科研项目立项书 | 区分科研经费（专款专用）和奖励 |
| 人社局 | 稳岗补贴通知 | 通常为与收益相关（已发生） |
| 财政局 | 专项资金拨付文件 | **重点**：确认是否专款专用、是否需单独核算 |

### Step 2：分类判断——确定会计处理

按决策树判断后，编制会计处理备忘录：

```
【政府补贴会计处理备忘录】
- 补贴名称：[]
- 资金来源：[发改委/科技厅/财政局/人社局/其他]
- 批复文件编号：[]
- 批复金额：[]
- 实际收到金额：[]
- 分类：[与资产相关 / 与收益相关-已发生 / 与收益相关-未来]
- 方法：[总额法 / 净额法]
- 递延收益分摊期限：[]
- 是否专款专用：[是/否]
- 编制人：[]  审核人：[]  日期：[]
```

### Step 3：入账

按分类执行对应分录（详见 `ccfts-intel-mof-government-subsidies` 分录模板）。

**注意**：专项资金必须**单独设置明细科目**核算，不可与自有资金混用。
审计时会逐笔检查专项资金的使用是否符合批复用途。

### Step 4：SASAC 决算填报（央企适用）

{IS_SOE_GROUP}

SASAC 决算报表中对政府补贴有专门的附表：

| 填报项目 | 填报内容 |
|---------|---------|
| 政府补助明细表 | 逐项列出补助名称、金额、分类、计入科目 |
| 其中：与资产相关 | 递延收益期初/本期增加/本期转入其他收益/期末 |
| 其中：与收益相关 | 本期直接计入其他收益的金额 |
| 非经常性损益 | 计入营业外收入的政府补助（非经常性） |

### Step 5：SASAC 考核口径调整

若需剔除一次性/非经营性补贴对利润总额的影响：
```
考核利润总额 = 报表利润总额 − 投资补助（本期计入损益部分） − 一次性专项拨款
```

(以 SASAC 年度决算通知的具体要求为准。)

## 三、常见处理错误

| 错误 | 后果 | 纠正 |
|------|------|------|
| 与资产相关的补贴**一次性**计入其他收益 | 当期利润虚增，后续折旧无递延收益匹配 | 递延收益按资产折旧期分摊 |
| 专项资金**未单独核算** | 审计发现后需整改，严重者收回资金 | 设立专项资金明细科目，单独归集 |
| 与收益相关（未来）**直接**计入其他收益 | 当期利润虚高，未来费用无补偿 | 收到时确认递延收益 |
| 稳岗补贴计入**营业外收入** | SASAC 考核可能剔除（非经常性标记） | 应计入其他收益（6605） |
| 补贴相关的 VAT 未处理 | 少缴 VAT → 税务风险 | 判断是否需要缴纳 VAT |

## 四、审计配合要点

专项资金审计关注：
1. 资金是否**专款专用**（与批复用途一致）
2. 支出凭证是否**单独归档**（不与自有资金支出混放）
3. 项目是否**按期完成**（如有进度要求）
4. 未使用资金是否**如实披露**（含使用限制说明）
5. 形成的资产是否**单独标识**（便于后续审计追踪）

## 五、自检清单

1. [ ] 每笔补贴是否已编制会计处理备忘录？
2. [ ] 与资产相关的是否设立了递延收益分摊计划？
3. [ ] 专款专用的是否设立了单独明细科目？
4. [ ] 是否需要缴纳 VAT？
5. [ ] SASAC 决算报表中政府补助明细是否逐项填写？
6. [ ] SASAC 考核口径是否需要剔除？
7. [ ] 审计所需的专款专用支出凭证是否单独归档？
