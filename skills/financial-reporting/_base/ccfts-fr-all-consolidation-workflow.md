## 一、合并范围判断流程

### 控制三要素（CAS 33）

```
要素 1：权力
  → 拥有对被投资方的现时权力（表决权、合同安排等）
  → 考虑：持股比例、董事会席位、一致行动协议

要素 2：可变回报
  → 因参与被投资方而享有可变回报
  → 考虑：股利、利息、服务费、业绩报酬

要素 3：权力-回报联系
  → 有能力运用权力影响回报金额
  → 考虑：是否为主要责任人 or 代理人

三个要素缺一不可。
```

### 合并范围决策树

```
被投资方
  │
  ├── 母公司持有 >50% 表决权？
  │   ├── YES → 通常合并（除非有证据证明不控制）
  │   └── NO → 继续判断
  │
  ├── 是最大股东 + 其他股东分散？
  │   ├── YES + 能任命多数董事 → 合并
  │   └── NO → 继续判断
  │
  ├── 通过合同安排拥有控制权？
  │   ├── YES → 合并
  │   └── NO → 不合并
  │
  └── 特殊排除：
      ├── 已宣告被清理整顿的子公司 → 不合并
      ├── 已宣告破产的原子公司 → 不合并
      └── 母公司是投资性主体 → 不合并（子公司非投资服务提供者）
```

### PPP 项目公司合并（施工企业特别关注）

国资委要求：如央企是项目公司的**最大实体企业股东**，原则上应**合并**，除非有确凿证据证明不控制。

| 合并 | 不合并 |
|------|--------|
| 持股 >50% + 能主导相关活动 | 共同控制（合营/联营） |
| 是最大股东 + 董事会多数 | 少数股东对关键经营决策有否决权 |
| 全部损益归施工方承担 | 施工方仅为代理人 |

## 二、合并程序

### 5 步标准合并流程

```
第 1 步：统一会计政策
  → 所有纳入合并范围的实体统一使用企业会计准则
  → 境外子企业调整为 PRC GAAP
  → 统一会计期间（全部为 1月1日-12月31日）

第 2 步：汇总
  → 汇总母子公司的资产、负债、权益、收入、费用
  → 逐项加总，不做任何抵销

第 3 步：抵销——权益
  → 母公司长期股权投资 vs 子公司所有者权益
  → 确认少数股东权益
  → 确认商誉或负商誉

第 4 步：抵销——内部交易
  → 内部往来余额抵销（应收 vs 应付）
  → 内部交易抵销（收入 vs 成本）
  → 内部未实现损益抵销（存货、固定资产）
  → 内部现金流抵销

第 5 步：确认少数股东损益
  → 少数股东损益 = 子公司净利润 × 少数股东持股比例
  → 在合并利润表中单独列示
```

### 合并抵销分录模板

#### 权益抵销
```
借：实收资本（子公司）                XXX
    资本公积（子公司）                XXX
    盈余公积（子公司）                XXX
    未分配利润（子公司）              XXX
    商誉（差额，如有）                XXX
  贷：长期股权投资（母公司）          XXX
      少数股东权益                    XXX
```

#### 内部往来抵销
```
借：应付账款 / 其他应付款            XXX
  贷：应收账款 / 其他应收款          XXX
```

```

```

#### 内部债券投资抵销

## 四、合并 vs 不合并的 B/S 影响

| 指标 | 合并 | 不合并（权益法） |
|------|------|-----------------|
| 资产 | 高（含在建PPP项目资产） | 低（仅长期股权投资） |
| 负债 | 高（含项目融资债务） | 低（表外） |
| 收入 | 高（含施工收入） | 低（仅投资收益） |
| 资产负债率 | **高** | **低** |

> 这是施工企业合并范围判断的核心权衡——合并提高资产/收入规模但推高负债率。

## 五、合并范围变更管理

### 禁止随意变更（财会〔2025〕33号）

不得以自愿破产、一致行动协议、董事会席位变动、章程修订等为**唯一依据**调整合并范围。
合并范围变更必须有实质性控制变更的商业理由。

### 变更披露要求
- 变更原因（商业理由）
- 被合并/不再合并的子企业名称
- 对合并报表的影响（资产/负债/收入/利润）
- 上年同期数的重述
