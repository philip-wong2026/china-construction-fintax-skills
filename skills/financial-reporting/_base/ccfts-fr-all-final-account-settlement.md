## 一、竣工结算五阶段

```
阶段 1：送审
  承包方编制竣工结算书 → 提交发包方/审计机构
    ↓
阶段 2：审计审减
  发包方/审计逐项审核 → 提出审减意见 →
  双方核对、争议协商
    ↓
阶段 3：确认与调整
  审定金额确认 → 收入调整（多退少补）→
  争议项处理（计提 or 继续协商）
    ↓
阶段 4：资产转换
  合同资产(1483) → 应收账款(1122)
  同时确认质保金（合同资产—质保金）
    ↓
阶段 5：竣工决算
  编制竣工财务决算表 → 资产移交 →
  项目关闭
```

## 二、阶段 1：送审

### 竣工结算书的构成

| 组成部分 | 内容 | 依据 |
|---------|------|------|
| 合同价 | 中标合同金额 | 施工合同 |
| 变更/签证 | 设计变更、现场签证 | 变更令/签证单 |
| 索赔 | 工期索赔、费用索赔 | 索赔报告及批复 |
| 调价 | 材料调差、政策性调价 | 合同调价条款 |
| 奖罚 | 工期奖罚、质量奖罚 | 合同奖罚条款 |
| **送审总价** | 以上合计 | |

### 送审前的内部核对

在送审前必须完成：
1. 与项目台账核对（合同收入、累计确认收入、累计收款）
2. 与科目余额表核对（6001 收入 / 5801 结算 / 1483 合同资产）
3. 变更签证的完整性检查（有无遗漏？业主是否已签认？）
4. 预估可能被审减的项目及金额

## 三、阶段 2：审计审减——最容易出问题的节点

### 常见审减类型

| 审减类型 | 原因 | 处理方式 |
|---------|------|---------|
| 工程量审减 | 实际工程量少于送审量 | 接受 → 调减收入 |
| 单价审减 | 合同外项目单价争议 | 协商 or 争议 |
| 变更不予认可 | 变更手续不完整 | **重点风险**：补手续 or 争议 |
| 索赔不予认可 | 证据不充分 | 补证据 or 争议 |
| 材料调差争议 | 基准价/调差公式分歧 | 协商 or 争议 |
| 取费标准审减 | 管理费/利润/规费比例争议 | 通常按合同约定 |

### 审减的三种结果

```
审减意见
  ├── 接受审减 → 调减收入（见阶段3）
  ├── 部分接受 → 协商折中 → 接受部分调减
  └── 不接受 → 列为争议项
       ├── 继续协商（可能拖延数月/数年）
       ├── 提请造价鉴定/仲裁
       └── 诉讼（极少，但存在）
```

### 关键会计判断：何时确认审减？

| 时点 | 处理 |
|------|------|
| 审计口头沟通审减意向 | **不调整**（非正式） |
| 收到正式审计报告（征求意见稿） | 评估可能性 → 可能需调整 |
| 双方签字确认审定金额 | **必须调整** |
| 争议项（我方认为不应审减） | 继续挂账，但需评估可收回性 → 可能需计提减值 |

## 四、阶段 3：确认与调整——账务处理

### 3.1 收入调整

```
审定金额 > 已累计确认收入：
  借：合同结算—收入结转（5801）    XXX
    贷：主营业务收入（6001）        XXX
  （或：合同资产 1483 增加）

审定金额 < 已累计确认收入（审减）：
  借：主营业务收入（6001）          XXX
    贷：合同结算—收入结转（5801）   XXX
  （或：合同资产 1483 减少）
```

### 3.2 争议项处理

| 争议金额 | 处理 |
|---------|------|
| 可能性 > 50% 能收回 | 继续保留在合同资产中，不调整 |
| 可能性 20-50% | 保留，但**披露**不确定性 |
| 可能性 < 20% | **计提减值**（1484 合同资产减值准备） |
| 金额重大 + 可能性 < 50% | 可能触发预计负债（若已开票需退票/冲红） |

### 3.3 最易出错的三种情况

**错误 1：提前确认未审定部分的收入**
- 送审价 5000 万，累计确认收入 4800 万（含未签证变更 300 万）
- 审计审减 400 万（其中 250 万为未签证变更）
- **正确**：调减收入 400 万 / **错误**：不调整，等待签证补签

**错误 2：争议部分长期挂账不提减值**
- 争议金额 500 万挂合同资产 3 年未处理
- **正确**：每期评估可收回性，3 年未解决应大幅计提减值

**错误 3：VAT 已开票但后续审减，未及时冲红**
- 送审时已按全额开票缴税，审减 200 万后未开红字发票
- **正确**：审减确认后及时开具红字发票，冲减销项税

## 六、阶段 5：竣工决算

竣工结算 ≠ 竣工决算：
- **结算**：确定工程造价（承包方←→发包方）
- **决算**：确定项目总投（发包方内部 + 向主管部门报送）

施工企业参与的主要是结算，决算由发包方主导。但承包方需配合提供：
- 竣工结算书
- 工程变更/签证汇总
- 材料设备清单
- 质量验收报告

## 七、竣工结算的常见拖延原因与对策

| 拖延原因 | 影响 | 对策 |
|---------|------|------|
| 业主资金不到位 | 不主动推进审计 | 合同约定审计时限 |
| 审计机构人员不足 | 审计周期长 | 提前沟通审计计划 |
| 变更签证手续不全 | 审计不予认可 | **施工过程中及时签证，不留后账** |
| 政府项目"新官不理旧账" | 无人签批 | 保持与业主的持续沟通和发函 |
| 双方对审减金额分歧大 | 反复协商 | 区分"可妥协"和"底线"项目 |

## 八、自检清单

1. [ ] 送审金额与合同及变更签证是否一致？
2. [ ] 审减项目是否逐项核对并分类（接受/协商/争议）？
3. [ ] 争议项是否每期评估可收回性？
4. [ ] 审定金额确认后是否及时调整收入？
5. [ ] 合同资产→应收账款转换是否正确？
6. [ ] VAT 已开票部分审减后是否及时冲红？
7. [ ] 质保金是否按合同比例正确扣留？
8. [ ] 竣工结算拖延是否在报表附注中充分披露？
