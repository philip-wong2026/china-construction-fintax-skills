## 一、合并工作底稿标准格式

### 底稿结构

| 列 | 内容 | 数据来源 |
|---|---|---|
| A | 报表项目 | 快报标准行项名称 |
| B | 母公司 | 母公司单户 TB |
| C-N | 各子公司 | 各子公司单户 TB |
| 汇总数 | Σ(B:N) | = SUM(B:N) |
| 抵销-借方 | 抵销分录借 | 抵销分录汇总 |
| 抵销-贷方 | 抵销分录贷 | 抵销分录汇总 |
| **合并数** | 汇总+借-贷 | 最终合并报表数据 |

### 科目映射（合并特有行项）

合并层面比单户层面多了以下行项：

| 合并特有项目 | 资产负债表位置 | 说明 |
|-------------|---------------|------|
| 商誉 | 非流动资产 | 合并成本 − 可辨认净资产公允价值份额 |
| 少数股东权益 | 所有者权益（末行之前） | 子公司净资产 × 少数股东比例 |
| 少数股东损益 | 利润表（净利润之后） | 子公司净利润 × 少数股东比例 |
| 归属于母公司股东的净利润 | 利润表（末行） | 净利润 − 少数股东损益 |

## 二、标准抵销分录

### 2.1 权益抵销（最重要）

```
场景：母公司持有子公司 X% 股权

借：实收资本（子公司账面）              XXX
    资本公积（子公司账面）              XXX
    盈余公积（子公司账面）              XXX
    未分配利润（子公司期末）            XXX
    商誉（差额）                        XXX
  贷：长期股权投资（母公司账面）        XXX
      少数股东权益（(1-X%) × 子公司净资产） XXX
```

若为负商誉（廉价收购），贷差计入**营业外收入**。

### 2.2 内部债权债务抵销

```
借：应付票据及应付账款                  XXX
    其他应付款                          XXX
    合同负债（内部）                    XXX
  贷：应收票据及应收账款                XXX
      其他应收款                        XXX
      合同资产（内部）                  XXX
```

**施工企业重点**：内部往来（3001）是施工企业最常见的内部债权债务，必须全额抵销。

### 2.3 内部存货/合同资产交易抵销

```
场景：母公司向子公司销售设备，子公司作为合同履约成本

借：营业收入（内部销售收入）            XXX
  贷：营业成本（内部销售成本）          XXX
      合同履约成本（未实现利润）        XXX
      （或：存货、合同资产）
```

**施工企业重点**：
- 工程局向项目部调拨材料的内部利润
- 集团内设备租赁公司的内部租赁收入
- 设计院向工程局收取的内部设计费

### 2.4 内部现金流抵销

```
借：经营活动现金流出（内部采购付款）     XXX
  贷：经营活动现金流入（内部销售收入）   XXX
```

### 2.5 内部债券投资抵销

```
借：应付债券                            XXX
  贷：债权投资                          XXX
借：投资收益（利息收入）                XXX
  贷：财务费用（利息支出）              XXX
```

## 三、PPP 项目公司合并处理（施工企业关键）

### 3.1 PPP 项目公司的特殊性

PPP 项目公司（SPV）通常具有以下特征：
- 高负债率（项目融资，负债 70-80%）
- 资产主要为在建工程/合同资产/无形资产
- 施工企业是最大股东 + EPC 总承包商
- 项目公司利润主要来源于运营期（非建设期）

### 3.2 合并 vs 不合并的定量影响

以典型 PPP 项目（总投 50 亿，资本金 20%，贷款 80%）为例：

| 指标 | 合并 | 不合并（权益法） | 差异 |
|------|------|-----------------|------|
| 资产总额 | +50 亿 | +10 亿（仅长投） | 40 亿 |
| 负债总额 | +40 亿 | 0 | 40 亿 |
| 资产负债率 | ↑↑ | ↓ | 关键差异 |
| 营业收入 | +EPC 施工收入 | 0（建设期） | 全部施工收入 |
| 经营活动现金流 | +EPC 收款 | 0 | 全部 |

### 3.3 国资委合并要求

**国资发财评规**：央企是 PPP 项目公司最大实体企业股东的，原则上应合并。

例外（可证明不控制的情形）：
- 政府方股东对关键经营决策有**实质性否决权**（非保护性权利）
- 施工方仅为 EPC 承包商+财务投资人，不主导项目公司相关活动
- 存在其他共同控制方且决策需全体一致同意

## 四、合并层面特殊处理

### 4.1 "两金"合并计算

SASAC "两金" = 应收账款 + 存货（含合同资产）

合并层面 = Σ 各子企业"两金" − 内部"两金"已抵销部分

**注意**：抵销内部往来后"两金"下降，但不是实质性压降——SASAC 考核区分合并前/合并后。

### 4.2 资产负债率合并计算

合并资产负债率 = 合并负债总额 / 合并资产总额

**建筑央企红线**：原则上不高于 75%（SASAC 差异化考核）。

合并 PPP 项目公司会显著推高资产负债率，这是施工企业合并范围判断的核心矛盾。

### 4.3 少数股东权益在"一利五率"中的应用

- **ROE 分母**：含少数股东权益（合并口径）
- **资产负债率分子**：含全部负债（含 PPP 项目融资）
