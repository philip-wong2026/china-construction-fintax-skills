## 一、质保金的本质

质保金是发包方（业主）从应付工程款中扣留的一定比例款项，作为承包方履行缺陷修复义务的担保。

### 法律依据

| 法规 | 要点 |
|------|------|
| 《建设工程质量保证金管理办法》（建质〔2017〕138号） | 质保金总预留比例**不得高于**工程价款结算总额的 **3%** |
| 《关于印发建设工程质量保证金管理办法的通知》 | 缺陷责任期一般为 **1 年**，最长不超过 **2 年** |
| 2026 增值税法 | 纳税义务发生时间：开票、收款、合同约定收款日**最早者** |

### 与履约保证金/投标保证金的区别

| 类型 | 目的 | 时点 | 比例 |
|------|------|------|------|
| 投标保证金 | 保证不撤标 | 投标时 | ≤ 2% 估算价 |
| 履约保证金 | 保证履约 | 签订合同时 | ≤ 10% 合同价 |
| **质保金** | 保证缺陷修复 | **竣工结算时扣留** | ≤ 3% 结算价 |

> 三者是不同阶段的担保工具，不可混用。本项目技能聚焦质保金。

## 二、质保金确认的四个阶段

### 阶段 1：竣工结算——初始确认

```
工程竣工验收通过
  ↓
按结算价 × {RETENTION_RATE_TYPICAL} 确认质保金
  ↓
借：合同资产—质保金（1483）    XXX
  贷：工程结算（5801—价款结算）   XXX
  （或：合同结算—价款结算）
```

**关键判断**：质保金属于合同资产还是长期应收款？

| 条件 | 分类 | 科目 |
|------|------|------|
| 缺陷责任期 ≤ 1 年 + 无条件收款权 | 合同资产 | 1483 |
| 缺陷责任期 > 1 年 + 含重大融资成分 | 长期应收款 | 1531 |
| 业主已出具质保金证书但未付款 | 应收账款 | 1122 |

### 阶段 2：质保期——持有期间

质保期内（通常 {WARRANTY_PERIOD_TYPICAL}），质保金处于"有条件"状态：
- 若发生缺陷 → 业主有权扣减质保金用于维修
- 若未发生缺陷 → 到期全额返还

**每期末需评估**：
1. 已发生的缺陷修复义务是否足额计提？
2. 质保金的可收回性是否发生减值？

```
若预计需承担缺陷修复费用：
  借：合同履约成本 / 营业成本    XXX
    贷：预计负债（2801）          XXX

若质保金回收存在不确定性：
  借：信用减值损失（6702）        XXX
    贷：合同资产减值准备（1484）  XXX
      （或：坏账准备 1231-02）
```

### 阶段 3：缺陷责任期满——转应收

```
缺陷责任期满，业主确认无未修复缺陷
  ↓
质保金从"有条件"变为"无条件"收款权
  ↓
借：应收账款—质保金（1122）      XXX
  贷：合同资产—质保金（1483）    XXX
```

**施工企业现实问题**：政府项目（尤其市政/地铁）的质保金到期后，业主往往以"财政资金未到位""审计未完成"等理由拖延支付，质保金实际回收周期可能远超 2 年。

## 三、坏账计提——施工企业最大的质保金风险

### 信用风险特征

政府/城投客户的质保金具有"低违约率、高拖欠率"的特征：
- 不会"不还"，但会"无限期拖延"
- 账龄 2-3 年的质保金应收款在施工企业中非常普遍
- 若按标准账龄模型计提，坏账准备可能大幅侵蚀利润

### 预期信用损失模型（CAS 22 简化版）

| 账龄 | 政府/城投类客户 | 商业类客户 |
|------|----------------|-----------|
| 1 年以内 | 1-2% | 3-5% |
| 1-2 年 | 5-10% | 10-20% |
| 2-3 年 | 15-25% | 30-50% |
| 3 年以上 | 30-50% | 50-100% |

**施工行业特殊性**：如能证明与业主保持正常业务关系（后续项目持续中标），可以降低计提比例，但需在报表附注中披露依据。

## 四、VAT 处理

### 纳税义务发生时间（2026 增值税法）

质保金的 VAT 纳税义务时点存在争议：

| 情景 | 纳税时点 | 依据 |
|------|---------|------|
| 结算时已开全额发票（含质保金） | **开票日** | 最早原则 |
| 结算时未开质保金部分发票 | 质保期满开票日 or 实际收款日 | 孰早 |
| 质保金被业主扣减（用于维修） | 扣减部分**不再产生纳税义务** | 实际收款原则 |

### 实际操作建议

```
结算时：全额开票 → 全额缴纳 VAT（质保金部分需垫税）
  或
结算时：部分开票（不含质保金） → 质保金部分待到期再开票
```

**现金流影响**：全额开票意味着质保金部分（{RETENTION_RATE_TYPICAL}）的 VAT（9%）需要提前垫付，
在质保金实际回收前形成资金占用。

## 五、不同项目模式的特殊处理

### 纯施工合同
- 质保金 = 结算价 × 3%
- 标准四阶段处理

### EPC 总承包
- 设计、采购、施工各有质保条款
- 设备采购质保期可能与土建质保期不同（设备通常更长）
- 需拆分各组成部分的质保金比例

### PPP/BOT 项目
- 建设期完成后进入运营期，质保金处理较为复杂
- 运营期的绩效评价可能影响质保金回收比例
- 需结合《企业会计准则解释第14号》
