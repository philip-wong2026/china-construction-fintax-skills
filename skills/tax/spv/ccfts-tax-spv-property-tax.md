# SPV 项目公司房产税与土地使用税

## 快速参考

| 税种 | 税率 | 计税基础 |
|------|------|---------|
| 房产税（自用） | **1.2%**（年税率，按季/年缴纳） | 房产原值 × (1 − 扣除比例) |
| 房产税（出租） | **12%**（年税率） | 租金收入 |
| 城镇土地使用税 | 各地不同（每平米几元到几十元） | 实际占用土地面积 |

## 一、SPV 何时开始缴纳房产税

### 触发时点

| 资产状态 | 房产税 | 土地使用税 |
|---------|--------|-----------|
| 建设期（在建工程） | ❌ 不缴 | ✅ 取得土地后即需缴纳 |
| 竣工验收后（固定资产） | ✅ 从验收次月起 | ✅ |
| 投入使用但未验收 | ✅ 从使用次月起 | ✅ |
| PPP 项目：建设期→运营期移交 | ✅ 移交后 | ✅ 自始 |

**关键**：房产税从"达到可使用状态"或"实际使用"次月起计税，不是从"办完产权证"起。

### 在建工程的争议

在建工程是否缴纳房产税？

| 情形 | 是否应税 |
|------|---------|
| 在建工程未投入使用 | ❌ 原则上不缴 |
| 在建工程部分投入使用（如办公楼已使用） | ✅ 已使用部分需缴 |
| 在建工程全部投入使用但未验收 | ✅ 全额缴纳 |

## 二、房产税计算

### 自用房产

```
年房产税 = 房产原值 × (1 − 扣除比例) × 1.2%

房产原值：包括土地成本 + 建造成本 + 资本化利息 + 不可分割的设备
扣除比例：10%-30%（各省不同，通常 20-30%）
```

### 计算示例

```
SPV 持有的建成设施：
  房产原值（含土地+建造+利息）：10,000 万元
  扣除比例（假设）：30%
  年房产税：10,000 × (1 − 30%) × 1.2% = 84 万元/年
  季度缴纳：84 / 4 = 21 万元/季
```

### 出租房产（如 SPV 将商铺/办公出租）

```
房产税 = 租金收入 × 12%

如果租金年收入 500 万元：
  房产税 = 500 × 12% = 60 万元/年
```

## 三、城镇土地使用税

```
年土地使用税 = 实际占用土地面积 × 适用税额

适用税额：各地不同
  - 大城市：1.5-30 元/平米
  - 中等城市：1.2-24 元/平米
  - 小城市：0.9-18 元/平米
  - 县城/建制镇/工矿区：0.6-12 元/平米
```

### SPV 的常见情况

PPP/BOT 项目 SPV 持有大面积土地（如车辆段/车场/轨交线路用地），土地使用税可能是**每年数百万元**级别的税负。

## 四、PPP 项目的特殊考量

### 资产分类对房产税的影响

| PPP 资产分类 | 所有权 | 房产税纳税人 |
|------------|--------|------------|
| 金融资产模式（政府付费） | 政府 or SPV | 谁持有产权谁纳税 |
| 无形资产模式（使用者付费） | SPV | **SPV** |
| 混合模式 | 分别判断 | 分别判断 |

### 运营期移交的税务风险

部分 PPP 项目约定运营期满后资产无偿移交政府。在运营期内：
- SPV 作为资产的会计/税务持有人，缴纳房产税
- 如当地税务机关认为运营期的资产属于"代政府持有" → 可能免税（需个案沟通）

## 五、自检清单

1. [ ] SPV 是否在取得土地后及时申报了土地使用税？
2. [ ] 竣工验收/投入使用后是否从次月起申报了房产税？
3. [ ] 房产原值的计算是否包含了土地成本+建造成本+资本化利息？
4. [ ] PPP 项目：资产的法律所有权归属对房产税纳税人认定是否有影响？
5. [ ] 部分已使用的在建工程是否已就使用部分申报房产税？

## 免责声明

房产税和土地使用税的具体政策和税率因各省市而异。PPP 项目的税务处理建议咨询当地税务机关和税务顾问。
