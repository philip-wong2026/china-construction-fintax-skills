# 施工企业印花税

> **SLOT 说明**：印花税是施工企业合同管理中的重要税种。适用于 {ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 合同类型 | 税率 | 计税基础 |
|---------|------|---------|
| 建设工程合同（施工/EPC） | **万分之三**（0.03%） | 合同金额（含税价） |
| 勘察/设计合同 | **万分之五**（0.05%） | 合同金额 |
| 材料/设备采购合同 | **万分之三**（0.03%） | 合同金额 |
| 设备租赁合同 | **千分之一**（0.1%） | 租金 |
| 技术合同 | **万分之三**（0.03%） | 合同金额 |
| 借款合同 | **万分之零点五**（0.005%） | 借款金额 |
| 运输合同 | **万分之三**（0.03%） | 运费 |

**全行业统一**——印花税税率与工程类型（铁路/公路/市政/房建/水利/港口/矿山/机电）完全无关。

## 一、施工企业的核心印花税合同

### 1.1 建设工程合同（税率万分之三）

```
印花税 = 合同金额（含税价）× 0.03%
```

**合同金额的确定**：
- 合同明确金额的：按合同金额
- 合同仅约定单价、按实际结算量：按**预计总价**贴花，最终按结算价**多退少补**
- 框架合同（无总金额）：先按 5 元贴花，每期结算时按结算金额补缴

**总包和分包各自纳税**：
- 总包方：总包合同 × 0.03%
- 分包方：分包合同 × 0.03%
- **不是差额纳税！总包和分包各自独立纳税**

### 1.2 EPC 合同——需拆分

EPC 合同包含设计(E)、采购(P)、施工(C)三部分：

| 组成部分 | 合同类型 | 税率 |
|---------|---------|------|
| E（设计） | 建设工程勘察设计合同 | **万分之五** |
| P（采购） | 买卖合同 | **万分之三** |
| C（施工） | 建设工程合同 | **万分之三** |

**操作建议**：EPC 合同中明确拆分 E/P/C 三部分价格 → 分别按各自税率缴纳。
若合同不拆分，税务机关可能要求**全部**按万分之五（设计合同）or 万分之三（施工合同），取决于合同定性。

### 1.3 材料/设备采购合同

采购合同（钢材/水泥/砂石/设备）= 万分之三

**体量大的施工企业**：年采购合同数百份→印花税可能达到**数百万**，是重要的税务成本。

## 二、印花税的计算与缴纳

### 2.1 纳税时间

| 情形 | 纳税时间 |
|------|---------|
| 签订书面合同 | 合同书立当日 |
| 以电子形式签订 | 电子合同书立当日 |
| 未签订合同但已实际履行 | **实际结算日** |

### 2.2 申报缴纳

印花税按**季/年或按次**申报：
- 合同量大的施工企业 → **按季申报**（季度终了 15 日内）
- 合同量小的 → 按次申报

### 2.3 全电发票时代的印花税

全电发票推行后，印花税的征管方式发生了变化：
- 税务机关通过发票金额**倒推**印花税 → 与申报比对
- 若发票金额大但印花税申报明显偏小 → 标记为风险
- 施工企业应**按开票金额**做好印花税计提，避免比对异常

## 三、不同施工合同类型的处理

| 合同类型 | 印花税处理 | 注意 |
|---------|-----------|------|
| 固定总价合同 | 按合同金额计税 | 一次性缴清 |
| 单价合同（按实结算） | 先按预计总价预缴，结算后多退少补 | 需跟踪管理 |
| 成本加酬金合同 | 按预计成本+酬金计税 | |
| 框架协议（无金额） | 先 5 元，后按每期结算补缴 | |
| 补充协议（变更/签证） | 若增加金额 → 补缴差额 | 不增加金额不补 |
| 三方协议（发包方-总包-分包） | 发包方与总包、总包与分包**各自**计税 | 注意：不是一份合同 |

## 四、常见错误

| 错误 | 后果 | 纠正 |
|------|------|------|
| 印花税按不含税价计算 | 少缴 9% or 3% | 按**含税价** |
| EPC 合同统一按万分之三 | E 部分按万分之五——少缴 | 拆分 E/P/C |
| 分包合同未缴纳印花税 | 漏税 | 总包和分包**各自独立纳税** |
| 补充协议增加金额未补税 | 少缴 | 增加金额部分补缴 |
| 仅按开票金额缴纳（忽视未开票合同） | 已签未开票合同漏税 | 按**合同金额**，非开票金额 |

## 五、施工企业印花税管理建议

1. **建立合同台账**：含合同金额、印花税计提金额、缴纳日期
2. **EPC 合同强制拆分**：在合同模板中明确 E/P/C 分项价格
3. **补充协议跟踪**：变更/签证增加金额的，及时补缴
4. **季度预提**：按合同金额预先计提印花税，避免遗漏
5. **关联交易注意**：集团内各法人之间的内部合同也需缴纳印花税

## 免责声明

印花税税率以《中华人民共和国印花税法》（2022年7月1日施行）为准。部分税目可享受减征优惠（如小规模纳税人减半），具体以最新政策为准。
