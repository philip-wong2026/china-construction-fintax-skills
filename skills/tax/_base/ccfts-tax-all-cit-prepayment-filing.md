# 施工企业 CIT 预缴申报

> **SLOT 说明**：CIT 预缴规则取决于是否为独立法人。{ENTITY_LABEL}（{ENTITY_ALIAS}），{IS_LEGAL_ENTITY}。

## 快速参考

| 项目 | 独立法人（工程局/子公司/SPV） | 非法人（项目部/分公司） |
|------|------------------------------|------------------------|
| 预缴周期 | 月度 or 季度 | 不单独预缴 |
| 预征率（跨省项目） | 0.2% | —（由上级法人统一预缴） |
| 计税基础 | 项目实际经营收入 | — |
| 适用行业 | 全行业统一 | 全行业统一 |

## 一、独立法人——跨地区经营汇总纳税

### 1.1 跨省项目 CIT 预缴

独立法人在外省施工，按项目实际经营收入的 **0.2%** 在项目所在地预缴 CIT：

```
项目所在地预缴 CIT = 项目实际经营收入 × 0.2%
```

**"实际经营收入"的含义**：
- 即项目部/总包部在该地区的**主营业务收入**（6001）
- 不含其他业务收入、营业外收入
- 按**会计核算**的收入确认口径，非开票/收款口径

### 1.2 汇总纳税分配

所有跨省项目（分支机构）的 CIT 由总机构（工程局/集团）汇总后按以下因素分配：

| 因素 | 权重 | 说明 |
|------|------|------|
| 营业收入 | **0.35** | 该分支机构的营业收入 / 各分支机构营业收入之和 |
| 职工薪酬 | **0.35** | 该分支机构的职工薪酬 / 各分支机构薪酬之和 |
| 资产总额 | **0.30** | 该分支机构的资产总额 / 各分支机构资产之和 |

```
某项目部分摊比例 = 0.35 × (该部收入 / 总收入)
                   + 0.35 × (该部薪酬 / 总薪酬)
                   + 0.30 × (该部资产 / 总资产)

该部分摊 CIT = 总机构统一计算的应纳税额 × 50% × 该部分摊比例

(50% 由总机构就地缴纳，50% 在各分支机构间分配)
```

### 1.3 省内跨市项目

各省规定不一：
- 部分省：比照跨省执行（0.2% 预缴 + 分配）
- 部分省：不预缴，全部在总机构所在地缴纳
- **以项目所在地税务机关口径为准**

### 1.4 预缴时间

| 预缴方式 | 时间 |
|---------|------|
| 按月预缴 | 月度终了后 **15 日内** |
| 按季预缴 | 季度终了后 **15 日内** |
| 年度汇算清缴 | 年度终了后 **5 个月内**（次年 5 月 31 日前） |

## 二、非法人——不单独缴纳 CIT

项目部/总包部/分公司（非法人）不单独预缴 CIT，包含在上级法人的汇总纳税中。

但需注意：
- 项目部仍需**核算**经营成果（收入/成本/利润），供上级法人汇总
- 跨区域项目部需配合上级法人完成 CIT 分配计算
- 利润总额的准确性直接影响 CIT 计算的准确性

## 三、年度汇算清缴——施工企业重点调整

### 3.1 常见纳税调整

| 调整项目 | 方向 | 说明 |
|---------|------|------|
| 安全生产费（计提未使用） | **调增** | 计提时计入成本但未实际支出 |
| 合同资产减值准备（1484） | **调增** | 未经核准的减值准备 |
| 坏账准备（1231） | **调增** | 超过规定比例的坏账准备 |
| 质保金坏账（实际核销） | **调减** | 实际发生且已备案的坏账损失 |
| 业务招待费（超过限额） | **调增** | 发生额 60% and 收入 5‰ 孰低 |
| 广告费和业务宣传费（超过限额） | **调增** | 收入 15% |
| 税收滞纳金/罚款 | **调增** | 不得税前扣除 |
| 研发费用加计扣除 | **调减** | 100% 加计扣除（2023-2027） |
| 固定资产加速折旧 | **调减** | 符合条件的一次性扣除 |

### 3.2 施工企业汇算清缴特殊调整

| 调整项目 | 说明 |
|---------|------|
| 未按完工进度确认的收入 | 税务与会计差异（税务要求按完工进度，不能随意推迟） |
| 跨期费用 | 已发生但未取得发票的费用（次年 5.31 前取得可扣除） |
| PPP 项目利息资本化 vs 费用化 | 税会差异 |
| EPC 项目 P 部分（设备）的收入确认时点 | 税务可能要求比会计更早确认 |

## 四、预缴申报表填报

### 月/季度预缴表（A 类）关键行次

| 行次 | 项目 | 取数 |
|------|------|------|
| 第 1 行 | 营业收入 | 利润表营业收入（累计） |
| 第 2 行 | 营业成本 | 利润表营业成本（累计） |
| 第 3 行 | 利润总额 | 利润表利润总额（累计） |
| 第 9 行 | 实际利润额 | 第 3 行 + 纳税调整（如有） |
| 第 12 行 | 税率 | **25%** |
| 第 13 行 | 应纳所得税额 | 第 9 行 × 25% |
| 第 14 行 | 减：特定业务已缴所得税额 | 跨省项目预缴的 0.2% 合计 |
| 第 15 行 | 本期应补(退)所得税额 | 第 13 行 − 第 14 行 |

## 五、自检清单

1. [ ] 是否为独立法人？（是 → 需预缴 / 否 → 不单独预缴）
2. [ ] 跨省项目是否按 0.2% × 实际经营收入预缴？
3. [ ] 总机构的 50% 分摊计算是否正确？
4. [ ] 安全生产费（未使用部分）是否在汇算清缴时调增？
5. [ ] 资产减值准备是否调增？
6. [ ] 研发费用是否享受 100% 加计扣除？
7. [ ] 跨期费用发票是否在次年 5.31 前取得？
8. [ ] 预缴和汇算清缴的截止日期是否正确？
