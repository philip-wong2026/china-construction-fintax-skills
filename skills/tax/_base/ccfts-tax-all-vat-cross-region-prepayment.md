# 施工企业跨区域 VAT 预缴

> **SLOT 说明**：跨区域施工是施工企业常态。本文件适用于 {ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 项目 | 一般计税 | 简易计税 |
|------|---------|---------|
| 预征率 | **2%** | **3%** |
| 计税基础 | (全部价款+价外费用 − 分包款) / (1+9%) | (全部价款+价外费用 − 分包款) / (1+3%) |
| 预缴地点 | 项目所在地（跨县/市/区）税务机关 | 同 |
| 预缴时间 | 收到工程款或合同约定收款日的**当月/次月** | 同 |
| 申报抵减 | 机构所在地申报时抵减 | 同 |
| 适用行业 | 铁路/公路/市政/房建/水利/港口/矿山/机电——**全行业统一** | 同 |

## 一、什么情况下需要跨区域预缴

### 判断标准

```
施工项目的地址 ≠ 施工企业机构所在地的县（市/区）
  │
  └── YES → 必须跨区域预缴
       │
       └── 例外：同一地级市内跨区？各省规定不一（部分省取消）
              → 以当地税务机关口径为准
```

### 无需预缴的情形

| 情形 | 说明 |
|------|------|
| 项目与机构在同一县（市/区） | 不跨区域 |
| 国家税务总局公告明确不需要的其他情形 | 以最新公告为准 |

### 铁路/公路/水利等线性工程的跨区域问题

铁路、公路、水利等线性工程跨越多个行政区划：
- 原则上按**项目整体**预缴（在项目主要所在地）
- 实际操作中，各地税务机关可能要求**按标段/区段**分别预缴
- 以施工合同的发包方和项目所在地税务机关的要求为准

## 二、预缴税额计算

### 一般计税（预征率 2%）

```
预缴税额 = (全部价款 + 价外费用 − 支付的分包款) / (1 + 9%) × 2%
```

**扣除分包款的条件**：
- 分包款必须取得**增值税发票**（专票或普票均可）
- 从**总包全部价款**中扣除
- 扣除的是**含税价**

### 简易计税（预征率 3%）

```
预缴税额 = (全部价款 + 价外费用 − 支付的分包款) / (1 + 3%) × 3%
```

### 计算示例

```
某公路项目（一般计税）：
  总包合同含税价 10,900 万元
  分包合同含税价 3,270 万元
  本期收到工程款 5,450 万元
  本期支付分包款 1,635 万元

预缴税额 = (5,450 − 1,635) / 1.09 × 2%
         = 3,815 / 1.09 × 2%
         = 3,500 × 2%
         = 70 万元
```

## 三、预缴操作流程

### 3.1 开工前——跨区域涉税事项报告

```
Step 1：在机构所在地电子税务局
  → 提交"跨区域涉税事项报告"
  → 填写：项目名称、项目地址、合同金额、合同期限
  → 系统自动生成"跨区域涉税事项报告表"（取代原来的外管证）

Step 2：在项目所在地电子税务局
  → "跨区域涉税事项报验"
  → 关联跨区域涉税事项报告表编号
```

### 3.2 每次收款后——预缴申报

```
Step 1：在项目所在地电子税务局
  → "增值税及附加税费预缴表"
  → 填写：销售额、扣除额（分包款）、预征率
  → 系统自动计算预缴税额

Step 2：缴款
  → 项目所在地缴纳入库
  → 取得完税凭证（电子）
  → 记录预缴台账
```

### 3.3 月末/次月——机构所在地申报抵减

```
Step 1：在机构所在地电子税务局
  → 正常填写增值税纳税申报表
  → 附表四第3栏"建筑服务预征缴纳税款"：填入本期预缴税额
  → 主表第28行自动带入

Step 2：抵减
  → 应纳税额 − 预缴税额 = 本期应补(退)税额
```

## 四、预缴台账管理（2026 年新规）

金税四期要求施工企业建立**按项目核算的预缴台账**：

| 台账字段 | 内容 |
|---------|------|
| 项目名称 | 与施工合同一致 |
| 项目地址 | 省/市/县/区 |
| 跨区域涉税事项报告编号 | 系统自动生成 |
| 合同总金额 | 含税 |
| 本期收款金额 | |
| 本期扣除分包款 | |
| 预征率 | 2% or 3% |
| 预缴税额 | |
| 完税凭证号码 | |
| 缴款日期 | |

**金税四期自动比对**：预缴税额与开票金额、申报收入进行交叉比对。
若开票金额大但预缴金额小 → 标记为异常。

## 五、附加税费的预缴

随增值税预缴同时，需预缴附加税费：

| 附加税费 | 税率 | 计税基础 |
|---------|------|---------|
| 城市维护建设税 | 7%/5%/1%（按项目所在地） | 预缴的增值税额 |
| 教育费附加 | 3% | 预缴的增值税额 |
| 地方教育附加 | 2% | 预缴的增值税额 |

```
预缴附加税费合计 = 预缴增值税 × (城建税率 + 3% + 2%)
```

**注意**：城建税税率按**项目所在地**，不是机构所在地。
农村项目 1%、县城/镇 5%、市区 7%。

## 六、常见错误

| 错误 | 后果 | 纠正 |
|------|------|------|
| 跨区域项目未做"跨区域涉税事项报告" | 无法在当地预缴 → 机构所在地税务机关追缴 | 开工前必须办理 |
| 分包款扣除凭证不合规（白条/收据） | 不得扣除，需补税 | 必须取得增值税发票 |
| 预缴后未在机构所在地申报抵减 | 双重纳税 | 附表四正确填写 |
| 简易计税按 2% 预缴 | 少预缴 | 简易计税预征率是 3%，不是 2% |
| 附加税费按机构所在地税率 | 多缴或少缴附加税 | 按**项目所在地**税率 |

## 七、自检清单

1. [ ] 项目是否跨县/市/区？（是 → 必须预缴）
2. [ ] 是否已办理跨区域涉税事项报告？
3. [ ] 预征率选择是否正确（一般 2% / 简易 3%）？
4. [ ] 分包款扣除凭证是否为增值税发票？
5. [ ] 预缴台账是否按项目建立并完整记录？
6. [ ] 机构所在地申报时是否正确填写了附表四？
7. [ ] 附加税费是否按项目所在地税率计算？
