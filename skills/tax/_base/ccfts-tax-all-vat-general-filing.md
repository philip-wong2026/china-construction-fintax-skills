# 施工企业增值税一般计税月度申报

> **SLOT 说明**：本文件定义一般计税（9%）的月度 VAT 申报全流程。适用于 {ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 项目 | 规则 |
|---|---|
| 适用纳税人 | {IS_GENERAL_TAXPAYER} |
| 建筑服务税率 | **9%** |
| 申报周期 | **月度**（次月 15 日前） |
| 申报表 | 增值税及附加税费申报表（一般纳税人适用） |
| 适用行业 | 铁路/公路/市政（含城轨）/房建/水利水电/港口航道/矿山/机电安装——**全行业统一 9%** |

## 一、施工企业销项税额计算

### 1.1 销售额确定

```
销项税额 = 销售额（不含税）× 9%

销售额（不含税）= 含税收入 / (1 + 9%)
```

### 1.2 纳税义务发生时间（2026 增值税法最早原则）

| 时点 | 触发条件 |
|------|---------|
| 开票日 | 开具增值税发票（含全电发票）的当天 |
| 收款日 | 收到工程款的当天 |
| 合同约定收款日 | 合同约定的付款日期 |
| 结算日 | 工程结算单经双方确认的日期（实际收款权利的取得日） |

**四者中最早者**即为纳税义务发生时间。

### 1.3 施工企业销售额包含内容

| 收入类型 | 是否计入销售额 | 说明 |
|---------|--------------|------|
| 工程结算款（含预付款） | ✅ | 收到预付款即产生纳税义务 |
| 变更/签证增加 | ✅ | 确认时计入 |
| 索赔收入 | ✅ | 确认时计入 |
| 奖励款（工期/质量） | ✅ | 收到或确认时计入 |
| 材料销售（剩余材料出售） | ✅ | 13% 税率（非 9%） |
| 设备租赁（带操作员） | ✅ | 9%（建筑服务） |
| 设备租赁（不带操作员） | ✅ | 13%（有形动产租赁） |
| 质保金 | 到期或开票时 | 见 \"VAT 质保金时点\" |
| 废弃/罚款收入 | ❌ | 非增值税应税收入 |
| 保险赔款 | ❌ | 同上 |

## 二、施工企业进项税额抵扣

### 2.1 主要可抵扣进项

| 支出类型 | 税率 | 抵扣凭证 |
|---------|------|---------|
| 分包工程款（一般计税） | 9% | 增值税专用发票 |
| 分包工程款（简易计税） | 3% | 增值税专用发票 |
| 材料采购（钢材/水泥/砂石等） | 13% | 增值税专用发票 |
| 商品混凝土（简易计税供应商） | 3% | 增值税专用发票 |
| 设备采购/租赁 | 13% | 增值税专用发票 |
| 设计费/勘察费 | 6% | 增值税专用发票 |
| 运输费 | 9% | 增值税专用发票 |
| 管理费用（办公/水电/物业） | 13%/6%/3% | 增值税专用发票 |
| 安全生产费支出（设备/防护用品） | 13% | 增值税专用发票 |
| 差旅费（住宿） | 6% | 增值税专用发票 |

### 2.2 不可抵扣进项

| 支出类型 | 原因 |
|---------|------|
| 旅客运输服务（员工通勤） | 个人消费性质 |
| 餐饮服务 | 个人消费 |
| 居民日常服务 | 个人消费 |
| 娱乐服务 | 个人消费 |
| 贷款服务（利息支出） | 不得抵扣（贷款服务进项税不得抵扣） |
| 福利费支出（集体福利） | 个人消费 |
| 免税项目对应的进项 | 对应免税收入 |
| 简易计税项目对应的进项 | **须单独核算，不得混抵** |

### 2.3 施工企业特有的进项转出

| 情景 | 处理 |
|------|------|
| 工程物资用于简易计税项目（已抵扣进项） | 按比例**转出** |
| 工程物资发生非正常损失（被盗/丢失/霉烂变质） | 全额**转出** |
| 已抵扣的材料退货/折让 | 冲减当期进项 |
| 兼营一般计税和简易计税项目 | 按收入比例分摊 → 简易计税部分转出 |

## 三、月度申报主表填报

### 主表关键行次

| 行次 | 项目 | 取数来源 |
|------|------|---------|
| 第 1 行 | 按适用税率计税销售额 | = 不含税建筑服务收入 |
| 第 2 行 | 其中：应税货物销售额 | 材料销售收入 |
| 第 11 行 | 销项税额 | = 第 1 行 × 9% + 第 2 行 × 13% |
| 第 12 行 | 进项税额 | = 认证通过的进项税额合计 |
| 第 17 行 | 应抵扣税额合计 | = 第 12 行 + 上期留抵 |
| 第 19 行 | 应纳税额 | = 第 11 行 − 第 17 行 |
| 第 23 行 | 应纳税额减征额 | 税控设备/技术维护费抵减 |
| 第 24 行 | 应纳税额合计 | = 第 19 行 − 第 23 行 |
| 第 28 行 | 分次预缴税额 | = 跨区域预缴税款（见跨区域预缴技能） |
| 第 34 行 | **本期应补(退)税额** | = 第 24 行 − 第 28 行 |

## 四、附表填报要点

### 附表一（销项明细）
- 建筑服务（9%）填"服务、不动产和无形资产"列
- 材料销售（13%）填"货物及加工修理修配劳务"列
- 一般计税和简易计税**分别填列**，不得串列

### 附表二（进项明细）
- 本期认证相符的增值税专用发票
- 注意：海关进口增值税专用缴款书、农产品收购发票单独列示

### 附表四（税额抵减情况）
- 建筑服务预缴税款（跨区域）填入第 3 栏
- 税控设备及技术维护费抵减

## 五、不同施工领域的特殊关注

| 工程类型 | 销项特殊点 | 进项特殊点 |
|---------|-----------|-----------|
| 铁路工程 | — | 轨道材料(13%)、信号设备(13%) |
| 公路工程 | — | 沥青混凝土(3% or 13%) |
| 市政/城轨 | 盾构施工、机电安装可能拆分 | 盾构机租赁(13%)、管片(13%) |
| 房建工程 | — | 商品混凝土(3%)、钢材(13%) |
| 水利水电 | — | 大型设备租赁(13%) |
| 港口航道 | — | 疏浚设备(13%) |
| 矿山工程 | 井下工程有特殊安全要求 | 矿建专用设备(13%) |
| 机电安装 | 设备+安装混合销售判断 | 设备采购(13%) |

## 六、自检清单
