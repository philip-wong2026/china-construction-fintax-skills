# 施工企业全电发票操作

> **SLOT 说明**：全电发票（数电票）已全面推行。适用于 {ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 项目 | 规则 |
|---|---|
| 发票类型 | 全电发票（数电票）——全面取代纸质发票和电子发票（税控） |
| 建筑服务发票 | 必须填写**备注栏**（项目地址 + 项目名称） |
| 税率/征收率 | 9%（一般）/ 3%（简易）/ 13%（货物销售）/ 6%（设计） |
| 红冲规则 | 全电发票**不可作废**，只能红冲 |
| 金税四期比对 | 发票金额 vs 申报收入 vs 预缴税额 自动三方比对 |
| 适用行业 | 全行业统一——与工程类型无关 |

## 一、建筑服务发票的开具要点

### 1.1 备注栏——施工企业最容易出错的地方

建筑服务全电发票的备注栏**必须**填写：

```
项目地址：某省某市某区xxx路（与施工许可证一致）
项目名称：xx项目总承包工程（与施工合同一致）
```

**跨区域项目的特殊要求**：
- 项目地址栏必须与《跨区域涉税事项报告》中的项目地址**完全一致**
- 金税四期自动比对项目地址 → 不一致即标记异常

**不规范后果**：
- 金税四期自动比对 → 字段缺失标记为**异常发票**
- 受票方（业主）收到异常发票 → 进项税额可能**不得抵扣**
- 影响业主付款（业主通常要求合规发票才付款）

### 1.2 货物销售 vs 建筑服务的区分

施工企业开具发票时需正确选择商品和服务税收分类：

| 业务 | 商品和服务分类 | 税率 |
|------|--------------|------|
| 一般施工 | **建筑服务**——工程服务 | 9% |
| 清包工 | **建筑服务**——工程服务（简易计税） | 3% |
| 剩余材料出售 | **货物**——xxx材料 | 13% |
| 设备租赁（带人） | **建筑服务**——其他建筑服务 | 9% |
| 设备租赁（不带人） | **现代服务**——经营租赁 | 13% |
| 设计服务 | **现代服务**——工程设计 | 6% |
| 项目管理费 | **现代服务**——企业管理服务 | 6% |

### 1.3 EPC 项目开票

EPC 项目应**分别开具**不同税率的发票：

| 组成部分 | 发票税率 | 一张发票 or 多张 |
|---------|---------|-----------------|
| E（设计） | 6% | 单独开具 |
| P（设备采购） | 13% | 单独开具 |
| C（施工） | 9% | 单独开具 |

**不建议合并开具**——合并开具会被认定为混合销售，可能的后果是全部从高（13%）or 从主（9%）。

## 二、红冲操作——全电发票关键变化

### 2.1 什么情况下红冲

| 情形 | 处理 |
|------|------|
| 开票有误（税率选错/金额填错/备注栏缺漏） | **全额红冲** → 重新开具 |
| 工程结算审减（审定金额 < 已开票金额） | **部分红冲**（差额部分） |
| 工程返工/折让 | **部分红冲**（折让金额） |
| 退货（材料销售） | **部分红冲**（退货金额） |
| 业主退票要求 | 确认原因后处理 |

### 2.2 全电发票不能作废

传统纸质/电子发票：当月开错 → 作废重开
全电发票：**不设"作废"功能 → 只能红冲**

**流程**：
```
Step 1：受票方（业主）如未抵扣 → 开票方发起红冲申请
  受票方（业主）如已抵扣 → 受票方发起红冲申请，开票方确认

Step 2：税务机关审核（系统自动比对）
  → 审核通过 → 开具红字全电发票

Step 3：原蓝字发票 + 红字发票 同时在税务数字账户中显示
```

### 2.3 审减后的红冲（施工企业特有场景）

竣工结算审减 → 已开票金额大于审定金额 → 需要红冲差额：

```
原开票金额：5,000 万元（含税）
审定金额：  4,800 万元（含税）
审减金额：    200 万元 → 部分红冲 200 万元

红字发票：
  建筑服务 −200 万元
  税额      −200 / 1.09 × 9% = −16.51 万元
```

**注意**：审减确认后才能红冲，不能凭审计征求意见稿就红冲。

## 三、金税四期的异常比对

金税四期对施工企业的发票数据做以下自动比对：

| 比对项目 | 数据源 A | 数据源 B | 阀值 |
|---------|---------|---------|------|
| 收入与开票 | 增值税申报收入 | 开票金额 | 差异 > 10% |
| 开票与预缴 | 开票金额 | 跨区域预缴税额 | 开票 > 0 but 预缴 = 0 |
| 税率异常 | 建筑服务 9% | 出现 13%（货物） or 6%（设计） | 比对逻辑 |
| 备注栏完整性 | 建筑服务发票 | 项目地址+项目名称 | 任一缺失 |

**触发异常后**：
1. 电子税务局推送"风险提示"
2. 要求企业在规定期限（通常 15 日）内自查并回复
3. 若无法合理解释 → 转人工核查 → 可能税务稽查

## 四、全电发票的日常管理

### 4.1 发票开具台账

为应对金税四期比对，施工企业应建立：

| 台账字段 | 用途 |
|---------|------|
| 开票日期 | |
| 发票号码 | |
| 受票方名称（业主/发包方） | |
| 项目名称 | 与备注栏一致 |
| 项目地址 | 与备注栏一致 |
| 开票金额（含税） | |
| 税率 | 9%/3%/13%/6% |
| 税额 | |
| 对应跨区域预缴税额 | 用于收入-开票-预缴 三方比对 |
| 是否红冲 | 红冲发票号码 |

### 4.2 发票归档

全电发票以电子文件（XML/PDF/OFD）形式存储：
- 需保存 **XML 格式**（税务机关的法定归档格式）
- PDF/OFD 仅为辅助格式（方便查阅）
- 保存期限：**30 年**（与会计档案期限一致）

## 五、自检清单

2. [ ] 发票税率是否正确（施工 9%、材料 13%、设计 6%、简易 3%）？
3. [ ] EPC 项目是否分别开具不同税率发票？
4. [ ] 红冲是否走完双方确认流程？

7. [ ] 发票台账是否完整（含预缴挂钩字段）？
8. [ ] XML 格式发票是否按规定保存？
