# 施工企业增值税简易计税

> **SLOT 说明**：简易计税适用于特定条件的施工项目。{ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 项目 | 规则 |
|---|---|
| 征收率 | **3%**（2027年前小规模纳税人暂按 **1%**） |
| 预征率（跨区域） | **3%** |
| 是否可抵扣进项 | **否**——选择简易计税即放弃进项抵扣 |
| 一经选择 | **36 个月内不得变更** |
| 适用行业 | 全行业——满足条件即可，与工程类型无关 |

## 一、适用条件

### 三种可适用简易计税的情形

| 情形 | 条件 | 关键限制 |
|------|------|---------|
| **清包工** | 施工方不采购建筑工程所需的材料，或只采购辅助材料 | 以提供人工/机械为主 |
| **甲供工程** | 工程所需全部或部分材料/设备/动力由发包方（甲方）提供 | **2026年1月1日起终止！必须用一般计税 9%** |
| **老项目** | 建筑工程施工许可证注明的合同开工日期在 **2016年4月30日之前** | 老项目逐渐退出，越来越少 |

### 甲供工程简易计税终止——2026年最大变化

```
2025年12月31日前：甲供工程可以选择简易计税 3%
2026年 1月 1日起：甲供工程必须使用一般计税 9%
  → 新签甲供合同按 9% 报价
  → 存量甲供合同：过渡期待明确
```

**影响全行业**：
- 铁路/公路/市政中常见的"甲供钢材""甲供水泥"项目 → 不能再选 3%
- 施工企业需在甲供合同报价中加入 9% VAT 成本
- 同时甲供材料的进项税归甲方，施工方无进项可抵，税负显著上升

### 清包工简易计税——保留，但分票从严

清包工仍可适用简易计税 3%，但 2026 年新规：
- 差额纳税的**分包部分**只能开具**普通发票**（不能开专票）
- 这对下游分包商的税务筹划产生影响

## 二、简易计税的申报

### 销售额与应纳税额

```
销售额（不含税）= 含税收入 / (1 + 3%)

应纳税额 = 销售额 × 3%

→ 无进项抵扣！全额按 3% 计税
```

### 差额纳税（清包工可选）

若清包工选择差额纳税：

```
销售额 = (全部价款 + 价外费用 − 支付的分包款) / (1 + 3%)
应纳税额 = 销售额 × 3%
```

**2026 年限制**：差额扣除的分包部分，只能取得**普票**（分包商不能开专票）。

## 三、简易计税 vs 一般计税——决策

### 税负比较（不考虑进项转嫁）

| 项目 | 一般计税 9% | 简易计税 3% |
|------|-----------|-----------|
| 销项税 | 收入 × 9% | 收入 × 3% |
| 进项税 | 可抵扣 | **不可抵扣** |
| 实际税负 | 9% − 进项占比 | 3% |
| 进项占比 > 6% | **选择一般计税** | — |
| 进项占比 < 6% | — | **选择简易计税** |

### 施工企业的典型进项占比

| 工程类型 | 材料占比 | 进项税率 | 综合进项 | 建议 |
|---------|---------|---------|---------|------|
| 房建（商品混凝土为主） | 30-40% | 3%（商混） | 较低 | 若清包工可选简易 |
| 公路（沥青/碎石） | 40-50% | 3% or 13% | 中 | 需测算 |
| 铁路/城轨（钢材为主） | 50-60% | 13% | 高 | **选一般计税（9%）** |
| 水利（砂石/水泥） | 35-45% | 3% or 13% | 中 | 需测算 |
| 清包工（人工为主） | <20% | — | **极低** | **选简易计税（3%）** |
| 机电安装（设备采购多） | 60-70% | 13% | 很高 | **必须选一般计税** |

## 四、全行业统一适用

简易计税的适用条件与工程类型**无关**——铁路的清包工和房建的清包工适用完全相同的 3% 规则。

行业差异仅体现在**进项结构**上（见上表），影响的是"要不要选简易计税"的决策，而非"能不能选"的资格。

## 五、跨区域预缴（简易计税）

见 `ccfts-tax-all-vat-cross-region-prepayment`，关键差异：

| 项目 | 一般计税 | 简易计税 |
|------|---------|---------|
| 预征率 | 2% | **3%** |
| 计税基础分母 | (1+9%) | (1+3%) |

## 六、一般计税和简易计税兼营

施工企业可能同时经营一般计税项目和简易计税项目。

**进项分摊要求**：
- 能直接归属的进项：直接计入对应项目
- 不能直接归属的（管理费用、公共设备）：按**收入比例**分摊
- 归属简易计税部分的进项 → **全额转出**（不得抵扣）

## 七、自检清单

1. [ ] 项目是否符合清包工/甲供/老项目的条件？
2. [ ] 甲供工程是否在 2026.1.1 后新签？（是 → 不能选简易计税）
3. [ ] 简易计税选择是否已报税务机关备案？
4. [ ] 36 个月内是否不得变更？
5. [ ] 简易计税的进项是否已全额转出（不得混抵）？
6. [ ] 差额纳税的分包款扣除凭证是否合规？
7. [ ] 跨区域预缴是否使用 3% 预征率（非 2%）？
