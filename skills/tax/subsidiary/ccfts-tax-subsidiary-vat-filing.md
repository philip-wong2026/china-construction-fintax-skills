# 工程局/子公司 VAT 月度申报

## 快速参考

| 项目 | 规则 |
|---|---|
| 申报周期 | 月度（次月 15 日前） |
| 适用税率 | 9%（一般）/ 3%（简易，如适用） |
| 进项认证 | 当月认证、当月抵扣 |
| 留抵退税 | 符合条件的可申请（增量留抵 + 存量留抵） |
| 跨区域预缴抵减 | 附表四填报 → 主表第 28 行自动抵减 |

## 一、子公司 VAT 申报的特殊性

工程局/工程公司（子公司）作为独立法人、一般纳税人，其 VAT 申报与项目部层面的差异：

| 维度 | 项目部 | 子公司 |
|------|--------|--------|
| 申报主体 | 非法人，不独立申报 | **独立法人，独立申报** |
| 进项归集 | 项目部取得进项→上转子公司 | 子公司统一认证抵扣 |
| 跨区域预缴 | 在项目所在地预缴 | 预缴税款汇总至子公司申报表抵减 |
| 简易计税管理 | 按项目单独核算 | 统一管理各项目的计税方式 |

## 二、进项税额认证管理

### 认证方式

| 方式 | 说明 |
|------|------|
| 增值税发票综合服务平台 | 登录平台→批量勾选认证 |
| 全电发票自动归集 | 税务数字账户自动接收、自动归集 |

### 认证时点

```
当月取得进项发票 → 当月认证 → 当月申报抵扣

逾期未认证：360 日内可认证（全电发票已无此限制）
```

### 子公司进项归集难点

子公司通常管理多个项目部，进项来源分散：

| 进项来源 | 归集方式 |
|---------|---------|
| 子公司本部采购 | 子公司取得发票，直接认证 |
| 项目部采购（子公司名义） | 供应商开票给子公司 → 子公司认证 |
| 项目部采购（项目部名义） | 项目部非法人不能认证 → 需以子公司名义采购 or 进项转子公司 |

**管控要点**：项目部以自己名义取得的进项发票，需上转给子公司方可认证抵扣。项目部不能独立认证——因其非法人。

## 三、留抵退税

### 适用条件

| 条件 | 说明 |
|------|------|
| 纳税信用等级 | A 级 or B 级 |
| 增量留抵税额 | 连续 6 个月增量留抵均 > 0 |
| 增量留抵金额 | 第 6 个月增量留抵 ≥ 50 万元 |

### 退还计算

```
增量留抵退税 = 增量留抵税额 × 进项构成比例 × 60%（一般行业）

进项构成比例 = 已抵扣的专票(含全电发票)+海关缴款书 / 全部已抵扣进项

建筑行业注意：2022年后部分行业适用 100% 退税——建筑行业是否适用以最新公告为准
```

### 施工企业的留抵常见原因

| 原因 | 说明 |
|------|------|
| 建设前期大量采购但收入确认滞后 | "进多销少" → 留抵 |
| EPC 项目设备采购（13%进项） | 设备到场但施工收入才 9% → 留抵 |
| 跨区域预缴已缴但进项未消化 | 预缴 > 应纳税 → 留抵 |

**现金流的"双重挤压"**：留抵退税到账前，企业已垫付进项税（支付给供应商）+ 跨区域预缴（缴给项目所在税局）→ 需管理层关注 VAT 现金流占用。

## 四、申报表填报要点（子公司版）

### 主表关键行次

| 行次 | 项目 | 子公司特别注意 |
|------|------|--------------|
| 第 11 行 | 销项税额 | 含所有项目部的一般计税收入 × 9% |
| 第 12 行 | 进项税额 | 子公司本部 + 所有项目部的认证进项 |
| 第 17 行 | 应抵扣税额合计 | 上期留抵 + 本期进项 |
| 第 19 行 | 应纳税额 | 可抵扣简易计税不得混入 |
| 第 28 行 | 分次预缴税额 | **所有跨区域项目的预缴税款合计** |

### 进项转出（附表二）

| 转出原因 | 说明 |
|---------|------|
| 简易计税项目的进项 | 按比例转出 |
| 集体福利/个人消费 | 全额转出 |
| 非正常损失 | 全额转出 |

## 五、兼营一般计税与简易计税——子公司管理要点

子公司可能同时管理一般计税项目和简易计税项目。

### 进项划分

| 进项类型 | 处理 |
|---------|------|
| 直接归属一般计税项目 | 全额抵扣 |
| 直接归属简易计税项目 | 全额转出（不抵扣） |
| 共同费用（总部管理费） | 按收入比例分摊 → 简易计税部分转出 |

### 分摊公式

```
不得抵扣的进项税额 = 共同进项税额 × (简易计税项目收入 / 全部收入)
```

## 六、自检清单

1. [ ] 所有项目部取得的进项发票是否已归集到子公司认证？
2. [ ] 简易计税项目的进项是否已全额转出（不混抵）？
3. [ ] 跨区域预缴税款是否全部在附表四填报并抵减？
4. [ ] 是否满足留抵退税条件？是否已申请？
5. [ ] 共同费用的进项是否按收入比例正确分摊？
6. [ ] 申报截止日（次月 15 日）前是否完成申报？

## 免责声明

VAT 月度申报的具体操作以电子税务局最新版申报表为准。留抵退税政策持续更新，以国家税务总局最新公告为准。
