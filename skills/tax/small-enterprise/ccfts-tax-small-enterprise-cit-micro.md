# 小型微利企业 CIT 优惠

## 快速参考

| 条件 | 标准 | 施工企业适用性 |
|------|------|-------------|
| 年应纳税所得额 | ≤ **300 万元** | 中小型施工企业可能满足 |
| 从业人数 | ≤ **300 人** | 中小型施工企业通常满足 |
| 资产总额 | ≤ **5,000 万元** | **主要门槛**——施工企业资产规模容易超标 |
| CIT 税率（优惠后） | **5%**（应税所得 ≤ 300 万部分） | |

## 一、施工企业的适用性判断

### 最容易卡住的指标——资产总额

施工企业的资产总额 = 货币资金 + 应收账款 + 合同资产 + 固定资产 + 其他资产

| 企业规模 | 典型资产总额 | 是否满足 ≤ 5,000 万 |
|---------|------------|-------------------|
| 年收入 3,000 万的施工队 | 1,000-2,000 万 | ✅ 可能满足 |
| 年收入 5,000 万的小型施工企业 | 2,000-3,500 万 | ✅ 通常满足 |
| 年收入 8,000 万的中型施工企业 | 3,500-6,000 万 | ⚠️ 需要核实 |
| 年收入 > 1 亿 | > 5,000 万 | ❌ 几乎不满足 |

**注意**：PPP 项目的合同资产（1483）价值巨大 → 中小型施工企业若参与 PPP → 资产总额极易超标 → 无法享受小型微利优惠。

### 从业人数计算

```
从业人数 = (年初人数 + 年末人数) / 2

包括：正式员工 + 劳务派遣人员
不包括：通过劳务分包公司招募的农民工（分包公司的人员不计入总包方）
```

## 二、税额计算

### 正常 CIT 税率 vs 小型微利优惠

| 情形 | 税率 | 应税所得 300 万的税额 |
|------|------|---------------------|
| 普通企业 | 25% | 75 万元 |
| 小型微利企业 | 5% | **15 万元** |
| 差额 | — | 节省 60 万元 |

### 分段计算（现行政策）

```
应税所得 ≤ 300 万元部分：减按 25% 计入应税所得 → 按 20% 税率
  = 应税所得 × 25% × 20%
  = 应税所得 × 5%
```

## 三、CIT 预缴时享受

小型微利企业在季度预缴时即可享受 5% 税率（不需等到汇算清缴）：

```
季度预缴 CIT = 累计实际利润额 × 5% − 前期已预缴
```

## 四、施工企业享受小型微利优惠的注意事项

### PPP/专项债项目的影响

- 若参与 PPP 项目（持股 SPV） → 资产总额包含对 SPV 的长期股权投资 → 容易超标
- 专项债项目：仅为 EPC 承包商 → 不影响资产总额（不持股）

### 资质申请对报表利润的需求

施工企业申请/升级资质时，通常需要展示一定的注册资本和资产规模 → 这与"控制资产总额以享受小型微利 CIT 优惠"存在天然矛盾。

**策略**：权衡资质需求 vs CIT 优惠。如果资质已满足，可以考虑通过分红/还债等方式缩减资产总额。

### 关联方分摊

若同一实际控制人控制的多个施工企业 → 不能通过拆分收入来满足小型微利条件（税务机关会合并判断）。

## 五、自检清单

1. [ ] 年应纳税所得额是否 ≤ 300 万元？
2. [ ] 从业人数是否 ≤ 300 人？
3. [ ] 资产总额是否 ≤ 5,000 万元？（最容易超标的指标）
4. [ ] 是否有 PPP 项目长期股权投资推高资产总额？
5. [ ] 季度预缴时是否已按 5% 计算（不等汇算清缴）？

## 免责声明

小型微利企业的具体认定和优惠以财政部和国家税务总局最新公告为准。
