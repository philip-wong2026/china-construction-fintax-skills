# 境外工程预提所得税与税收抵免

## 快速参考

| 税种 | 说明 |
|------|------|
| 境外预提所得税 | 项目所在国对支付给境外企业的款项征收的预扣税（WHT） |
| 税收协定优惠 | 中国与项目所在国签订的避免双重征税协定 → 降低 WHT 税率 |
| 境外已纳税款抵免 | 境外已缴的所得税可在中国 CIT 中**抵免**（限额内） |
| 居民身份证明 | 向境外税务机关证明中国税收居民身份 → 享受税收协定待遇的前提 |

## 一、境外预提所得税（Withholding Tax）

### 什么情况下涉及

| 境外支付 | 可能涉及 WHT | 税率（无协定时） |
|---------|------------|----------------|
| 工程承包款（服务费） | 部分国家征收 | 10-30% |
| 管理费/技术服务费 | 通常征收 | 10-20% |
| 利息（集团贷款利息） | 通常征收 | 10-20% |
| 股息（SPV 分红） | 通常征收 | 10-20% |
| 特许权使用费 | 通常征收 | 10-20% |

### 税收协定的保护

中国已与 100+ 个国家/地区签订税收协定，通常能将 WHT 税率**降低至 5-10%**，甚至**免征**。

| 协定待遇示例 | 说明 |
|------------|------|
| 工程承包服务（常设机构条款） | 工期不超过 6-12 个月的 → **免征**（视协定具体约定） |
| 利息 | 10% → **5%**（协定优惠） |
| 特许权使用费 | 10% → **5-7%** |

## 二、中国税收居民身份证明

### 为什么需要

境外税务机关给予税收协定优惠待遇的前提是：**收款方是中国的税收居民**。

施工企业需向主管税务局申请开具《中国税收居民身份证明》。

### 申请流程

```
Step 1：向主管税务局提交申请
  → 申请表 + 营业执照 + 境外合同 + 情况说明

Step 2：税务机关审核
  → 年度汇算清缴后申请更容易获批

Step 3：取得证明文件（中英文）
  → 有效期通常为 1 个自然年度

Step 4：将证明提交给境外税务机关
  → 作为享受税收协定待遇的依据
```

## 三、境外已纳税款抵免（CIT 抵免）

### 抵免原则

中国 CIT 对境外所得采用**抵免法**：

```
境外已缴所得税（含 WHT）→ 可在中国 CIT 中抵免

抵免限额 = 境外所得 × 中国 CIT 税率（25%）

实际抵免 = min(境外已缴税款, 抵免限额)
```

### 计算示例

```
境外项目利润（PRC GAAP 口径）：1,000 万元

境外已缴税：
  当地 CIT：200 万元（20% 税率）
  预提 WHT：80 万元（10%）
  境外合计：280 万元

中国 CIT 计算：
  境外所得：1,000 万元
  应纳中国 CIT：1,000 × 25% = 250 万元
  境外已缴税抵免：min(280, 250) = 250 万元
  实际应补缴中国 CIT：250 − 250 = 0

  超出抵免限额部分（280 − 250 = 30 万元）：
    → 可结转以后 5 个年度继续抵免
```

### 分国不分项 vs 综合抵免

| 方法 | 说明 | 现行政策 |
|------|------|---------|
| 分国不分项 | 各国境外已缴税分别计算抵免限额 | 默认方法 |
| 综合抵免 | 各国境外已缴税合并计算抵免限额 | 需税务机关批准 |

## 四、避免双重不征税

施工企业需注意"双重不征税"安排的风险：
- 不当利用税收协定 → 在来源国和居民国都未缴税
- 被认定为"择协避税" → 境外取消协定待遇 → 补税+罚款

## 五、自检清单

1. [ ] 境外项目所在国是否与中国有税收协定？
2. [ ] 是否申请了《中国税收居民身份证明》？
3. [ ] WHT 是否按税收协定优惠税率缴纳（而非标准税率）？
4. [ ] 境外已缴税款是否在中国 CIT 中正确抵免？
5. [ ] 超出抵免限额的部分是否在台账中记录（5 年结转）？
6. [ ] 有无不当的"择协避税"风险？

## 免责声明

WHT 税率及税收协定待遇以中国与项目所在国之间的具体协定为准。建议咨询国际税务顾问。
