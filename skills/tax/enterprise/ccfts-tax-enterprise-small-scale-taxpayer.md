# 小规模纳税人 VAT 处理

## 快速参考

| 项目 | 规则 |
|---|---|
| 年销售额标准 | ≤ **500 万元**（应税服务） |
| 征收率 | **3%**（2027年12月31日前暂按 **1%**） |
| 免税额度 | 月销售额 ≤ **10 万元**（季度 ≤ 30 万元）→ 免征 VAT |
| 进项抵扣 | **不可以**（小规模纳税人不抵扣进项） |
| 开票 | 可开具**普通发票**；可申请代开**专用发票** |

## 一、小规模纳税人的认定

### 施工企业的年销售额判断

```
年应征增值税销售额 = 全年建筑服务收入（不含税）

≤ 500 万元 → 可以选择作为小规模纳税人
> 500 万元 → 必须登记为一般纳税人
```

### 一般纳税人 vs 小规模纳税人——施工企业决策

| 考量 | 小规模 | 一般纳税人 |
|------|--------|-----------|
| 税率 | 1%（暂） | 9% |
| 进项抵扣 | 不可 | 可 |
| 给业主开票 | 普票 or 代开专票（1%） | 专票（9%） |
| 业主需求 | 部分业主不接受 1% 专票 | 业主通常要求 9% 专票 |
| 适合企业 | 小型分包商/个体施工队 | 中大型施工企业 |

**关键判断**：如果业主（总包方）需要 9% 的专票抵扣进项税 → 必须是一般纳税人。小规模只能开出 1% 的专票（代开），总包方的进项抵扣大幅减少，通常不被接受。

## 二、月度申报

### 免税（月销售额 ≤ 10 万元）

```
月建筑服务收入（不含税）≤ 10 万元
  → 免征增值税
  → 无需预缴（跨区域项目也免征）
```

### 应税（月销售额 > 10 万元）

```
应纳增值税 = 不含税销售额 × 1%（暂）
           = 含税销售额 / (1 + 1%) × 1%
```

## 三、跨区域施工（小规模纳税人）

小规模纳税人的跨区域 VAT 预缴：

| 月销售额 | 预缴 |
|---------|------|
| ≤ 10 万元 | **免征**（不预缴） |
| > 10 万元 | 按 1% 预征率在项目所在地预缴 |

## 四、代开增值税专用发票

小规模纳税人不能自行开具专票，但可向税务机关申请**代开专票**：

```
代开专票税率：1%（暂）
受票方（购买方）可抵扣：1%

施工企业场景：
  小规模分包商为一般纳税人总包提供劳务
  → 总包要求提供专票
  → 小规模分包商到税务局代开 1% 专票
  → 总包可抵扣 1%
```

**注意**：代开专票的部分**不免税**（即使月销售额 ≤ 10 万元）。

## 五、自检清单

1. [ ] 年销售额是否超过 500 万元？（超过 → 必须转一般纳税人）
2. [ ] 月销售额是否 ≤ 10 万元？（是 → 免税）
3. [ ] 业主是否要求 9% 专票？（是 → 必须转一般纳税人）
4. [ ] 代开专票的部分是否单独缴纳了 VAT？
5. [ ] 1% 暂按征收率的政策是否即将到期（2027.12.31）？

## 免责声明

小规模纳税人的具体政策以国家税务总局最新公告为准。1% 暂按征收率在 2027 年底后可能恢复 3%。
