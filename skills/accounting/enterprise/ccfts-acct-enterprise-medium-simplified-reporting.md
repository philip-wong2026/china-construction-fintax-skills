# 中型施工企业简化报表

## 快速参考

| 企业规模 | 年营业收入 | 适用报表 |
|---------|-----------|---------|
| 大型（央企） | ≥ 8 亿 | SASAC 全套决算（11张主表 + 补充表） |
| 中型（非央企） | 6,000万 ≤ Y < 8亿 | **简化报表 + 税务报表** |
| 小型 | 300万 ≤ Y < 6,000万 | 企业会计准则 or 小企业会计准则 |

## 一、中型企业与央企的报表差异

中型施工企业（非央企/非上市公司）不需要报送 SASAC 决算报表，报表负担显著轻于央企，但基本会计准则仍需遵守。

| 报表 | 央企 | 中型企业 |
|------|------|---------|
| 资产负债表 | ✅ | ✅（必须） |
| 利润表 | ✅ | ✅（必须） |
| 现金流量表 | ✅ | ✅（必须） |
| 所有者权益变动表 | ✅ | ✅ |
| 附注 | ✅ 详细 | ✅ 简化 |
| SASAC 决算附表 | ✅ 11+ 张 | ❌ |
| "一利五率"填报表 | ✅ | ❌ |
| 国有资本经营预算 | ✅ | ❌ |
| 审计报告 | ✅（必须） | 视融资/投标需求 |

## 二、中型企业的最小合规报表包

### 必须编制的

1. **资产负债表**（月/季/年）
2. **利润表**（月/季/年）
3. **现金流量表**（年）
4. **所有者权益变动表**（年）
5. **附注**（年，简化版）

### 主要简化

| 央企详细披露 | 中型企业简化 |
|------------|------------|
| 业务板块分部 | 可不单独披露 |
| PPP 项目详细情况 | 仅重大影响需披露 |
| 关联交易明细 | 金额不重大可合并 |
| 金融衍生品 | 通常不适用 |
| "两金"明细 | 可在附注中简要说明 |

### 税务报表（必须）

| 报表 | 频率 |
|------|------|
| 增值税纳税申报表 | 月/季 |
| 企业所得税预缴表 | 月/季 |
| 企业所得税年度申报表 | 年 |
| 印花税申报 | 季/年 |
| 社保/公积金缴费申报 | 月 |

## 三、中型施工企业的特殊考量

### 投标报表需求

虽然不需要 SASAC 决算，但中型企业在投标时可能需要提供经审计的财务报表：
- 政府项目通常要求近 3 年审计报告
- 部分大型项目要求"资产负债率不高于 X%"
- 银行授信需要年度审计报告

### 税务导向

中型企业比央企更关注**税务优化**（央企有 SASAC 考核约束，利润太低会被问责）：
- 合理利用小型微利企业 CIT 优惠（如适用）
- 研发费用加计扣除（若有研发活动）
- 固定资产加速折旧
- 小规模纳税人 VAT 优惠（如适用）

## 四、自检清单

1. [ ] 月/季度是否及时编制了资产负债表和利润表？
2. [ ] 年度是否编制了四张报表 + 附注？
3. [ ] 增值税/CIT/印花税是否按时申报？
4. [ ] 投标是否需要审计报告？是否已安排年审？
5. [ ] 是否需要享受小型微利企业 CIT 优惠？

## 免责声明

中型企业的具体报表要求因企业的实际规模和融资/投标需求而异。
