# 施工企业关联交易

## 快速参考

| 关联交易类型 | SASAC 关注度 | 审计重点 |
|------------|------------|---------|
| 内部总分包 | 高 | 定价是否公允、利润是否内部转移 |
| 资金拆借 | **极高** | 是否变相抽逃出资、资金占用费是否合理 |
| 内部担保 | 高 | 是否超出净资产比例、反担保措施 |
| 共同费用分摊 | 中 | 分摊方法是否合理一致 |
| 资产转让 | 高 | 评估定价是否公允 |

## 一、内部总分包——最常见的关联交易

建筑央企体系内，工程局（母公司）与下属工程公司（子公司）之间、工程公司与项目部之间普遍存在内部总分包关系。

### 定价原则

| 方式 | 说明 | SASAC 态度 |
|------|------|-----------|
| 市场价（正常交易价格） | 参照外部市场同类分包价格 | ✅ 推荐 |
| 成本加成 | 分包成本 + 合理利润率 | ✅ 可接受（需披露加成率） |
| 协商定价 | 双方协商 | ⚠️ 需证明合理性 |
| 利润转移（低价分包给子公司） | 差价实质为利润转移 | ❌ 审计重点 |

### 合并层面的抵销

内部总分包的收入和成本在合并层面**必须全额抵销**：
- 总包确认的收入 vs 分包确认的成本（内部交易部分）
- 若不抵销 → 虚增合并收入

## 二、资金拆借——SASAC 重点红线

### 内部资金拆借

集团内各法人之间的资金调剂，通过"资金池"（见 `ccfts-mgmt-soe-group-cash-pooling`）或直接拆借。

| 要素 | 要求 |
|------|------|
| 利率 | 参照同期银行贷款利率（不得明显低于或高于） |
| 期限 | 明确约定期限，不得"无期限" |
| 合同 | 必须签订书面借款协议 |
| 利息 | 出借方确认利息收入，借入方确认利息支出 |

### 禁止行为——SASAC 红线

- ❌ 以"往来款"名义掩盖实质借款
- ❌ 无偿占用子公司资金（母公司向子公司借款不计息）
- ❌ 通过内部拆借变相抽逃子公司注册资本
- ❌ 借款超出子公司净资产的一定比例
- ❌ 通过多层嵌套拆借规避监管

### 资金占用费

```
出借方（独立法人）：
  借：其他应收款—资金拆借—xx公司       XXX
    贷：银行存款                        XXX

  借：其他应收款—应收利息               XXX
    贷：财务费用—利息收入（6603-02）    XXX
      （or 其他业务收入）

借入方（独立法人）：
  借：银行存款                          XXX
    贷：其他应付款—资金拆借—xx公司      XXX

  借：财务费用—利息支出（6603-01）       XXX
    贷：其他应付款—应付利息             XXX
```

### VAT 处理

企业内部法人之间的资金拆借：
- 统借统还（集团统一借款→分拨子公司）：**免征 VAT**（符合条件时）
- 非统借统还：利息收入需缴纳 VAT（6%），利息支出进项**不得抵扣**

## 三、内部担保

建筑央企普遍存在母公司为子公司提供担保（投标担保/履约担保/融资担保）。

### 担保能力限制

| 规定 | 内容 |
|------|------|
| 累计担保总额 | 不得超过母公司净资产的 **50%**（部分央企有内部更低限制） |
| 单笔担保 | 不得超过被担保方净资产的 10% |

### 担保的会计处理

```
母公司（担保方）：
  表内不确认（或有负债）
  报表附注中披露：
    - 被担保方名称
    - 担保金额、期限
    - 是否提供反担保
    - 被担保方的财务状况

被担保方（子公司）：
  表内正常确认借款/履约义务
  不确认担保本身
```

## 四、共同费用分摊

集团内多个法人实体共用的费用需合理分摊：

| 共用费用 | 分摊基础 |
|---------|---------|
| 集团管理费 | 收入/资产/人数 |
| 共享服务中心（财务/HR） | 实际使用量 or 人数 |
| 品牌使用费/资质使用费 | 合同约定 |
| 研发费用（集团统一研发） | 受益比例 |

**SASAC 关注点**：分摊方法是否合理、是否一贯应用、是否成为利润转移的工具。

## 五、关联交易披露要求

SASAC 决算报表和上市公司年报均需披露：
1. 关联方清单（含子企业、合营/联营企业）
2. 关联交易类型和金额
3. 定价政策
4. 关联方应收/应付款项余额
5. 对外担保（含对子企业担保）的明细

## 六、自检清单

1. [ ] 内部总分包定价是否有合理依据（市场价/成本加成）？
2. [ ] 内部资金拆借是否签订了书面协议并约定了利率和期限？
3. [ ] 有无"无偿"占用子公司资金的情况？
4. [ ] 内部担保是否在报表附注中完整披露？
5. [ ] 共同费用分摊方法是否合理且一贯应用？
6. [ ] 合并层面的内部交易是否全额抵销？

## 免责声明

关联交易的合规要求以 SASAC 和证监会（如为上市公司）最新规定为准。
