# 施工合同变更与索赔的会计确认

## 快速参考

| 类型 | 确认条件 | 计量 |
|------|---------|------|
| 变更（已批准） | 业主已书面批准 | 按批准金额 |
| 变更（未批准） | "极可能"不会重大转回 | 估计最可能金额 |
| 索赔（已确认） | 业主已确认 or 极可能实现 | 按确认金额 or 估计金额 |
| 可变对价约束 | 累计已确认收入**极可能**不会发生重大转回 | 上限约束 |

## 一、合同变更的三种会计处理

CAS 14 对合同变更有三种处理方式，取决于变更的性质：

### 类型 1：变更作为单独合同

```
条件：变更增加了可明确区分的商品/服务 + 价格反映单独售价
处理：原合同不变，变更部分作为新合同处理
```

施工企业示例：原合同为车站施工，增补一个独立出入通道（可明确区分+单独定价）。

### 类型 2：变更终止原合同部分义务

```
条件：变更终止了原合同的部分履约义务 + 新增部分不反映单独售价
处理：终止部分从原合同移除 + 新增部分并入原合同
→ 调整累计收入（追溯调整当期）
```

施工企业示例：原合同含装饰工程，业主取消装饰部分并改为幕墙工程。

### 类型 3：变更并入原合同

```
条件：变更不满足类型1和类型2的条件
处理：变更并入原合同 → 调整交易价格和履约进度
→ 按累计调整确认当期收入
```

施工企业最常见的情形：工程量增加/减少、材料规格变更等。

## 二、索赔的可变对价——核心难点

### 确认条件

索赔收入计入交易价格的条件（CAS 14 第 56-58 段）：

| 条件 | 说明 |
|------|------|
| 合同/法律支持 | 合同条款赋予索赔权 or 法律判例支持 |
| 业主已知悉 | 业主已收到索赔报告 |
| 金额能可靠估计 | 有详细的计算依据 |
| **"极可能"不会重大转回** | **最高门槛**——必须高度确定 |

### "极可能"的判断——施工企业实务

| 情形 | 是否达到"极可能" |
|------|-----------------|
| 业主已书面确认索赔金额 | ✅ 是（确认） |
| 业主口头认可但未书面确认 | ⚠️ 需评估（Assumed） |
| 监理已签字认可 | ✅ 是（监理代表业主） |
| 索赔报告已提交但业主未回应 | ❌ 否（不满足"极可能"） |
| 同类索赔在项目其他标段已成功 | ⚠️ 有一定支持但非充分条件 |
| 仅凭行业惯例/口头依据 | ❌ 否 |

### 实务策略——"保守原则"

施工企业确认索赔收入的保守做法：
```
只确认"已签认"的部分：
  变更令（业主已签） → ✅ 确认
  签证单（监理已签） → ✅ 确认
  索赔报告（业主已回复同意）→ ✅ 确认
  索赔报告（已提交但未回复）→ ❌ 暂不确认，附注披露
  口头承诺 → ❌ 不确认
```

## 三、变更/索赔对履约进度的影响

### 变更增加工作量 → 调整预计总成本

```
原合同收入：10,000 万元
变更增加收入：500 万元（已批准）
新交易价格：10,500 万元

原预计总成本：8,000 万元
变更增加成本：400 万元（估计）
新预计总成本：8,400 万元

履约进度重算 → 按新进度确认累计收入
当期收入调整 = 新累计收入 − 旧累计收入
```

### 工期延长（非承包方责任）

工期延长本身的会计处理：
- 不直接增加合同收入（除非合同约定工期奖罚）
- 间接费用的分摊期延长 → 影响各期的成本分配
- 若产生额外成本（窝工/机械闲置）→ 需评估是否属于索赔范围

## 四、可变对价计入交易价格的上限

**"极可能不会发生重大转回"**——这是 CAS 14 对可变对价的最高门槛。

```
估计索赔可能回收金额：1,000 万元

可能性评估：
  - 90% 概率回收 ≥ 900 万元 → 计入交易价格 900 万元
  - 50% 概率回收 1,000 万元 → 极可能转回 → 计入 0
  - 70% 概率回收 800 万元 → 需判断"重大转回" → 可能计入 500-600 万
```

## 五、常见的"伪确认"——审计红线

| 行为 | 问题 | 审计处理 |
|------|------|---------|
| 索赔未确认但提前确认了收入 | 收入高估 | 审计调整冲回 |
| 已提交但业主未回复的索赔全额确认 | 不满足"极可能" | 至少计提减值或冲回 |
| 口头承诺按书面确认处理 | 证据不足 | 审计不同意 |
| 前期索赔被拒绝但仍挂账 | 资产虚增 | 需冲销 or 100%计提减值 |

## 六、自检清单

1. [ ] 未获批的变更/索赔是否满足"极可能不会重大转回"？
2. [ ] 有无仅凭口头承诺确认收入的情况？
3. [ ] 合同变更类型是否正确判断（类型1/2/3）？
4. [ ] 变更是否触发了预计总成本的更新？
5. [ ] 长期未解决的索赔是否评估了可收回性？
6. [ ] 前期高估的索赔收入是否本期冲回？
