# 质保金会计处理实操

## 快速参考

| 阶段 | 会计分录 | 关键科目 |
|------|---------|---------|
| 竣工结算扣留 | 合同资产—质保金 ↑ | 1483 |
| 缺陷责任期 | 预计负债（如有修复义务） | 2801 |
| 实际修复（使用质保金） | 冲减合同资产—质保金 | 1483 |
| 质保期满返还 | 合同资产→银行存款 | 1483→1002 |
| 质保金被业主扣减 | 合同资产↓ + 冲减收入 | 1483 + 6001 |

## 一、竣工结算——质保金扣留

```
工程竣工结算完成，审定结算价（含税）10,000 万元，质保金比例 3%。

确认质保金：
  借：合同资产—质保金（1483）          300 万元
    贷：合同结算—价款结算（5801）       300 万元

剩余结算款：
  借：应收账款—工程款（1122）         9,700 万元
    贷：合同结算—价款结算（5801）     9,700 万元
```

**注意**：质保金是合同资产而非应收账款——因为它的收款权仍附有条件（缺陷责任期内不发生需扣减的缺陷）。

## 二、缺陷责任期——修复义务

### 2.1 发生缺陷需要修复

```
确认修复义务（估计修复成本）：
  借：合同履约成本—保修成本（5601）    XXX
    贷：预计负债—保修义务（2801）       XXX

实际发生修复支出：
  借：预计负债—保修义务（2801）         XXX
    贷：银行存款/原材料/应付职工薪酬     XXX

若实际支出 > 预计负债：
  差额部分：
    借：合同履约成本—保修成本           XXX
      贷：银行存款                       XXX
```

### 2.2 预计负债的评估

每期末评估：
- 已完工项目的缺陷发生概率和历史修复成本
- 正在质保期内的项目数量和金额
- 按**最佳估计数**确认预计负债

```
预计负债 = Σ(各项目质保金额 × 历史缺陷发生率 × 平均修复成本占比)
```

## 三、缺陷责任期满——质保金到期

### 3.1 无缺陷——全额返还

```
缺陷责任期满，业主确认无未修复缺陷：

质保金从合同资产转为无条件收款权：
  借：应收账款—质保金（1122）          300 万元
    贷：合同资产—质保金（1483）        300 万元

收到质保金：
  借：银行存款（1002）                 300 万元
    贷：应收账款—质保金（1122）        300 万元
```

### 3.2 有部分缺陷——业主扣减

```
业主因缺陷未修复，扣减质保金 50 万元：

确认扣减（视为收入减少）：
  借：合同结算—价款结算（5801）         50 万元
    贷：合同资产—质保金（1483）         50 万元

  借：主营业务收入（6001）             45.87 万元（50/1.09）
      应交税费—应交增值税(销项税额)      4.13 万元
    贷：合同结算—收入结转（5801）       50 万元

到期返还剩余部分：
  借：银行存款（1002）                 250 万元
    贷：合同资产—质保金（1483）        250 万元
```

### 3.3 业主拖延支付

质保期满后业主拖延支付（施工企业最常见的质保金问题）：

```
质保期满 → 合同资产转应收账款（完全合法）：
  借：应收账款—质保金（1122）          300 万元
    贷：合同资产—质保金（1483）        300 万元

按预期信用损失模型计提坏账（账龄 2-3 年的政府应收款示例）：
  借：信用减值损失（6702）              60 万元（300 × 20%）
    贷：坏账准备—质保金（1231-02）      60 万元
```

## 四、质保金被业主扣减后的 VAT 处理

业主扣减质保金意味着施工方实际收款减少 → 对应的 VAT 处理：

| 情形 | VAT 处理 |
|------|---------|
| 结算时已全额开票（含质保金） | 被扣减部分 → 开具红字发票 → 冲减销项税 |
| 结算时未开质保金部分发票 | 被扣减部分不开票 → 不产生纳税义务 |
| 业主用质保金自行修复（不给施工方机会） | 仍需冲减收入和销项税 |

## 五、质保金的资产减值

施工企业质保金的资产减值信号：

| 信号 | 处理 |
|------|------|
| 业主财务状况恶化 | 加速计提减值 |
| 项目所在地区财政困难（政府项目） | 评估但一般不作为减值依据（政府不会"破产"） |
| 质保期满超 2 年未收回 | 考虑大幅计提减值（30-50%） |
| 业主与承包方发生纠纷 | 评估回收可能性 |
| 业主方人事变动（推诿拖延） | 不构成减值依据但需加强催收 |

3. [ ] 质保期满是否正确转为应收账款？
4. [ ] 业主拖延支付的质保金是否按预期信用损失模型计提坏账？
5. [ ] 业主扣减的质保金是否开具了红字发票（如已开票）？
6. [ ] 有无长期挂账未转应收的质保金？
