# 工程保险理赔款会计处理

## 快速参考

| 险种 | 保障范围 | 理赔款处理 |
|------|---------|-----------|
| 建筑工程一切险 | 工程本身的物质损失（自然灾害/意外事故） | **冲减工程成本** or 营业外收入 |
| 安装工程一切险 | 安装工程的物质损失 | 同 |
| 第三者责任险 | 对第三方的财产损失/人身伤害赔偿 | 冲减赔偿支出 |
| 建工意外险 | 施工人员意外伤害 | 冲减赔偿支出（差额进损益） |

## 一、理赔款的基本原则

### 补偿原则——冲减对应的损失/支出

保险理赔款是对**已发生损失/支出的补偿**，会计上应**冲减对应的损失或支出**，而非直接确认为收入。

```
工程损失发生 → 确认损失/支出
    ↓
保险理赔确认 → 冲减前述损失/支出
    ↓
差额处理：
  赔付 > 损失 → 差额计入营业外收入（极少见）
  赔付 < 损失 → 差额留在损失/支出（自留风险）
```

## 三、理赔确认的时点

| 时点 | 是否可确认 |
|------|-----------|
| 事故刚发生，尚未报险 | ❌ |
| 已报险，保险公估进场但未定损 | ❌ |
| 保险公估已出具定损报告 | ⚠️ 可估计（Assumed） |
| 保险公司已确认赔付金额 | ✅ 可以确认 |
| 理赔款已到账 | ✅ 确认（最确定） |

**保守做法**：收到理赔款才冲减损失（最保守），或保险公司书面确认赔付后确认（正常做法）。

## 四、VAT 处理

工程保险理赔款**不属于增值税应税收入**——不缴 VAT，不开具增值税发票。

但需注意：
- 支付修复费用时取得的增值税进项（13%/9%）**可以正常抵扣**
- 理赔款是补偿，不影响进项抵扣权

## 五、常见问题

| 问题 | 处理 |
|------|------|
| 理赔款跨年到账（损失在去年） | 理赔款在到账当期冲减成本（不追溯调整去年） |
| 理赔金额大于已确认的损失 | 差额计入**营业外收入** |
| 保险公司拒赔 | 损失金额全部自行承担 |
| 理赔款用于购买新设备（非修复） | 如全额重置 → 按新资产入账 |
| 免赔额部分 | 施工企业自行承担，不确认理赔 |

## 六、自检清单

1. [ ] 理赔款是否冲减了对应的损失/支出（而非直接确认收入）？
2. [ ] 理赔确认时点是否合适（保险公司确认赔付后）？
3. [ ] 免赔额部分是否已自行承担？
4. [ ] 理赔款跨年是否在到账当期处理？
5. [ ] VAT：理赔款不缴 VAT，修复进项可正常抵扣——是否正确？

## 免责声明

具体理赔处理以保险合同条款和保险公估定损报告为准。
