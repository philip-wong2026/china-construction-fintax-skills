# 安全生产费会计处理全流程

## 快速参考

| 工程类别 | 计提比例 | 计提基数 |
|---------|---------|---------|
| 矿山工程 | 3.5% | 当月建安产值 |
| 铁路/房建/城轨 | 3.0% | 当月建安产值 |
| 水利水电/电力 | 2.5% | 当月建安产值 |
| 机电安装/化工/通信 | 2.0% | 当月建安产值 |
| 市政/港口/公路 | 1.5% | 当月建安产值 |

## 一、计提

### 月度计提分录

```
借：合同履约成本—安全生产费（5601）   XXX
  贷：专项储备（4301）                  XXX

计提金额 = 当月完成建安产值 × 计提比例
```

**关键**：计提时**不涉及**银行存款——专项储备是一个权益类科目（4301），代表已计提但尚未使用的安全生产费"储备池"。

### 计提基数

"当月完成建安产值" = 当月按履约进度确认的施工收入（不含税）对应的产值，不是开票金额或收款金额。

## 二、使用——费用性支出

### 安全培训、防护用品等

```
借：专项储备（4301）                    XXX
    应交税费—应交增值税(进项税额)       XXX
  贷：银行存款                          XXX
```

**同时**（冲减专项储备的对应成本不再重复计入损益）——这是专项储备会计的核心特点。

### 使用范围（只能用于）

- 安全防护设施设备购置/维护
- 应急演练和救援装备
- 安全培训教育
- 安全评价检测检验
- 现场作业人员安全防护用品
- 安全生产适用的新技术新标准新工艺推广

## 三、使用——资本性支出

### 购置安全设备

```
购置时：
  借：在建工程 / 固定资产               XXX
      应交税费—应交增值税(进项税额)     XXX
    贷：银行存款                        XXX

一次性计提折旧：
  借：专项储备（4301）                  XXX
    贷：累计折旧                        XXX（全额）
```

**关键**：安全设备购入后，通过专项储备**一次性全额计提折旧**，设备账面净值直接归零。这是安全生产费会计最独特的地方。

## 四、资金拨付（总分包之间）

| 节点 | 比例 | 拨付方→接收方 |
|------|------|-------------|
| 开工1月内 | ≥ 50% | 业主→总承包 |
| 分包开工1月内 | ≥ 50% | 总承包→分包 |
| 竣工后 | 余款 | 退回业主 |

### 总包收到安全生产费

```
借：银行存款                          XXX
  贷：预收账款—安全生产费             XXX
（收到后按规定计提和使用）
```

### 总包拨付给分包

```
借：合同履约成本—分包安全生产费       XXX
  贷：银行存款                        XXX
```

## 五、税务处理差异——年度汇算清缴重点

| 时间点 | 会计 | 税务 | 差异 |
|--------|------|------|------|
| 计提时 | 借：合同履约成本 | **不得扣除** | 纳税**调增** |
| 实际支出时 | 借：专项储备 | **允许扣除** | 纳税**调减** |

### A105000 纳税调整明细表

```
本年计提安全生产费：100 万元
本年实际使用：      80 万元
未使用余额：        20 万元

纳税调增：100 万元（计提全额不得扣除）
纳税调减： 80 万元（实际支出允许扣除）
净调增：   20 万元（= 年末未使用余额）
```

**施工企业注意**：年末未使用的安全生产费需纳税调增，这会增加当期应纳所得税。
但是否能足额使用取决于项目的安全投入节奏——建设前期使用少、后期使用多，形成"前期多缴税、后期少缴税"的时间性差异。

## 六、竣工时——余款退回业主

安全生产费在竣工后的余款**不可截留**，需退回业主：

```
借：专项储备（4301）                  XXX
  贷：银行存款                        XXX
```

## 七、自检清单

1. [ ] 计提比例是否按工程类别正确选择？
2. [ ] 计提基数是否为当月建安产值（非开票/收款）？
3. [ ] 费用性支出和资本性支出的分录是否正确区分？
4. [ ] 安全设备是否一次性全额计提折旧？
5. [ ] 分包单位是否收到 ≥50% 的首期拨付？
6. [ ] 年末未使用的安全生产费是否纳税调增？
7. [ ] 竣工后余款是否退回业主？
