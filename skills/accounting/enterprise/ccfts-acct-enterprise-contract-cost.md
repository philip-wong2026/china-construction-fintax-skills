# 预计总成本管理与亏损合同处理

## 快速参考

| 项目 | 规则 |
|---|---|
| 预计总成本更新频率 | **至少每季度**（季末重估） |
| 亏损合同确认时点 | 预计总成本 > 合同总收入的**当期** |
| 预计损失计提 | 亏损金额 × (1 − 履约进度) |
| 分包成本归集 | 按分包合同进度匹配总包进度 |

## 一、预计总成本动态调整

### 1.1 调整触发因素

| 类别 | 触发因素 | 影响方向 |
|------|---------|---------|
| 材料价格 | 钢材/水泥/沥青价格大幅变动 | ↑ or ↓ |
| 人工成本 | 工人工资上涨、社保基数调整 | ↑ |
| 工程量变化 | 设计变更/地质变化 | ↑ or ↓ |
| 工期变化 | 赶工/延误 | ↑ |
| 分包价格 | 分包商涨价/竞争降价 | ↑ or ↓ |
| 政策变化 | 环保要求/安全标准提高 | ↑ |

### 1.2 调整的会计影响

```
原预计总成本：8,000 万元
新预计总成本：8,500 万元（增加 500 万元）
合同总收入：10,000 万元（不变）

原履约进度：40%（已发生 3,200 / 8,000）
新履约进度：37.6%（已发生 3,200 / 8,500）

原毛利率：(10,000 − 8,000) / 10,000 = 20%
新毛利率：(10,000 − 8,500) / 10,000 = 15%

当期影响：
  收入调整：10,000 × (37.6% − 40%) = −240 万元
  说明：前期进度高估 → 当期收入减少
```

**关键**：预计总成本的变动**追溯调整**当期及未来的收入确认（不追溯调整已确认的以往期间）。

## 二、亏损合同——施工企业的达摩克利斯之剑

### 2.1 识别

亏损合同 = 预计总成本 > 合同总收入（含可变对价中"极可能"实现的部分）

| 预警信号 | 风险程度 |
|---------|---------|
| 材料价格持续上涨 + 合同无调价条款 | 高 |
| 地质条件超预期（隧道/地铁） | 高 |
| 业主频繁变更但未增加合同价款 | 中-高 |
| 工期延误（业主原因）但索赔尚未确认 | 中 |
| 分包商违约/退出 | 中 |
| 投标时报价过低（"低价中标"） | 先天缺陷 |

### 2.2 计提预计损失

CAS 14 要求：亏损合同**必须立即确认预计损失**，不能等到项目结束。

```
预计总损失 = 预计总成本 − 合同总收入（不可收回部分）

已通过履约进度确认的累计损失
  = 预计总损失 × 履约进度

还需计提的预计损失（在 B/S 中体现）
  = 预计总损失 × (1 − 履约进度)
```

**分录**：
```
借：主营业务成本（6401）              XXX
  贷：预计负债—亏损合同（2801）       XXX
```

### 2.3 后续期间

每期末重估预计总成本：
- 若亏损扩大 → 补提预计负债
- 若亏损缩小（如索赔成功增加了合同收入） → 冲回部分预计负债
- 项目完工 → 冲销全部预计负债

**施工企业实务痛点**：
亏损合同计提预计损失会直接冲击当期利润。很多企业在"是否计提"的时点上犹豫，
但审计（尤其四大）对此非常严格——识别出亏损即要求计提，不能以"后续可能扭亏"为由延迟。

## 三、分包成本归集

### 3.1 分包成本的匹配原则

| 分包类型 | 归集时点 | 匹配方法 |
|---------|---------|---------|
| 专业分包（按进度结算） | 与总包履约进度**同期** | 分包进度 = 总包确认的进度 |
| 劳务分包（按月/按量） | 实际发生期 | 按人工费同期归集 |
| 分包变更/签证 | 与总包确认同步 | 总分包同步 |

### 3.2 总分包的成本联动

```
总包确认收入时：
  借：合同结算—收入结转               XXX
    贷：主营业务收入                   XXX
  借：主营业务成本（含分包成本）        XXX
    贷：合同履约成本—分包成本          XXX

关键：分包成本的结转必须与总包收入确认的履约进度匹配。
不能出现"总包已确认收入但分包成本未确认"或反向的错配。
```

## 四、自检清单

1. [ ] 预计总成本是否在最近季度末更新过？
2. [ ] 有没有新出现的亏损合同预警信号？
3. [ ] 已知亏损合同是否已计提足额预计损失？
4. [ ] 预计总成本变动是否追溯调整了当期收入？
5. [ ] 分包成本是否与总包履约进度匹配？
6. [ ] 材料价格变动是否已反映到预计总成本中？

## 免责声明

亏损合同的判断涉及对未来成本的预估，具有不确定性。预计损失的计提金额和时点以审计确认为准。
