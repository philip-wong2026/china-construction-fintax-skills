# 合同履约成本（5601）归集与结转

> **SLOT 说明**：本文件为合同履约成本处理的通用 Base。适用于 {ENTITY_LABEL}（{ENTITY_ALIAS}）。

## 快速参考

| 项目 | 规则 |
|---|---|
| 科目编号 | **5601 合同履约成本** |
| 四大要素 | 人工费 + 材料费 + 机械使用费 + 间接费用 |
| 归集方式 | 按**单项合同**设置明细科目 |
| 结转时点 | 按履约进度（时段确认）→ 转入合同结算（5801） |
| 期末余额 | 未完工部分留在 5601；已完工且已结算→对应冲销 |

## 一、四要素归集标准

### 1.1 人工费

| 费用项目 | 归集依据 | 注意 |
|---------|---------|------|
| 自有工人工资 | 工资表 + 考勤 | 区分生产工人 vs 管理人员（后者进间接费） |
| 劳务分包费 | 分包合同 + 结算单 | 取得增值税发票 |
| 农民工工资专户代发 | 银行代发记录 | 与工资表核对一致 |
| 社保/公积金 | 缴费清单 | 按受益对象分摊 |

### 1.2 材料费

| 费用项目 | 计价方法 | 注意 |
|---------|---------|------|
| 主要材料（钢材/水泥/砂石） | 实际成本（加权平均 or 个别计价） | 甲供材不进入施工方成本 |
| 辅助材料 | 实际成本 | |
| 周转材料（模板/脚手架） | 摊销 | 按使用次数或工期摊销 |
| 商品混凝土 | 实际成本 | 注意 3% or 13% 进项税 |

### 1.3 机械使用费

| 费用项目 | 归集方式 |
|---------|---------|
| 自有机械 | 折旧费 + 维修费 + 燃料动力费 + 操作人员工资 |
| 租赁机械（按台班） | 租赁费 |
| 租赁机械（按月） | 租赁费（按月分摊） |

### 1.4 间接费用

| 费用项目 | 分摊基础 |
|---------|---------|
| 项目管理人员薪酬 | 按各合同的直接成本比例 |
| 现场临建摊销 | 按各合同的工期/面积 |
| 水电费 | 按实际用量 or 产值比例 |
| 检验试验费 | 按直接归属 |
| 安全生产费（费用性支出） | 按法规要求 |

## 二、成本归集的账务处理

### 2.1 发生时

```
借：合同履约成本—xx项目—人工费     XXX
    合同履约成本—xx项目—材料费     XXX
    合同履约成本—xx项目—机械费     XXX
    合同履约成本—xx项目—间接费     XXX
    应交税费—应交增值税(进项税额)   XXX
  贷：银行存款/应付账款等           XXX
```

### 2.2 期末结转（按时段确认收入时）

```
借：合同结算—收入结转（5801）       XXX（本期履约进度对应的成本）
  贷：合同履约成本—xx项目           XXX
```

**注意**：Cost 转入的不是"主营业务成本"（6401），而是通过 5801（合同结算）的过渡科目，最终在利润表中体现为营业成本。这是 CAS 14 与旧 CAS 15 的重大区别。

### 2.3 期末余额的含义

```
合同履约成本期末借方余额
  = 已发生但尚未结转至合同结算的成本
  = 尚未达到收入确认条件的在建工程成本
  → 在 B/S 中列示为"合同资产"或其他非流动资产
```

## 三、完工进度（履约进度）计量

见 `ccfts-intel-mof-cas14-revenue`，本文件仅涉及成本角度的进度计算。

### 投入法（成本比例）— 最常用

```
履约进度 = 累计已发生成本 / 预计总成本

本期应结转成本 = 预计总成本 × 履约进度 − 前期已结转成本
```

**关键约束**：
- 未安装材料（已到场但未安装）的 cost 不能计入分子
- 废品/返工/浪费的 cost 不能计入分子（除非合同约定业主承担）

### 产出法（工程计量）

以监理/业主确认的工程计量为进度依据。适用于产出可明确计量的项目（如隧道掘进米数、土方量）。

## 四、预计总成本的动态调整

预计总成本不是一成不变的，需持续评估：

| 调整触发条件 | 处理 |
|-------------|------|
| 材料价格大幅变动 | 调整预计总成本 → 影响当期及未来毛利率 |
| 设计变更导致工程量变化 | 调整预计总成本 |
| 工期延误（非承包方责任） | 评估额外成本 → 调整预计总成本 |
| 发现地质条件与勘察不符 | 评估额外成本 + 索赔可能性 |
| 亏损合同（预计总成本 > 合同总收入） | **立即确认预计损失**（见 `ccfts-acct-ent-contract-cost`） |
