# 个体施工队/挂靠模式

## 快速参考

| 经营模式 | 法律地位 | 税务处理 | 风险等级 |
|---------|---------|---------|---------|
| 个体工商户 | 合法 | 核定征收 or 查账征收 | 低 |
| 挂靠（有资质企业） | ⚠️ 灰色 | 被挂靠方开票纳税 | **高** |
| 纯包工头（无资质） | ❌ 违法 | 个人劳务报酬 or 个体户 | 高 |

## 一、合法模式——个体工商户

### 注册与资质

个体施工队可以注册为个体工商户，承接小型/零散工程：
- 不需建筑业企业资质（小型工程/维修/装修）
- 需要营业执照（经营范围含建筑施工）
- 可申请小规模纳税人

### 税务处理——核定征收 vs 查账征收

| 方式 | 适用 | CIT/个税 |
|------|------|---------|
| 核定征收 | 账簿不健全的小型个体户 | 按收入 × 核定利润率 × 税率 |
| 查账征收 | 账簿健全 | 按实际利润纳税 |

**核定征收的优势**：
```
年收入 200 万元（不含税）
核定利润率：10%（建筑业的典型核定利润率）
核定应税所得：200 × 10% = 20 万元
个税：20 × 20% − 1.05 = 2.95 万元（经营所得五级累进税率）

VAT：200 × 1% = 2 万元（小规模，暂按 1%）
总税负：约 5 万元 / 200 万元 = 2.5%
```

### 个体施工队的局限

| 局限 | 影响 |
|------|------|
| 不能开具 9% 专票 | 只能开 1% 普票 or 代开 1% 专票 → 总包方通常不接受 |
| 无法承接大型项目 | 无建筑业企业资质 |
| 无限责任 | 个人财产与经营财产不分离 |
| 难以融资 | 银行授信极难 |

## 二、灰色模式——挂靠

### 挂靠的运作方式

```
个体施工队（"挂靠方"）
  │
  ├── 找到工程项目 → 以被挂靠方名义投标/签约
  │
  ├── 被挂靠方（有资质企业）：
  │     - 签订施工合同（名义承包方）
  │     - 开具 9% 专票给业主
  │     - 收取挂靠费（合同金额的 2-5%）
  │
  └── 个体施工队：
        - 实际施工
        - 承担全部成本
        - 收到被挂靠方转付的工程款（扣除挂靠费）
```

### 挂靠的税务处理——被挂靠方视角

| 会计项目 | 处理 |
|---------|------|
| 收入 | 全额确认施工收入（6001） |
| 成本 | 个体施工队提供的发票（材料/人工/机械） |
| VAT | 全额开 9% 专票 + 抵扣进项 |
| 挂靠费 | "管理费收入"（6051） or 隐含在利润中 |

### 挂靠的税务处理——挂靠方（个体施工队）视角

挂靠方需要向被挂靠方提供发票以结算工程款：
- 材料采购：取得 13% 专票 → 提供给被挂靠方抵扣
- 人工成本：需通过劳务公司开票 or 以工资形式发放
- 利润部分：被挂靠方扣除挂靠费后转付（不能直接转——无票支出）

### 挂靠的法律风险

| 风险 | 说明 |
|------|------|
| **合同无效** | 最高法院司法解释：挂靠合同无效 |
| **连带责任** | 被挂靠方对工程质量/安全/债务承担连带责任 |
| **行政处罚** | 挂靠属于"出借资质" → 罚款 + 资质降级 |
| **税务风险** | 挂靠方利润转付无发票 → 被挂靠方 25% CIT + 挂靠方个税 |

**结论**：挂靠是建筑业普遍存在但法律不认可的模式。本文件不鼓励挂靠，仅做实务描述。

## 三、合法替代方案

### 方案 1：注册为合法分包商

个体施工队 → 注册有限责任公司 → 取得建筑业企业资质 → 以合法分包商身份与总包签约。

| 优势 | 劣势 |
|------|------|
| 合法合规 | 需要注册公司 + 申请资质 |
| 可开 9% 专票 | 管理成本增加 |
| 有限责任 | 税务从核定征收变为查账征收 |

### 方案 2：劳务分包模式

个体施工队 → 提供劳务（非整体工程） → 作为劳务分包 → 不涉及"挂靠"的合同无效问题。

需取得施工劳务资质（相对容易获取）。

## 四、自检清单

1. [ ] 施工主体是否具有合法经营资格（营业执照/个体户执照）？
2. [ ] 是否避免了"挂靠"模式（合同无效风险）？
3. [ ] 如需承接工程，是否考虑注册有限责任公司 + 申请资质？
4. [ ] VAT：个体户是否能满足业主/总包的发票需求？
5. [ ] CIT：是否了解核定征收 vs 查账征收的差异？

## 免责声明

挂靠是法律不认可的模式，个体施工队应向合法合规的经营模式转型。本文件不构成对挂靠的认可。
