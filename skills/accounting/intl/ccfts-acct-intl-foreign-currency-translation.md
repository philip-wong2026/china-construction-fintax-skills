# 境外施工项目外币报表折算

## 快速参考

| 项目 | 规则 |
|---|---|
| 功能货币判断 | 主要影响商品/服务价格的货币 + 融资活动货币 |
| 记账本位币 | 境外子企业可能为当地货币（非人民币） |
| 报表折算 | 资产/负债→期末汇率；收入/费用→近似汇率；权益→历史汇率 |
| 折算差额 | 计入**其他综合收益**（OCI），不进入当期损益 |
| 合并层面 | 折算后纳入集团合并报表 |

## 一、功能货币 vs 记账本位币

### 功能货币的判断（CAS 19）

| 判断因素 | 示例 |
|---------|------|
| 影响商品/服务价格的货币 | 境外项目以美元结算 → 功能货币为美元 |
| 融资活动的主要货币 | 项目贷款为美元 → 支持功能货币为美元 |
| 成本支出的主要货币 | 当地工人工资以当地货币支付 → 支持功能货币为当地货币 |

**施工企业的典型情况**：
- 境外EPC项目以美元结算收入、从中国采购材料（人民币）、当地工人（当地货币）→ 功能货币为美元
- 境外子企业主要在当地经营 → 功能货币为当地货币

### 记账本位币

境外子企业的记账本位币可能是当地货币或美元。向中国母公司报送时需折算为人民币。

## 二、外币报表折算方法

### 资产负债表

| 项目 | 折算汇率 |
|------|---------|
| 资产 | 期末汇率（资产负债表日） |
| 负债 | 期末汇率 |
| 实收资本/资本公积 | 历史汇率（投入时） |
| 盈余公积/未分配利润 | 混合（各期折算结果累计） |
| **折算差额** | 轧差 → 计入 OCI |

### 利润表

| 项目 | 折算汇率 |
|------|---------|
| 收入/费用 | 交易发生日的即期汇率 or **近似汇率**（月/季平均汇率） |
| 利润分配 | 实际分配日的汇率 |

### 折算差额（其他综合收益）

```
折算差额 = 资产折算后 − 负债折算后 − 权益折算后（历史汇率部分）

→ 计入"其他综合收益—外币报表折算差额"
→ 在合并 B/S 的所有者权益下列示
→ 不影响当期利润
```

## 三、纳入集团合并——双层次折算

### 境外子企业 → 中国母公司

```
境外子企业（当地货币账本）
  ↓ 折算为人民币（CAS 19 现行汇率法）
  ↓
境外子企业人民币报表
  ↓ 纳入合并（CAS 33 控制三要素判断）
  ↓
集团合并报表（人民币）
```

### 施工企业特殊问题——境外项目部的折算

如果境外项目部不是独立法人（分公司/项目部）：
- 其资产/负债/收入/成本直接纳入上级法人的账套
- 日常按交易发生日汇率确认
- 期末货币性项目（现金/应收/应付）按期末汇率重估 → 汇兑损益进当期利润

## 四、汇兑损益——货币性项目的期末重估

### 区分

| 类型 | 处理 |
|------|------|
| 货币性项目的期末重估（现金/应收/应付/借款） | **汇兑损益进当期利润表**（财务费用—汇兑损益） |
| 境外子企业报表折算差额 | **OCI**（不进当期利润） |

### 货币性项目的期末重估

```
境外项目有美元应收账款 100 万美元：

交易发生日：1 USD = 7.20 CNY → 720 万元人民币
期末报表日：1 USD = 7.50 CNY → 重估为 750 万元人民币
汇兑收益：30 万元 → 计入财务费用—汇兑收益（6603）
```

**施工企业影响**：境外项目的应收账款/应付账款/外币借款的汇率波动可能产生较大的汇兑损益，直接影响当期利润。对于汇率波动剧烈的国家，这是境外项目利润的重大不确定因素。

## 五、高通胀经济体的特殊处理

若境外项目所在国为高通胀经济体（三年累计通胀 > 100%）：
- 先对境外报表进行**通胀重述**（消除通胀影响）
- 再用现行汇率法折算为人民币

目前适用此情况的包括部分非洲/南美/中亚国家。

## 六、自检清单

1. [ ] 境外项目的功能货币是否已正确判断？
2. [ ] 资产负债表折算是否使用期末汇率？
3. [ ] 利润表折算是否使用近似汇率（月/季平均）？
4. [ ] 折算差额是否计入 OCI（非当期利润）？
5. [ ] 期末是否对货币性项目进行了汇兑损益重估？
6. [ ] 合并层面是否包含了境外子企业折算后的报表？

## 免责声明

外币折算的具体处理因境外项目的功能货币和当地会计/税务法规而异。
