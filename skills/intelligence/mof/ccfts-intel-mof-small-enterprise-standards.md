# 小企业会计准则——施工企业适用性

## 快速参考

| 项目 | 标准 |
|---|---|
| 施工企业小型标准 | 年营业收入 < 6000 万 **且** 总资产 < 5000 万 |
| 能否适用小企业会计准则？ | **技术上可以，但强烈不建议** |
| 能否从企业会计准则切换到小企业准则？ | **禁止**（一旦选定企业会计准则，不可逆向切换） |
| 年营业收入 > 6 亿属大型施工企业 | 必须使用企业会计准则 |

## 施工企业为何不建议使用小企业会计准则

### 缺失的关键科目

小企业会计准则**没有**以下施工核心科目：

| 缺失科目 | 影响 |
|---|---|
| 合同资产(1483)、合同履约成本(5601) | 无法正确记录 CAS 14 收入确认 |
| 工程结算(5801)、合同负债 | 无法区分已结算和未结算 |
| 专项储备 | 安全生产费无法正常计提和使用 |
| 递延所得税资产/负债 | 使用应付税款法，无法反映暂时性差异 |
| 资产减值准备（除坏账外） | 合同资产减值完全缺失 |

### 行业实操

即使小型施工企业也普遍采用**企业会计准则**：
- 甲供材、分包、清包工等业务模式在小企业准则下账务处理困难
- 税务申报（VAT 预缴、跨区域 CIT）需要更细颗粒度的科目支持
- 完工百分比/时段确认收入在小企业准则下无法准确核算

## 企业规模划分标准（建筑业）

| 指标 | 大型 | 中型 | 小型 | 微型 |
|---|---|---|---|---|
| 营业收入(Y) | Y ≥ 8 亿 | 6000万 ≤ Y < 8亿 | 300万 ≤ Y < 6000万 | Y < 300万 |
| 资产总额(Z) | Z ≥ 8 亿 | 5000万 ≤ Z < 8亿 | 300万 ≤ Z < 5000万 | Z < 300万 |

## 小规模纳税人 VAT 提示

小型施工企业可能同时为**小规模纳税人**（年销售额 ≤ 500 万）：
- 征收率 3%（2027年前暂按1%执行）
- 不能开具增值税专用发票（需代开）
- 不能抵扣进项税额

**建议**：即使企业规模小，若客户为一般纳税人（需要专票抵扣），应主动申请一般纳税人资格。
