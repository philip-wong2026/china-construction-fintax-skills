# 专项债基建项目——法规与会计框架

## 快速参考

| 项目 | 说明 |
|---|---|
| 发行主体 | **省级政府**（市县级政府由省级代发） |
| 资金流向 | 省级财政 → 市县财政 → 项目单位 → 施工企业 |
| 施工企业角色 | **EPC 承包商**（不是借款人，不是项目业主） |
| 收入确认 | 正常按时段确认施工收入（与资金来源无关） |
| 回款风险 | **低**（财政资金，但拨付进度可能延迟） |

## 一、专项债基本框架

专项债是地方政府为有一定收益的公益性项目发行的债券，以项目对应的政府性基金收入或专项收入偿还。

### 与一般债/城投债的区别

| 类型 | 发行主体 | 偿还来源 | 施工企业角色 |
|------|---------|---------|------------|
| **专项债** | 省级政府 | 项目自身收益（土地出让/通行费/票款） | EPC 承包商 |
| 一般债 | 省级政府 | 一般公共预算收入 | EPC 承包商 |
| 城投债 | 城投公司（企业） | 城投公司自身收入 | EPC 承包商（但城投可能是关联方） |
| PPP 社会资本融资 | 项目公司（SPV） | 项目公司运营收益 | **股东 + EPC 承包商** |

> **关键区分**：专项债项目 ≠ PPP 项目。专项债项目业主是政府单位/事业单位，施工企业只是 EPC 承包商，没有投资和运营责任。

## 二、专项债资金流向

```
省级财政厅发行专项债
  ↓ 拨付
市/县财政局
  ↓ 拨付
项目单位（政府单位/事业单位/城投公司）
  ↓ 按工程进度支付
施工企业（大型建筑央企）
```

### 施工企业的收款特征

| 特征 | 影响 |
|------|------|
| 资金来源稳定（财政预算） | 坏账风险低 |
| 拨付审批链条长 | **收款周期可能较长**（财政→项目单位→施工企业） |
| 受地方财政状况影响 | 欠发达地区的拨付可能延迟 |
| 专款专用 | 资金不能挪用，但确保了项目有资金来源 |

## 三、施工企业的会计处理

### 核心原则：与普通施工合同完全一致

专项债只是**资金来源**不同，不改变施工企业的收入确认方式：

```
正常按时段确认收入（CAS 14）：
  借：合同履约成本（5601）        XXX
    贷：银行存款/应付账款等        XXX

  借：合同结算—收入结转（5801）   XXX
    贷：主营业务收入（6001）      XXX

  借：应收账款（1122）            XXX
    贷：合同结算—价款结算（5801）  XXX
```

**不需要**因为资金来源是专项债而做任何特殊会计处理。

### 唯一需要关注的：合同资产 vs 应收账款的时点

| 普通项目 | 专项债项目 |
|---------|-----------|
| 进度款申请 → 业主审核 → 付款 | 同，但业主付款依赖于财政拨付 |
| 已完工未结算 → 合同资产 | **同**——即使资金来自专项债，未结算就是合同资产 |
| 已结算未收款 → 应收账款 | **同**——结算后转入应收账款 |
| 收款延迟 | 一般商业信用风险 | 财政拨付延迟风险（但通常不是信用减值） |

### 不应做的处理

❌ 因为项目"有专项债资金保障"就提前确认收入
❌ 在未达到收入确认条件时（如未通过验收），仅凭"专项债已到位"就确认
❌ 将专项债资金当作"业主预付款"（预收账款）

## 四、"债贷组合"项目

部分大型项目采用"专项债 + 银行贷款"组合融资：

```
专项债（资本金部分）→ 项目单位
银行贷款 → 项目单位
  ↓ 两者合并支付
施工企业
```

对施工企业而言，无需区分资金来源，统一按合同收款即可。
但需注意：银行贷款部分可能附带**资金监管条款**（银行直接支付给施工方，不经项目单位），这可能改变收款的对手方。

## 五、与 PPP 的核心差异总结

| 维度 | 专项债项目 | PPP 项目 |
|------|----------|---------|
| 施工企业角色 | **承包商**（只做，不投） | **股东+承包商**（又投又做） |
| 是否控股项目公司 | 不 | 通常是最大股东 |
| 是否合并项目公司 | 不适用 | 按 CAS 33 判断（国资委要求原则性合并） |
| 收入确认 | 仅施工收入（6001） | 施工收入 + 投资收益 + 运营收入 |
| 资产负债表影响 | 仅合同资产/应收账款 | 含 PPP 项目资产/负债（如合并） |
| 回款风险 | 财政拨付延迟 | 运营期收益实现风险 |
| 资产负税率影响 | 低（表外） | 高（如合并 PPP 项目公司） |

## 六、风险提示

1. **财政拨付延迟**：虽然资金来源有保障，但地方财政紧张时拨付可能显著延迟
2. **配套融资不到位**：专项债通常只覆盖部分投资（如 50%），剩余配套资金如不到位，项目可能停工
3. **收益不及预期**：若项目自身收益不足以偿还专项债本息，可能影响后续项目的专项债额度分配
4. **隐性债务红线**：施工企业不应为专项债项目提供任何形式的"垫资"或"回购承诺"——这是隐性债务的红线

## 免责声明

专项债政策由财政部和国家发改委持续更新。具体项目的专项债安排以省级财政厅批复文件为准。
