# CAS 14 新收入准则（施工企业适用）

## 快速参考

| 项目 | 旧 CAS 15 | 新 CAS 14 |
|---|---|---|
| 核心概念 | 完工百分比法 | 履约义务 + 控制权转移 |
| 关键科目 | 工程施工、工程结算 | 合同履约成本、合同结算、合同资产/负债 |
| 资产负债表列示 | 存货或预收账款 | 合同资产（1483）或合同负债（5801） |
| 生效日期 | — | 境内外上市：2020.1.1；非上市：**2021.1.1** |

## 五步法模型

1. **识别合同**：客户合同满足5项条件（批准、权利义务、支付条款、商业实质、对价可收回）
2. **识别履约义务**：施工合同可能含多义务（设计+施工+设备安装）
3. **确定交易价格**：固定价 + 可变对价（调价、索赔、奖励）— 仅"极可能不会重大转回"时确认
4. **分摊交易价格**：按单独售价比例
5. **在履约时点/时段确认收入**：施工企业通常属于"时段"（三个条件满足其一）

## 时段确认三条件（满足任一即可）

1. 客户在企业履约的同时即取得并消耗履约利益
2. 客户能控制履约过程中的在建商品（如客户在客户土地上施工）
3. 商品不可替代 + 有强制收款权

施工合同**通常满足条件2或3**。

## 新科目体系

| 新科目 | 对应旧科目 | 说明 |
|---|---|---|
| 合同履约成本(5601) | 工程施工—合同成本 | 实际发生的建造成本 |
| 合同结算—价款结算 | 工程结算 | 已与客户结算的金额 |
| 合同结算—收入结转 | — | 已确认的累计收入 |
| 合同资产(1483) | 已完工未结算 | "结算 < 收入"的部分 |
| 合同负债(5801贷) | 预收账款(工程款) | "结算 > 收入"或预收款 |
| 合同取得成本 | — | 销售佣金等增量成本 |
| 工程施工—合同毛利 | — | **已取消** |

## 施工企业完工进度计量

| 方法 | 适用 | 注意 |
|---|---|---|
| 投入法（成本比例） | 最常用 | 未安装材料价值不能计入进度 |
| 产出法（工程计量） | 监理确认进度时 | 需剔除业主提供的材料设备 |

## PPP 项目（企业会计准则解释第14号）

| 模式 | 资产类别 | 收入确认 |
|---|---|---|
| 政府付费/可行性缺口 | 长期应收款（金融资产） | 利息收入 |
| 使用者付费 | 无形资产-特许经营权 | 运营收入 |
| 混合模式 | 金融资产+无形资产 | 分别确认 |

**大型建筑央企案例**：2021年采用解释第14号时，某央企将大量合同资产（在建PPP项目）重分类至其他非流动资产。

## 自检清单

1. [ ] 合同是否识别了多个履约义务？
2. [ ] 完工进度方法是否与上年一致（不可随意变更）？
3. [ ] 未安装材料是否从完工进度中剔除？
4. [ ] 合同资产是否已足额计提减值（1484）？
5. [ ] PPP 项目是否按解释第14号正确分类？
