# CAS 33 合并财务报表准则

## 快速参考

| 项目 | 要求 |
|---|---|
| 合并基础 | **控制**（三要素） |
| 控制三要素 | 权力 + 可变回报 + 权力-回报的联系 |
| 评估频率 | **持续**评估，情势变更时重评 |
| 境外子企业 | 调整为 PRC GAAP 后再合并 |
| 会计政策统一 | 所有纳入合并范围的实体使用统一会计政策 |

## 控制的三个要素

1. **权力**：拥有对被投资方的现时权力（通过表决权、合同安排等）
2. **可变回报**：因参与被投资方而享有可变回报（股利、利息、服务费等）
3. **权力-回报联系**：有能力运用权力影响回报金额

**三个要素缺一不可。**

## 合并范围判断

### 必须合并
- 母公司**持有 >50% 表决权**（除非有证据证明不控制）
- **即使 <50%**，但满足：其他股东分散、与其他股东协议、能任命多数董事会成员等

### 禁止合并
- **已宣告被清理整顿**的子公司
- **已宣告破产**的原子公司
- 母公司是**投资性主体**（且子公司非投资服务提供者）

### 禁止随意变更合并范围（财会〔2025〕33号）
不得以自愿破产、一致行动协议、董事会席位变动、章程修订等为**唯一依据**调整合并范围。合并范围变更必须有实质性控制变更的商业理由。

## 合并程序

1. 汇总母子公司的资产、负债、权益、收入、费用
2. 抵销：母公司长期股权投资 vs 子公司所有者权益
3. 抵销：内部往来余额、内部交易、内部未实现损益
4. 确认少数股东权益和少数股东损益
5. 调整境外子企业的会计政策差异

## PPP 项目公司合并判断（施工企业适用）

国资发财评规：如央企是项目公司的**最大实体企业股东**，原则上应**合并**该项目公司，除非有确凿证据证明不控制。

| 合并 | 不合并 |
|---|---|
| 持股 >50% + 能主导相关活动 | 共同控制（合营/联营） |
| 是最大股东 + 董事会多数 | 少数股东对关键经营决策有否决权 |
| 全部损益归施工方承担 | 施工方仅为代理人 |

## 合并 vs 不合并的影响

| 指标 | 合并 | 不合并（权益法） |
|---|---|---|
| 资产 | 高（含在建PPP项目资产） | 低（仅长期股权投资） |
| 负债 | 高（含项目融资债务） | 低（表外） |
| 收入 | 高（含施工收入） | 低（仅投资收益） |
| 资产负债率 | 高 | 低 |

## 自检清单

1. [ ] 控制三要素是否同时满足？
2. [ ] 合并范围与上年是否有变动？变动是否有正当商业理由？
3. [ ] 境外子企业是否已调整为 PRC GAAP？
4. [ ] 内部交易和未实现损益是否完整抵销？
5. [ ] PPP 项目公司是否按国资委要求原则性合并？
