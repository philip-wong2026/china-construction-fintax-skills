# SASAC 财务快报时间表与报送要求

## 快速参考

| 报告类型 | 截止日 | 报送系统 |
|---|---|---|
| 月度快报预览（主要效益指标） | 每月 **5 日**前 | SASAC 财务监管系统 |
| 正式月度财务快报 | 每月 **9 日**前 | SASAC 财务监管系统 |
| 生产经营快报 | 每月 **9 日**前 | 同上 |
| "两金"（应收+存货）专项附表 | 每月 **9 日**前 | 同上 |
| 季度补充附表（境外/捐赠/债务风险/发债/金融衍生品） | 季度末次月 **15 日**前 | 同上 |

财政部快报（财资〔2024〕174号）：
| 报告类型 | 截止日 | 报送系统 |
|---|---|---|
| 月度经济效益月报 | 每月 **10 日**前 | 财政部统一报表平台 czbtybb.mof.gov.cn |

## 月度快报内容

### SASAC 系统
- 主要经济效益指标表（预览/正式均含）
- "两金"压降专项监测表（应收款项 + 存货）
- 债务风险监测表（季度）
- 债券发行及持有情况表（季度）
- 金融衍生品业务情况表（季度）
- 境外子企业财务情况表（季度）
- 对外捐赠支出情况表（季度）

### 财政部系统
- 封面代码
- 主要指标表
- 生产经营关键指标表
- 中央企业债务风险监测表
- 月度经济运行分析报告（文字说明）

## 数据列标准

| 列 | 含义 |
|---|---|
| D（本月数-境内） | 当月境内发生数 |
| E（本月数-境外） | 当月境外发生数 |
| F（本月数-合计） | D + E |
| G（本年累计-境内） | 1月到当月的累计 |
| 上年同期 | 去年同期可比数据 |

## 报告类型代码

| 代码 | 含义 |
|---|---|
| 0 | 单户企业 |
| 1 | 集团差额表 |
| 2 | 金融子企业 |
| 3 | 境外子企业 |
| 4 | 事业单位-企业 |
| 5 | 基建-企业 |
| 9 | 集团合并 |

## 施工行业补充表

建筑施工行业有特定的补充指标表，包括：
- 新签合同额
- 施工产值
- 竣工面积/里程
- 重大项目进展情况

## 内部报送倒排

集团合并快报截止 9 日，各级组织需内部倒排：

| 层级 | 内部截止（建议） |
|---|---|
| 项目部/总包部 → 工程公司 | 每月 **5 日**前 |
| 工程公司 → 工程局 | 每月 **7 日**前 |
| 工程局 → 集团总部 | 每月 **8 日**前 |
| 集团总部 → SASAC | 每月 **9 日**前 |

## 自检清单

1. [ ] 本月数 D/E/F 三列填列正确？
2. [ ] 本年累计数是否与上月累计 + 本月增加一致？
3. [ ] "两金"数据是否与快报主表一致？
4. [ ] 境外数据（E 列）是否单独列示？
5. [ ] 下属单位数据是否全覆盖（无遗漏项目）？

## 免责声明

本文件基于公开监管文件整理，具体企业的内部报送要求可能有所不同。
