# SASAC 一利五率考核指标体系

## 快速参考

| 指标 | 2025 目标 | 计算方式 |
|---|---|---|
| **利润总额** | 稳定增长 | 营业收入 − 营业成本 − 税费 − 费用 + 收益 |
| **资产负债率** | 保持总体稳定 | 负债总额 / 资产总额 × 100% |
| **净资产收益率(ROE)** | 同比提升 | 净利润 / 平均净资产 × 100% |
| **研发经费投入强度** | 同比提升 | 研发费用 / 营业收入 × 100% |
| **全员劳动生产率** | 同比提升 | 劳动生产总值 / 平均从业人员人数 |
| **营业收现率**（2025新增） | 同比提升 | 销售商品提供劳务收到的现金 / 营业收入 × 100% |

## 一、指标体系演进

| 阶段 | 指标组合 | 年代 |
|---|---|---|
| "两利一率" | 利润总额 + 净利润 + 资产负债率 | 2019年前 |
| "两利三率" | + 营业收入利润率 + 研发经费投入强度 | ~2020 |
| "两利四率" | + 全员劳动生产率 | ~2021-2024 |
| **"一利五率"** | 净利润→利润总额，+ 营业收现率 | **2025年起** |

## 二、各指标详解

### 利润总额

**重视原因**：净利润受所得税政策（如加计扣除、优惠税率到期）干扰，利润总额更能反映**经营层可控的经营成果**。

计算公式：
```
利润总额 = 营业总收入 − 营业总成本 + 其他收益 + 投资净收益
```

### 资产负债率

**施工行业关注**：建设施工央企负债率普遍偏高（PPP/BOT 项目融资导致）。SASAC 对不同行业设差异化红线：
- 工业企业：原则上不高于 70%
- 建筑企业：原则上不高于 75%（施工行业特殊性）

### 净资产收益率(ROE)

核心关注：央企资本回报效率。分母含少数股东权益口径。

### 研发经费投入强度

施工企业适用：新技术、新工法、BIM、智能建造设备的研发投入。
**注意**：加计扣除 100%（2023-2027 延续），填报时需区分费用化 vs 资本化。

### 全员劳动生产率

施工行业：分子含施工产值、自行完成产值，不含分包产值。
**分子取值**：劳动生产总值（增加值口径）= 劳动者报酬 + 生产税净额 + 固定资产折旧 + 营业盈余。

### 营业收现率（2025新增，最重要的变化）

**替换了**"营业现金比率"（经营活动现金流量净额/营业收入）。

| 新指标 | 旧指标 | 区别 |
|---|---|---|
| 销售商品提供劳务收到的现金 / 营业收入 | 经营活动现金流量净额 / 营业收入 | 聚焦**销售回款本身**，过滤营运资金波动 |
| 分子来自现金流量表"销售商品、提供劳务收到的现金" | 分子来自"经营活动现金流量净额" | 不含采购付款、税费支付等干扰 |

**施工企业影响**：工程款回款率直接体现。这是 SASAC 针对央企"两金"高企（尤其政府项目拖欠工程款）出台的关键指标。

## 三、"一业一策、一企一策"

SASAC 在统一"一利五率"基础上，针对不同行业和战略定位增加**定制化指标**：

| 行业 | 可能增加的指标 |
|---|---|
| 建筑央企 | 工程款回收率、新签合同额增长率、"两金"占流动资产比重 |
| 科技央企 | 专利授权量、科技成果转化率 |
| 国际工程 | 境外收入占比、汇率风险管理有效性 |

## 四、考核应用

| 等级 | 结果 |
|---|---|
| A 级 | 通报表扬、负责人薪酬上浮 |
| D 级 | 通报批评、负责人薪酬下浮、或予调整 |

## 自检清单

1. [ ] 营业收现率是否正确取数自"销售商品、提供劳务收到的现金"？
2. [ ] ROE 的分母是否为平均净资产（期初+期末/2）？
3. [ ] 研发费用是否区分了资本化和费用化？
4. [ ] 全员劳动生产率的分子是否剔除了分包产值？

## 免责声明

考核指标口径以SASAC年度正式文件为准，不同子企业可能有个性化调整。
