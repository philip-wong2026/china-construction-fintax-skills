# 国有资本经营预算

## 快速参考

| 事项 | 数值 |
|---|---|
| 施工企业利润上缴比例（2025年调整前） | 15%（第三档） |
| 施工企业利润上缴比例（2025年调整后） | **30%** |
| 2025年中央国有资本经营预算收入 | ~8547 亿元(+25.8% YoY) |
| 调入一般公共预算比例 | 67.2%（5741 亿元） |

## 上缴比例分档

| 档次 | 行业 | 比例 |
|---|---|---|
| 第一档 | 烟草 | 25% |
| 第二档 | 石油石化、电力、电信、煤炭 | 20% |
| 第三档 | 冶金、运输、电子、贸易、**建筑施工** | **15%→30%** |
| 第四档 | 军工、科研院所、邮政、铁路、文化 | 10% |
| 第五档 | 政策性企业（粮食储备等）、小型微型国企 | 免缴 |

## 2025年重大变化

财预〔2025〕97号将各档上缴比例**普遍提高 10-15 个百分点**：
- 建筑央企从 15% 提至 **30%**
- 利润上缴总规模大幅增长（中央非金融央企上缴 3750.77 亿元，同比增 78.5%）

## 施工企业影响

| 方面 | 影响 |
|---|---|
| 现金流 | 上缴比例翻倍，可分配利润减少 |
| 资金管理 | 需提前规划上缴资金，避免年末集中划款 |
| 预算编制 | 年度预算中必须单列国有资本经营预算支出 |
| 对项目影响 | 间接影响上级单位对项目部的资金调度 |

## 自检清单

1. [ ] 净利润是否已扣减上缴金额（按30%计提）？
2. [ ] 资金计划是否考虑了年中预缴+年末清算？

## 免责声明

具体上缴比例以SASAC对本企业的年度批复为准。
