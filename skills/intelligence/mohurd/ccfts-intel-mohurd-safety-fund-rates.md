# 施工企业安全生产费

## 快速参考

| 工程类别 | 计提比例 |
|---|---|
| 矿山工程 | 3.5% |
| 铁路、房屋建筑、**城市轨道交通** | **3.0%** |
| 水利水电、电力 | 2.5% |
| 冶炼、机电安装、化工石油、通信 | 2.0% |
| 市政公用、港口航道、公路 | 1.5% |

## 计提方式

```
当月安全生产费 = 当月完成建安产值 × 计提比例
```

- 按月末工程进度确认的建安产值计提
- 合同价中包含安全生产费，投标时需单独列示，**竞标时不得删减**

## 资金拨付（财务人员必须知道）

| 节点 | 比例 | 说明 |
|---|---|---|
| 开工1月内 | ≥ 50% | 业主支付给总承包 |
| 分包开工1月内 | ≥ 50% | 总承包拨付给分包 |
| 竣工后 | **余款退回业主** | 不可截留 |

## 会计处理

### 计提
```
借：合同履约成本 / 工程施工
  贷：专项储备
```

### 费用性支出（安全培训、防护用品等）
```
借：专项储备
   应交税费-应交增值税(进项税额)
  贷：银行存款
```

### 资本性支出（购安全设备）
```
借：在建工程 / 固定资产
  贷：银行存款
借：专项储备
  贷：累计折旧    ← 一次性全额计提折旧
```

## 税务处理差异

| 项目 | 会计 | 税务 |
|---|---|---|
| 计提时 | 计入成本 | **不得扣除**（汇算清缴时纳税调增） |
| 实际支出时 | 冲减专项储备 | 允许扣除 |
| 性质 | 预提 | 实际发生制 |

**年度汇算清缴**：A105000 表纳税调增 = 本年计提但未使用的安全生产费。

## 使用范围（安全生产费只能用于）

- 安全防护设施设备
- 应急演练和救援
- 安全培训教育
- 安全评价检测检验
- 现场作业人员安全防护用品
- 安全生产适用的新技术新标准新工艺推广应用

## 自检清单

1. [ ] 计提比例是否正确（区分工程类别）？
2. [ ] 计提基数是否为当月确认的建安产值？
3. [ ] 分包单位的安全生产费是否按期拨付（≥50%）？
4. [ ] 年末未使用的安全生产费是否已纳税调增？
5. [ ] 竣工时安全生产费余款是否计划退回业主？

## 免责声明

安全生产费管理以财政部和应急管理部最新文件为准。
