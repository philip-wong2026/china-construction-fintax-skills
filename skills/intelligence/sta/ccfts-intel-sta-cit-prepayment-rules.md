# 施工企业所得税预缴规则

## 快速参考

| 项目 | 规定 |
|---|---|
| 预缴地点 | **项目所在地**税务机关 |
| 预缴率 | **0.2%** × 实际经营收入 |
| 预缴基数 | 收入概念（**非**利润） |
| 分包款是否可扣除 | **不可**（以收入总额为基数，不可减分包） |
| 总机构汇总纳税 | 统一计算、分级管理、就地预缴、汇总清算、财政调库 |

## 计算

```
项目地预缴税额 = 项目实际经营收入 × 0.2%
```

**注意**：
- 基数含**全部价款和价外费用**
- **不可扣除分包款**（与 VAT 预缴不同）
- 如项目当月无收入，预缴额为 0

## VAT vs CIT 预缴对比

| 项目 | VAT 预缴 | CIT 预缴 |
|---|---|---|
| 预缴率 | 2%（一般）/ 3%（简易） | **0.2%** |
| 基数 | 可减分包款 | **不可减分包款** |
| 附加税费 | 同步预缴城建税/教育费附加 | 无附加 |
| 申报频率 | 按月/按次 | 按月/按季 |

## 汇总纳税流程

```
1. 总机构 ──────────────────────────────────────────────┐
   │ 统一计算集团全部应纳税所得额                        │
   │ 按营业收入/职工薪酬/资产总额三因素分摊              │
   │                                                    │
2. 各项目部 ────────────────────────────────────────── │
   │ 在项目地预缴 0.2%                                  │
   │ 取得完税凭证                                       │
   │                                                    │
3. 总机构汇总清算 ──────────────────────────────────────┤
   │ 全年应纳 CIT - 各地已预缴 = 应补（退）税额          │
   │ 年度汇算清缴截止日：次年 5月31日                    │
```

## 项目部不独立汇算

项目部（非法人）**不进行独立的年度汇算清缴**。年度 CIT 由总机构统一汇算。

## 项目公司（独立法人SPV）

项目公司作为独立法人，**不在汇总纳税范围内**：
- 独立计算年度应纳税所得额
- 适用 25% 法定税率
- 符合条件的可申请高新技术企业 15% 优惠

## 总包部注意事项

- 总包部作为内部管理单位（非法人），不独立缴纳 CIT
- 下属各项目部的外地预缴，由总机构汇总时统一抵扣
- 总包部内部之间的总分包现金流需在主表和附表间一致列示

## 自检清单

1. [ ] 项目部 CIT 预缴基数是否以收入为基数（不可减分包）？
2. [ ] 各地预缴完税凭证是否已归集至总机构？
3. [ ] 总包部 CIT 是否已正确汇总至总机构（非独立申报）？
4. [ ] SPV 项目公司是否独立申报 CIT（不参与汇总纳税）？

## 免责声明

具体执行口径以主管税务机关核定为准。
