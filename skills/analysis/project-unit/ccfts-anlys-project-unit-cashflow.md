# 项目现金流分析

## 快速参考

| 指标 | 计算 | 含义 |
|------|------|------|
| 营业收现率（项目级） | 实际收到工程款 / 累计确认施工收入 | 确认的收入中有多少真的收到了钱 |
| 计量差 | 对上确认收入 − 对下确认分包成本 | 总包vs分包的计量时间差 |
| 垫资比例 | (累计支出 − 累计收款) / 合同总价 | 项目垫资规模 |
| 资金集中度 | 归集至上级的资金 / 项目全部资金 | 资金池贡献度 |

## 一、项目收付款节奏分析

### 收款节奏

```
每月收款进度：
  计划收款 vs 实际收款 vs 累计收款

收款滞后天数：实际收款日 − 合同约定付款日

业主付款延迟率 = 延迟付款金额 / 全部应收款
```

### 付款节奏

```
每月付款进度：
  对分包付款：按分包合同约定
  对材料商付款：按采购合同约定
  对员工/农民工：按月

付款与收款的匹配度：
  付款 / 收款 > 100% → 垫资（消耗自有资金）
  付款 / 收款 < 100% → 资金积累（正向现金流）
```

## 二、计量差分析——施工企业现金流的关键

### 对上 vs 对下计量

```
对上计量（总包→业主）：
  已完工 → 监理/业主确认 → 计入合同资产 → 开票 → 收款

对下计量（总包→分包）：
  分包已完工 → 总包确认 → 计入应付账款 → 付款
```

### 计量差的三层含义

| 层次 | 差异 | 现金流影响 |
|------|------|-----------|
| 时间差 | 对上计量比收款晚 1-3 个月 | 正常营运资金 |
| 金额差 | 对上计量金额 > 对下结算金额（总包利润部分） | 总包利润不需要对下支付 |
| 确认差 | 变更/签证：对下已确认但对上未确认 | **垫资风险** |

### 不良信号

| 信号 | 含义 |
|------|------|
| 对上计量滞后期持续延长 | 业主故意拖延结算 |
| 对下付款被迫提前（分包/材料商施压） | 双向挤压 → 现金流恶化 |
| 垫资比例持续上升 | 项目现金流不可持续 |

## 三、垫资分析

### 垫资来源

| 垫资方 | 方式 |
|--------|------|
| 上级法人（工程公司/局） | 资金下拨 |
| 分包商 | 延迟支付分包款 |
| 材料商 | 赊购材料（应付账款） |
| 自有资金 | 项目前期收款不足以覆盖支出 |

### 垫资比例

```
累计垫资 = 累计支出（人工+材料+机械+分包+间接费） − 累计收款（业主付款）

垫资比例 = 累计垫资 / 合同总价

正常：< 10%（施工行业常见）
关注：10-20%
预警：> 20%（垫资过大 → 资金链风险）
```

### 垫资的隐性成本

垫资不是"免费"的——资金成本 = 垫资金额 × 资金占用时间 × 资金成本率（通常 5-8%）。

```
垫资 500 万元，占用 6 个月，资金成本率 6%
隐性成本 = 500 × 6/12 × 6% = 15 万元

这 15 万元直接侵蚀项目利润。
```

## 四、营业收现率——下沉至项目层面

### 项目级营业收现率

```
项目营业收现率 = 累计实际收款 / 累计确认施工收入

> 100%：收款 > 收入确认（良性，含预付款）
= 100%：收款 = 收入（理想）
80-100%：正常（施工行业常态，质保金/滞后收款）
< 80%：预警（大量"白条收入"）
< 50%：严重（资金链危机）
```

### 与集团营业收现率的关联

SASAC 考核的集团营业收现率 = Σ 各项目实际收款 / Σ 各项目收入

项目层面的营业收现率是集团指标的底层驱动。集团要"同比提升"，关键在于各项目的收款管理。

## 六、自检清单

1. [ ] 项目营业收现率是否 ≥ 80%？
2. [ ] 垫资比例是否 < 20%？
3. [ ] 对上计量和对下计量的时间差是否在正常范围内（1-3个月）？
4. [ ] 有无"变更已施工但对上未确认"导致的隐性垫资？
5. [ ] 月度资金计划是否编制并执行？
6. [ ] 资金缺口是否提前向上级预警？

## 免责声明

项目现金流的具体管理要求因企业的资金管理制度而异。政府项目的回款周期通常长于商业项目。
