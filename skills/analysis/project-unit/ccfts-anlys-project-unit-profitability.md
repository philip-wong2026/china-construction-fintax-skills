# 项目盈利能力分析

## 快速参考

| 盈利指标 | 计算 | 含义 |
|---------|------|------|
| 毛利率 | (收入 − 成本) / 收入 | 项目最基本的盈利水平 |
| 净利率 | 净利润 / 收入 | 含管理费和税金后的最终回报 |
| 投资回报率 | 净利润 / 项目总投入 | 资本回报效率 |
| 责任成本节超率 | (责任预算成本 − 实际成本) / 责任预算成本 | 正=节余（好），负=超支（不好） |

## 一、项目盈利分析框架

### 收入端分析

| 分析项 | 内容 |
|--------|------|
| 合同收入 | 签约金额 + 变更/签证/索赔 |
| 累计确认收入 | 按履约进度累计确认 |
| 剩余合同收入 | = 合同收入 − 累计确认 |
| 收入质量 | 开票率（已开票 / 已确认收入）、收现率 |

### 成本端拆解（施工企业最核心的分析）

| 成本项 | 占比（典型） | 偏差分析重点 |
|--------|------------|------------|
| 人工费 | 15-25% | 工效、工资率 |
| 材料费 | 40-60% | 量差（超耗）+ 价差（涨价） |
| 机械费 | 10-20% | 利用率、台班单价 |
| 分包费 | 10-30% | 分包结算、范围变化 |
| 间接费 | 5-10% | 管理人员、临建、办公 |

### 毛利率趋势

```
月度毛利率追踪：
  1月：x% → 2月：x% → ... → 累计：x%

趋势下降 → 预警（成本上升 or 收入确认滞后）
趋势上升 → 关注是否有人为调节（提前确认收入）
```

### 亏损项目专题分析

若项目出现亏损（or 预计亏损）：

| 分析项 | 内容 |
|--------|------|
| 亏损金额 | 累计亏损 + 预计后续亏损 |
| 亏损原因 | 报价过低/成本超支/变更未批/业主延误/地质意外 |
| 责任归属 | 投标阶段/施工阶段/外部不可抗 |
| 扭亏措施 | 索赔推进/成本压缩/分包重新谈判 |
| 预计损失 | 是否需要计提亏损合同预计负债 |

## 三、项目盈利评级

| 等级 | 毛利率 | 说明 |
|------|--------|------|
| A（优秀） | > 15% | 大幅超责任预算 |
| B（良好） | 10-15% | 略超 or 达标 |
| C（正常） | 5-10% | 正常施工利润 |
| D（微利） | 0-5% | 盈亏平衡附近 |
| E（亏损） | < 0% | 需专题分析+追责 |

## 四、项目间横向对比

同一工程公司/工程局下的项目横向对比：

| 排名 | 项目 | 毛利率 | 材料成本率 | 机械成本率 | 分包成本率 |
|------|------|--------|-----------|-----------|-----------|
| 1 | A项目 | 12% | 42% | 18% | 20% |
| 2 | B项目 | 8% | 48% | 22% | 15% |
| ... | ... | ... | ... | ... | ... |

通过横向对比找出"优秀项目"的管理经验，和"落后项目"的改进空间。

## 五、自检清单

1. [ ] 项目毛利率是否按月追踪？（趋势比单点更重要）
2. [ ] 成本偏差是否拆解到量差和价差？
3. [ ] 有无亏损项目？（是 → 是否已计提预计损失？）
4. [ ] 各成本项的占比是否在合理范围内？
5. [ ] 项目间横向对比是否定期进行？
