# 一利五率经营业绩分析

> **SLOT 说明**：本文件为 KPI 分析的通用 Base。{INDICATOR_TARGETS}，数据来源 {DATA_SOURCES}。

## 快速参考

| 指标 | 计算 | 目标 | 数据源 |
|------|------|------|--------|
| **利润总额** | 营业总收入−营业总成本+其他收益+投资收益 | 稳定增长 | 利润表 |
| **资产负债率** | 负债总额 / 资产总额 × 100% | 保持稳定（建筑≤75%） | B/S |
| **净资产收益率(ROE)** | 净利润 / 平均净资产 × 100% | 同比提升 | P&L + B/S |
| **研发经费投入强度** | 研发费用 / 营业收入 × 100% | 同比提升 | P&L |
| **全员劳动生产率** | 劳动生产总值 / 平均从业人数 | 同比提升 | P&L + HR |
| **营业收现率** | 销售商品提供劳务收到的现金 / 营业收入 × 100% | 同比提升 | CF + P&L |

## 一、各指标详细分析框架

### 利润总额

| 分析维度 | 方法 |
|---------|------|
| 同比分析 | (本期 − 上年同期) / 上年同期 |
| 环比分析 | (本期 − 上期) / 上期 |
| 预算偏差 | (实际 − 预算) / 预算 |
| 结构分析 | 各业务板块（基建/市政/房建/其他）利润占比 |
| 驱动因素 | 收入端（量/价）+ 成本端（人工/材料/机械/分包）+ 费用端 |

**施工企业关注**：利润总额是否剔除了一次性非经营性损益（政府补贴/资产处置）？

### 资产负债率

| 分析维度 | 方法 |
|---------|------|
| 绝对水平 | vs 75% 红线、vs 行业均值 |
| 变动趋势 | 环比/同比变动百分点 |
| 驱动因素 | 资产增长 vs 负债增长哪个更快？ |
| 结构分析 | 合并 vs 不含PPP的负债率对比 |

**施工企业关注**：PPP 项目合并推高负债率——需区分"经营性负债率"和"含PPP负债率"。

### ROE（净资产收益率）

| 分析维度 | 方法 |
|---------|------|
| 杜邦分析 | ROE = 净利率 × 资产周转率 × 权益乘数 |
| 行业对标 | vs 行业ROE均值、vs 同类央企 |
| 含少数股东 vs 不含 | 合并口径含少数股东权益 → ROE被稀释 |

### 研发经费投入强度

| 分析维度 | 方法 |
|---------|------|
| 绝对水平 | 研发费用 / 营业收入 |
| 结构分析 | 费用化 vs 资本化 |
| 产出评价 | 研发投入→专利/工法/省部级奖项 |
| CIT 优惠 | 100% 加计扣除 → 税收节省额 |

### 全员劳动生产率

| 分析维度 | 方法 |
|---------|------|
| 绝对水平 | 劳动生产总值 / 平均从业人数 |
| 分子 | 劳动者报酬 + 生产税净额 + 固定资产折旧 + 营业盈余 |
| 分母注意 | 不含分包产值对应的人数 |
| 行业差异 | 铁路/城轨劳动生产率 > 房建/公路 |

### 营业收现率（2025年最重要指标）

| 分析维度 | 方法 |
|---------|------|
| 绝对水平 | 销售商品提供劳务收到的现金 / 营业收入 |
| 含义 | > 100% → 本期回收 > 本期确认收入（良性） |
|   | < 100% → 确认了收入但没收回来（"白条收入"） |
| 对标 | 一利五率考核要求"同比提升" |
| 驱动因素 | 结算进度 + 收款进度 + 质保金 + 政府项目回款 |

## 二、同比/环比分析方法

### 标准分析模板

```
【指标名称】
本期：xxx
上年同期：xxx → 同比：+x%
上期：xxx → 环比：+x%
预算目标：xxx → 偏差：x%
行业对标：xxx（行业均值/同类央企）
评价：正常 / 关注 / 预警
原因：[简要原因]
建议：[改进建议]
```

### 信号灯预警

| 色 | 条件 | 行动 |
|----|------|------|
| 绿灯 | 偏差 ≤ 5% | 正常 |
| 黄灯 | 偏差 5-15% | 关注 + 分析原因 |
| 红灯 | 偏差 > 15% | 预警 + 专题分析 + 采取措施 |
