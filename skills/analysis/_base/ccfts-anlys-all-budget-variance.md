# 施工企业预算偏差分析

> **SLOT 说明**：预算结构 {BUDGET_TEMPLATES}，偏差阈值 {VARIANCE_THRESHOLDS}。

## 快速参考

| 偏差类型 | 公式 | 正负含义 |
|---------|------|---------|
| 绝对偏差 | 实际 − 预算 | + = 超预算（成本）/ + = 超目标（收入） |
| 偏差率 | (实际 − 预算) / 预算 × 100% | |
| 收入偏差 | + = 有利（超收） | |
| 成本偏差 | + = 不利（超支） | |
| 利润偏差 | + = 有利（超额利润） | |

## 一、预算偏差分析框架

### 分析维度

```
总收入偏差
  ├── 量差（实际工程量 − 预算工程量）× 预算单价
  └── 价差（实际单价 − 预算单价）× 实际工程量

总成本偏差
  ├── 人工费偏差
  │    ├── 量差（实际用工 − 定额用工）× 预算工资率
  │    └── 价差（实际工资率 − 预算工资率）× 实际用工
  ├── 材料费偏差
  │    ├── 量差（实际用量 − 定额用量）× 预算价
  │    └── 价差（实际采购价 − 预算价）× 实际用量
  ├── 机械费偏差
  │    ├── 台班差（实际台班 − 预算台班）× 预算单价
  │    └── 价差（实际台班单价 − 预算单价）× 实际台班
  ├── 分包费偏差
  └── 间接费偏差
```

## 二、收入偏差分析

### 量差分析

| 偏差原因 | 说明 |
|---------|------|
| 工程变更增加/减少 | 设计变更/签证 → 工程量变化 |
| 计量差异 | 对上计量 vs 对下计量的时间差 |
| 施工进度超/滞后 | 超前完成 → 收入超前确认 |
| 合同调整 | 材料调差/索赔确认 |

### 价差分析

| 偏差原因 | 说明 |
|---------|------|
| 材料调差 | 合同约定材料调差机制 → 单价调整 |
| 新增项目单价 | 合同外新增项目的协商单价 |
| 奖励/罚款 | 工期/质量奖惩 |

## 三、成本偏差分析

### 材料费偏差（施工企业最大成本项，占比 40-60%）

| 偏差方向 | 量差 | 价差 |
|---------|------|------|
| 超额(不利) | 浪费/损耗/偷盗/定额不准 | 市场涨价/采购渠道差 |
| 节余(有利) | 工艺改进/限额领料有效 | 锁价/集中采购/替代材料 |

**根因追踪**：超耗必须追溯到具体材料（钢材/水泥/砂石）和具体工点/工班。

### 分包费偏差

| 偏差原因 | 说明 |
|---------|------|
| 分包范围变化 | 新增分包内容 |
| 分包单价调整 | 合同变更/市场变化 |
| 结算差异 | 总包确认进度 vs 分包确认进度的差异 |

## 四、红黄绿灯预警

| 等级 | 条件 | 行动 |
|------|------|------|
| 绿灯 | 偏差率 ≤ 5% | 正常监控 |
| 黄灯 | 偏差率 5-15% | 书面说明原因 + 对策 |
| 红灯 | 偏差率 > 15% | 专题分析报告 + 问责 |

**施工企业特别关注**：
- 材料费偏差 > 10% → 必须逐项分解到量差/价差
- 分包费偏差 > 10% → 检查分包结算是否合规
- 收入偏差 > 10% → 检查是否有人为调节收入确认进度

## 五、偏差分析报告模板

```
【预算偏差分析报告 — 202X年X月】

一、总体评价
  营业收入偏差：+x%（有利/不利）
  营业成本偏差：+x%（有利/不利——成本低为有利）
  利润总额偏差：+x%

二、重大偏差项目（>10% or 金额 > xxx 万元）
  1. 钢材成本超支 +15%（+50万）
     原因：量差 +8%（超耗）+ 价差 +7%（市场涨价）
     对策：加强限额领料 + 询价后锁价采购

三、上期对策执行跟踪
  1. [对-策] [执行情况] [效果]

四、本期红灯项目清单
  [项目名称] [偏差率] [原因] [责任人]
```

## 六、自检清单

1. [ ] 偏差是否分解到量差和价差（非笼统的"超支"）？
2. [ ] 材料费偏差是否追溯到具体材料和工点？
3. [ ] 红黄绿灯预警是否触发并采取了对应行动？
4. [ ] 上期的对策是否跟踪了执行情况？
5. [ ] 偏差分析是否排除了会计政策/估计变更的影响？
