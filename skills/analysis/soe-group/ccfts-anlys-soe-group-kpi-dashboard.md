## 一、集团经营看板结构

### 月度经营快报分析要点

| 分析板块 | 核心指标 | 频率 |
|---------|---------|------|
| 规模指标 | 新签合同额、营业收入、施工产值 | 月度 |
| 效益指标 | 利润总额、毛利率、净利率 | 月度 |
| 效率指标 | ROE、劳动生产率、营业收现率 | 季度 |
| 风险指标 | 资产负债率、"两金"、经营现金流 | 月度/季度 |
| 发展指标 | 研发强度、新签合同额增长率 | 季度/年度 |

### 新签合同额分析

| 维度 | 分析内容 |
|------|---------|
| 总金额 | 同比/环比/预算偏差 |
| 业务板块 | 基建/市政/房建/水利/港口/矿山/机电/海外 占比 |
| EPC 占比 | EPC 合同在新签中的比例趋势 |
| 重大项目 | 单合同 > X 亿的项目清单 |
| 区域分布 | 各省/境外 分布 |
| 合同质量 | 预计毛利率、业主类型（政府/国企/民企） |

### "两金"压降进度

| 指标 | 分析 |
|------|------|
| 应收账款 + 存货(含合同资产) | 总额 / 占流动资产比 / 同比趋势 |
| 压降目标 | 年度压降目标 vs 实际进度 |
| 账龄结构 | 1年以内/1-2年/2-3年/3年以上 占比变化 |

## 二、月度经营分析会议要点

### 标准议程

1. **上月经营指标通报**（5 分钟）——一利五率 + 新签 + 产值
2. **偏差分析报告**（15 分钟）——重大偏差项目和原因
3. **"两金"压降进度**（10 分钟）——各单位排名、滞后单位表态
4. **重大项目进展**（10 分钟）——重点项目的进度/收入/利润
5. **本月重点事项**（5 分钟）——需集团协调的事项

### 报告要点

| 章节 | 内容 |
|------|------|
| 摘要 | 一句话总结：哪些指标达标、哪些预警 |
| 规模篇 | 新签/产值/营收 同比环比 |
| 效益篇 | 利润/毛利率/净利率 同比 + 驱动因素 |
| 风险篇 | 资产负债率/"两金"/现金流 |
| 排名篇 | 各工程局/工程公司的主要指标排名 |
| 专题篇 | 本月重点专项（如PPP、境外、审计整改） |

## 三、季度/年度深度分析

季度和年度需增加：

| 深度分析 | 内容 |
|---------|------|
| 业务板块盈利分析 | 各板块（基建/市政/房建/水利/港口/矿山/机电）的收入/利润/毛利率 |
| 区域盈利分析 | 各省/境外的收入/利润 |
| 项目盈利分布 | 盈利项目 vs 亏损项目 → 亏损项目专题分析 |
| 经营性现金流分析 | 收入-收现匹配度、垫资规模 |
| 行业对标 | 与同类央企的指标对比 |

## 四、自检清单
