# "两金"结构分析与压降

## 快速参考

| "两金"构成 | 会计科目 | 关注点 |
|-----------|---------|--------|
| **应收款项** | 应收账款(1122) + 应收票据 | 账龄、周转率、坏账 |
| **存货+已完工未结算** | 合同资产(1483) + 合同履约成本(5601)余额 | 已完工未结算规模、挂账周期 |

## 一、应收款项分析

### 账龄结构

| 账龄 | 金额 | 占比 | 趋势 | 行动 |
|------|------|------|------|------|
| 1年以内 | xxx | x% | → | 正常催收 |
| 1-2年 | xxx | x% | ↑/↓ | 重点催收 |
| 2-3年 | xxx | x% | ↑/↓ | 法律评估 |
| 3年以上 | xxx | x% | ↑/↓ | 核销/以资抵债 |

**预警**：账龄 > 1 年占比上升 → 回款恶化信号。

### 周转率

```
应收账款周转率 = 营业收入 / 平均应收账款
应收账款周转天数 = 365 / 周转率

行业均值：周转天数 90-180 天（建筑行业回款周期长）
```

| 周转天数 | 评价 |
|---------|------|
| < 90 天 | 优秀 |
| 90-150 天 | 正常 |
| 150-210 天 | 关注 |
| > 210 天 | 预警 |

### 客户集中度

前 5 大客户的应收款占比：
- > 50% → 集中度风险高（一个客户出问题影响大）
- 政府/城投类占比 → 信用风险低但流动性风险高

## 二、已完工未结算（合同资产）分析

### 构成分析

| 已完工未结算类型 | 原因 |
|----------------|------|
| 正常进度差异 | 对业主计量周期 vs 施工进度的正常时间差（1-3个月） |
| 变更/签证待确认 | 已施工但业主未签认（**重点关注**） |
| 索赔待确认 | 已发生但业主未确认 |
| 结算周期异常 | 业主故意拖延结算（政府项目多见） |

### 挂账周期

| 挂账周期 | 信号 | 行动 |
|---------|------|------|
| 1-3 个月 | 正常 | — |
| 3-6 个月 | 关注 | 了解原因 |
| 6-12 个月 | 预警 | 加快推进结算 |
| > 12 个月 | **严重** | 评估是否需要计提减值 |

### "已完工未结算 / 营业收入" 比率

```
比率 = 期末合同资产 / 年度营业收入

正常：10-25%（1-3 个月计量周期）
预警：> 30%（大量已完工但无法结算）
严重：> 50%（结算严重滞后 or 业主问题）
```

## 三、坏账计提充足性

### 按账龄计提 vs 预期信用损失

SASAC 和审计关注坏账计提是否充足：

| 检查项 | 方法 |
|--------|------|
| 账龄迁徙率 | 各账龄段的迁徙率 → 预期损失率 |
| 单项计提 | 重大单项应收款是否逐项评估 |
| 与行业对比 | 坏账准备率是否低于行业均值 |
| 政府项目特殊考量 | 低违约但需考虑超长账龄的减值 |

**常见问题**：央企坏账计提比例偏低（依赖政府信用），审计建议提高计提比例。

## 四、"两金"压降措施

| 应收款项压降 | 已完工未结算压降 |
|------------|----------------|
| 加强催收（分级/现场/法律） | 加快对上计量结算 |
| 以资抵债 | 推进变更/签证的业主确认 |
| 保理/应收账款证券化（ABS） | 控制"先干后签"（变更先获批后施工） |
| 业主付款节点奖励/罚息 | 清理长期挂账的已完工未结算 |
| 不再承接回款差的项目 | |
