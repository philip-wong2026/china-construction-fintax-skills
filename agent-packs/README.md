# 场景包：给普通 AI 用户直接用

这个目录是给不会 GitHub、不会 MCP、不会项目配置的人用的。

一句话：

> 场景包就是“复制给豆包、马维斯看的简化版说明书”。

## 为什么要有场景包

完整 CCFTS 有 96 个 skill 文件。普通用户不可能知道该读哪个。

所以这里把常见场景整理成一个文件：

- 你该上传什么
- 你该复制哪段提示词
- AI 应该输出什么
- 哪些地方必须人工复核

## 怎么用

1. 打开你的 AI 工具，比如豆包或马维斯。
2. 选一个场景包。
3. 上传这个场景包，或者把内容复制给 AI。
4. 上传脱敏业务资料。
5. 复制场景包里的提示词。
6. 看 AI 输出，重点看“人工复核清单”。
