# 马维斯场景包：本地施工企业财税文件检查

## 适用场景

这个场景包适合把 CCFTS 当成本地文件夹里的财税规则，让马维斯读取后辅助检查 Word、Excel、PDF、截图或 CSV。

## 使用步骤

1. 把 `china-construction-fintax-skills` 保存在本地。
2. 打开马维斯。
3. 让马维斯读取本文件，以及必要时读取 `START_HERE.md`。
4. 提供脱敏后的业务文件。
5. 要求马维斯输出“业务草稿 + 人工复核清单”。

## 复制给马维斯的提示词

```text
请读取当前文件夹中的 CCFTS 马维斯场景包，并按中国施工企业财税技能包的规则帮我检查这些脱敏业务文件。

请先说明你读取了哪些文件和规则，然后按以下结构输出：
1. 当前业务场景判断。
2. 已确认事实。
3. 推断结论。
4. 缺失信息。
5. 风险点。
6. 建议下一步。
7. 人工复核清单。

不要把输出作为最终合规意见。涉及税务、审计、报送、重大经营决策的事项，必须提示人工复核。
```

## 数据安全提醒

不要把未经批准的真实企业数据、合同编号、税号、项目名称、银行账号、个人信息交给个人版 AI 工具处理。
