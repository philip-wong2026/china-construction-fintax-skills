# CCFTS 人工复核模板

> AI agent 输出任何财税结论前，必须逐项通过以下人工复核。此模板应作为每次使用 CCFTS 技能的标准 checklist。

## 一、主体与组织架构复核

| # | 复核项 | 说明 |
|---|--------|------|
| 1.1 | 主体类型判定是否与实际一致？ | 先分清项目公司、项目部、分公司、子公司、集团汇总层，不使用内部技术代号 |
| 1.2 | 组织层级（项目公司/项目部/工程局/集团）是否正确？ | 不同层级的取整、权益、税务规则完全不同 |
| 1.3 | 是否有混合主体（同一系统下同时有项目公司和总包部）？ | 混合场景必须分别处理 |
| 1.4 | 独立法人和非法人的税务义务区分是否正确？ | 法人通常独立承担企业所得税、增值税等纳税义务；非法人项目部通常不单独作为法人主体处理 |
| 1.5 | 一般纳税人和小规模纳税人的身份是否正确？ | 影响增值税税率和进项抵扣权 |

## 二、合同与业务事实复核

| # | 复核项 | 说明 |
|---|--------|------|
| 2.1 | EPC / 纯施工 / PPP / 专项债的项目模式是否准确？ | 不同模式收入确认和增值税处理不同 |
| 2.2 | 甲供材的计税方式：2026.1.1 后新签合同是否已改为 9%？ | 甲供工程简易计税已终止 |
| 2.3 | 清包工是否满足条件？分包发票限制（只能普票）是否已告知分包商？ | |
| 2.4 | 变更/签证/索赔是否满足"极可能不会重大转回"的确认条件？ | 口头承诺 ≠ 可确认 |
| 2.5 | 质保金比例是否 ≤ 3%？缺陷责任期是否明确？ | 法规上限 |
| 2.6 | 是否有亏损合同的预警信号？是否已计提预计损失？ | 审计重点关注 |

## 三、税务口径复核

| # | 复核项 | 说明 |
|---|--------|------|
| 3.1 | 纳税义务发生时间是否正确（最早原则）？ | 开票日/收款日/合同约定收款日 孰早 |
| 3.2 | 跨区域增值税预缴的预征率（2%/3%）选择是否正确？ | 简易计税是 3%，不是 2% |
| 3.3 | 附加税费是否按项目所在地税率（非机构所在地）？ | 城建税 7%/5%/1% |
| 3.4 | 简易计税项目的进项税额是否已全额转出？ | 绝对不可混抵 |
| 3.5 | EPC 项目 E/P/C 是否分别开票（避免混合销售）？ | 混合销售从高（13%）or 从主（9%） |
| 3.6 | 企业所得税汇算清缴——安全生产费是否已做纳税调增/调减？ | 未使用部分必须调增 |
| 3.7 | 境外 WHT 是否按税收协定优惠税率（非标准税率）？ | 需持有《中国税收居民身份证明》 |
| 3.8 | 全电发票备注栏是否填写了项目地址+项目名称？ | 缺失 → 异常发票 → 受票方不得抵扣 |
| 3.9 | 印花税是否按含税价格计算（非不含税）？ | |

## 四、会计口径复核

| # | 复核项 | 说明 |
|---|--------|------|
| 4.1 | 收入确认方法（时段/时点）是否符合 CAS 14 五步法？ | 施工企业通常是时段（条件 2 或 3） |
| 4.2 | 履约进度是投入法还是产出法？未安装材料是否已剔除？ | 投入法最常见 |
| 4.3 | 预计总成本是否在最近季度末更新过？变动是否追溯调整了当期收入？ | 至少每季度重估 |
| 4.4 | 亏损合同的预计损失是否足额计提？ | CAS 14 要求立即确认 |
| 4.5 | 合同资产(1483)/合同资产减值准备(1484)的取整是否做了组件级？ | 绝对不能先合计再取整 |
| 4.6 | 项目公司等独立法人主体的其他应付款是否使用了合适的校验方法？ | 不要只机械读取 2241，还要检查负债合计、税费、薪酬等勾稽关系 |
| 4.7 | 项目部等非法人管理主体的资产负债表取整规则是否一致？ | 利润表和资产负债表可能使用不同取整规则，必须按企业口径复核 |
| 4.8 | 净利润取数是否正确？ | 不要只看期末余额，要结合本期发生额、利润总额和所得税费用复核 |
| 4.9 | 利息收入是否已取反为正数填列？ | |
| 4.10 | 期末调整（2211-02/6602）是否已在非季度末应用？ | 需用户显式激活 |

## 五、输出责任边界声明

>

> - 审计资料提供
