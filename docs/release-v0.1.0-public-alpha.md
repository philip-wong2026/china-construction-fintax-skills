# v0.1.0-public-alpha 发布说明

这是 CCFTS 的第一个公开试用版本。

## 版本定位

`v0.1.0-public-alpha` 是公开试验版。

它适合：

- 研究中国施工企业财税 AI skills 的组织方式
- 在脱敏资料上试用施工企业快报、税务、分析等场景
- 给 Codex、Claude、Trae Solo、豆包、马维斯等 AI 工具提供行业上下文
- 收集建筑企业财税、项目管理、数字化从业者的反馈

它不适合：

- 直接用于正式纳税申报
- 直接用于审计、监管报送、财务报告定稿
- 替代注册会计师、税务师、律师或企业正式制度
- 未经脱敏上传真实企业敏感数据到公开 AI 工具

## 本版本包含

- 96 个 Markdown skill 文件
- 6 个职能领域：财务报表、会计核算、税务申报、经济活动分析、管理办法、法规知识库
- 5 类组织层级：央企集团、独立法人子公司、分公司、项目部、项目公司
- 桌面 AI 场景包：豆包、马维斯、Trae Solo
- 可选 MCP 服务
- 脱敏 demo：项目部快报、项目公司快报、增值税跨区域预缴
- 新手文档：10 分钟上手、项目背景包模板、人工复核模板

## 使用建议

第一次使用时，请按以下顺序阅读：

1. `docs/quick-start-10min.md`
2. `docs/context-pack-template.md`
3. `examples/README.md`
4. `START_HERE.md`

本项目不构成法律、税务、审计、会计或投资建议。AI 输出必须由具备相应专业能力的人员结合最新法规、地方口径和企业制度复核后使用。
