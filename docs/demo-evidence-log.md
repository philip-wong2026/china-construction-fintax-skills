# Demo 和验证记录

本文件记录公开 demo 和后续验证案例的状态。

## 当前公开 demo

| Demo | 当前级别 | 作用 | 主要限制 |
| --- | --- | --- | --- |
| 项目部/总包部资料检查 | L1 教学样例 | 演示 AI 先识别非法人项目部/总包部边界 | 简化数据，尚不能证明真实快报效率提升 |
| 项目公司资料检查 | L1 教学样例 | 演示 AI 区分独立法人项目公司和项目部 | 简化数据，尚未覆盖融资、投资回收、治理风险 |
| 增值税跨区域预缴检查 | L1 教学样例 | 演示 AI 先追问项目所在地、机构所在地、计税方式等事实 | 未覆盖不同地区电子税务局操作差异 |

## 当前结论

当前 demo 足以说明 CCFTS 的使用方式和输出结构，但还不足以证明它已经在真实企业管理中形成稳定效果。

下一阶段应优先补充实务脱敏样例和人工复核记录。
